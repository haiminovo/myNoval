# 伊瑟尔大陆世界设定

## 平行宇宙与本源能量

### 平行宇宙

穿越不是"空间穿越"，而是"平行宇宙"的概念。每个宇宙都有自己的本源能量和法则体系：

| 宇宙 | 本源能量 | 法则体系 | 特点 |
|------|---------|---------|------|
| **伊瑟尔大陆** | 以太 | 以太法则、王座规则 | 以太是世界的血液，掌控者可引动世界法则 |
| **主角原世界** | 元素力 | 物理法则、科学体系 | 元素力是能量的一种表现形式，物理法则主导 |
| **其他宇宙** | 各有不同 | 各有不同 | 每个宇宙都有自己的本源能量和法则 |

### 穿越的本质

穿越是"平行宇宙"之间的意识转移：
- 意识从原宇宙转移到伊瑟尔大陆
- 原宇宙的记忆和知识会保留
- 但身体必须适应新的本源能量（以太）
- 系统是世界法则生成的"适配器"，帮助异界意识适应以太

### 本源能量：以太

以太是伊瑟尔大陆的本源能量，是一切力量的本质：

**以太的本质**：
- 以太是世界的"血液"，是因果的载体
- 所有力量形式本质上都是以太的不同表现
- 掌控者可以依靠以太，根据自己的主观意识引动世界法则
- 以太不是"物质"，而是一种"能量形式"

**以太的三种形态**：

| 形态 | 特性 | 表现形式 |
|------|------|---------|
| **创造之以太** | 温和、滋养、有序 | 生命、创造、治愈 |
| **毁灭之以太** | 狂暴、破坏、混乱 | 爆炸、毁灭、腐败 |
| **观测之以太** | 中性、渗透、记录 | 时空、因果、记录 |

## 核心概念：以太（Ether）

以太是世界的血液，是因果的载体，掌控者可以依靠以太，根据自己的主观意识引动世界法则。

### 三种以太

| 类型 | 特性 | 代表 |
|------|------|------|
| **创造之以太** | 温和、滋养、有序 | 人类主流 |
| **毁灭之以太** | 狂暴、破坏、混乱 | 魔物、兽人 |
| **观测之以太** | 中性、渗透、记录 | 混血、源晶 |

### 以太与力量的关系

所有力量形式本质上都是以太的不同表现：

| 力量形式 | 本质 | 特点 |
|---------|------|------|
| **斗气** | 以太在身体中的凝聚 | 强化身体、爆发力量 |
| **魔法** | 以太在精神中的引导 | 元素操控、术式释放 |
| **进化** | 以太在存在层面的重塑 | 改变生命本质、获得特殊能力 |
| **领域** | 以太在空间的展开 | 控制一定范围内的以太流动 |
| **法则** | 以太的终极形态 | 直接干涉世界法则，触碰因果 |

**以太的作用机制**：
1. **斗气**：以太凝聚在身体各处，形成"以太脉络"，强化肉体
2. **魔法**：以太由精神引导，形成"以太回路"，释放术式
3. **进化**：以太渗透到存在层面，重塑生命本质
4. **领域**：以太以自身为中心向外展开，控制范围内以太流动
5. **法则**：以太达到极高密度，可以直接干涉世界法则

### 原住民的以太适应

原住民天生适应以太：

| 阶段 | 适应程度 | 能力 |
|------|---------|------|
| **出生** | 天然适应以太环境 | 可以感知以太，但无法操控 |
| **觉醒** | 开始操控以太 | 学习斗气或魔法的基础 |
| **掌控** | 熟练操控以太 | 可以使用中下位术式，形成以太脉络 |
| **精通** | 精通某种以太操控 | 可以开创新术式，形成领域 |
| **王阶** | 触碰法则层面 | 可以直接干涉世界法则，获得王座权限 |

### 穿越者的以太适应

穿越者的原世界可能有不同的本源能量（如元素力），需要进行适应：

**穿越者的适应阶段**：

| 阶段 | 适应状态 | 系统作用 | 身体反应 |
|------|---------|---------|---------|
| **穿越初期** | 不适应以太 | 生成系统，提供适应指引 | 身体虚弱，排斥反应（结晶纹路） |
| **适应期** | 逐渐适应以太 | 整合异界知识与本世界以太 | 结晶纹路消失，开始感知以太 |
| **掌控期** | 开始操控以太 | 提供进化方向选择 | 可以学习斗气或魔法 |
| **精通期** | 熟练操控以太 | 显示可进化方向 | 可以开创新术式 |
| **法则期** | 触碰法则层面 | 提供法则进化选择 | 可以干涉世界法则 |

**系统的适应机制**：

| 功能 | 说明 |
|------|------|
| **以太整合** | 帮助异界意识适应本世界以太，减轻排斥反应 |
| **知识转换** | 将异界知识转换为适应以太的知识 |
| **能力映射** | 将原世界的力量概念映射到以太体系 |
| **进化引导** | 基于以太本质，提供适合的进化方向 |
| **法则保护** | 在极端情况下，系统可以提供临时的法则保护 |

**穿越者的优势**：

由于原世界可能有不同的本源能量体系，穿越者在适应以太后可能具有特殊优势：

| 优势 | 说明 |
|------|------|
| **多元视角**：见过不同的能量体系，对以太有不同视角的理解 |
| **创新术式**：可以融合不同能量体系的理念，开创新的以太术式 |
| **独特进化**：异界知识可以拓宽可选择的进化方向 |
| **系统辅助**：系统帮助适应，比原住民更容易掌控以太 |

### 以太风暴带

环绕大陆边缘的狂暴以太区域，是世界的"边界"。穿越风暴带会到达"另一边"——一个被观测者支配的领域。

黑岛位于风暴带边缘，风暴不是偶发天气，而是会持续数日到数月的边界期。风暴持续时，晶纹、源晶和观测反馈都会被逼醒；白铃罗盘会变慢，因此半成品常利用风暴期藏匿、训练或逃亡。

## 势力分布

### 进化教派（The Cult of Evolution）

**核心信仰**：制造"完美生命"

**历史**：
- 起源：古代炼金术师试图人工制造"神"，却远达不到神的层次，直到三神开始对世界进行干预，真正授予 6 个因果席位给当世最强者
- 鼎盛："成功体"中有人成功进入王阶
- 衰落：成功体反叛，教派分裂
- 现状：地下活动，继续进行人体实验，试图再次造神，却意外引渡了很多异界意识体，也造出了很多天灾级的怪物，因此被大部分势力排斥

**实验体分类**：
- **成品**：稳定融合源晶，获得超常能力，并被教派精神约束
- **半成品**：融合部分成功，但精神控制失败或残缺；拥有价值，也是不稳定因素，教派中对这类实验体的处置方式存在分歧
- **失败**：排斥反应死亡，或异常融合成为怪物
- **异常**：外来意识占据失败体身体，或意识融合

### 六王座（The Six Thrones）

六王不是单纯的六个强者，而是六个被世界承认的"因果席位"。

**王座规则**：
- 王座不是王国的椅子，而是一段被世界承认的规则权限；坐上王座的人会获得对应领域的因果优先级
- 王座可以被继承、篡夺、空置或污染；持有者死亡不等于王座消失
- 每个王座代表一种极端规则，六席互相制衡，维持世界不被某一种理念吞没
- 王座会选择能把对应规则推到极端的人；血统、种族、国家只是外壳

**六王座结构**：

| 层次 | 王座 | 核心规则 | 持有者 | 状态 |
|------|------|---------|--------|------|
| **基础层** | 万灵王座 | 生命必须延续，万生万物生生不息 | 无 | 争夺中 |
| | 白骨王座 | 生命的终结是白骨，万物归死 | 莫特 | 亡者国度 |
| **构建层** | 熔炉王座 | 世界必须有序，熔炉锻造一切 | 伊格 | 灰烬帝国 |
| | 铁血王座 | 力量与征服，铁血毁灭一切 | 无 | 争夺中（瓦伦是主要争夺者之一） |
| **超越层** | 千衍王座 | 时空记录，见证一切，但无法干涉 | 维德 | 无固定势力 |
| | 虚无王座 | 规则之外，一切都被取消 | 诺缇 | 刚登座 |

**王座争夺的意义**：

王座争夺不是为了单纯的权力或统治，而是为了**争夺理念的极致**：

- 每个王座代表一个极致的理念
- 争夺者必须证明自己的理念"最正确"
- 王座会选择能把对应理念推到极端的人
- 争夺方式是战争——战争是证明理念的最终方式

**王座与王阶的关系**：

| 事实 | 说明 |
|------|------|
| **王座不是王阶的必要条件** | 王阶可以通过其他途径达到，但非常困难 |
| **有王座＝王阶先天优势** | 有王座的王阶拥有法则掌控权，成长速度和上限远高于无王座者 |
| **无王座王阶的特殊地位** | 无王座的王阶被称为"伪王"，拥有王阶实力但不被世界承认，无法触及法则层面 |
| **法则掌控的本质** | 王座赋予持有者对特定法则的掌控权，是"因果席位"的权限 |

**王阶与法则的关系**：

| 王阶 | 法则掌控能力 | 王座持有者 | 无王座者 |
|------|----------|----------|---------|
| **传奇** | 无法触碰法则 | 可以开创新术式，但无法干涉因果 | 可以开创新术式 |
| **王** | 触碰因果律层面 | 拥有法则掌控权，可以直接干涉特定法则 | 极难触及，几乎不可能 |
| **神** | 超越因果律 | 成为法则本身，可以重写世界法则 | 不可能达到 |

**无王座成为王阶的案例**（极罕见）：
- 需要获得大量的法则碎片
- 需要极度特殊的条件和机缘
- 需要付出巨大的代价或诅咒
- 即使成功，也不被世界正式承认，是"伪王"

**阵营分布**：
- **创造阵营**（2席）：万灵王座（生）、熔炉王座（创造）
- **毁灭阵营**（2席）：白骨王座（死）、铁血王座（毁灭）
- **超越阵营**（2席）：千衍王座（时空记录）、虚无王座（规则之外）

**王座争夺的格局**：

王座争夺不是简单的势力对抗，而是一个复杂的博弈：

| 夺杂的关系 | 说明 |
|-----------|------|
| **伊格与莫特的秘密合作** | 表面：伊格派遣锻火军团对抗亡灵，莫特劫持战场；秘密：伊格为莫特提供战场，莫特为伊格提供亡灵情报 |
| **瓦伦的背叛** | 争夺铁血王座的多个势力中，有一个是伊格的亲信，但他是创造者选中的种子，最终会背叛伊格 |
| **三神干预** | 创造者和毁灭者在暗中引导——创造者暗中帮助瓦伦，毁灭者暗中帮助某个争夺者 |
| **观测者的干扰** | 观测者通过千衍王座干扰争夺，试图补全自己的本质 |
| **虚无王座的变量** | 诺缇和露希在暗中破坏双方的计划，让争夺局势不可预测 |

**王座争夺的真相**：

王座争夺的真相是：**创造者和毁灭者正在暗中引导战争，为击杀观测者做准备。**

| 阶段 | 创造者的暗中引导 | 毁灭者的暗中引导 |
|------|-----------------|----------------|
| **争夺万灵王座** | 暗中支持某个争夺者，让万灵王座归属于创造者阵营的种子 | 暗中破坏其他争夺者，确保万灵王座不会落入毁灭者阵营 |
| **争夺铁血王座** | 暗中帮助瓦伦，让他成为双王持有者（熔炉+铁血） | 暗中支持某个争夺者，制造与瓦伦的对立 |
| **最终目的** | 创造者和毁灭者的种子联手，击杀观测者 | 同左 |

**虚无王座的异常**：

虚无王座原本被归入毁灭阵营，却不是稳定的统治权，而是世界给"规则之外的变量"预留的伤口。多年前虚无王座持有者（真名"露希"，自抹除后无人知晓）将自身存在抹除，隐匿于因果之外，实力达到准神层次。各国史书对虚无王座的记录互相矛盾：有人说虚无王座已死，有人说虚无王座从未存在。实则是因为持有者一直在暗中筹备击杀三神的计划，但实力始终差一线，因此只能隐匿以防被三神注视。

近期，虚无王座有了新的持有者——诺缇（Noctis，意为"属于夜的"），他登上王座后利用虚无法则抹除了身上的观测者标记，开始与前任持有者露希（Lucis，意为"属于光的"）合作。

### 人类国度：灰烬帝国

灰烬帝国是大陆最大的人类政权，也是主角离开风暴带后进入的主要舞台。

**表面秩序**：
- 以人类为统治主体，创造之以太被视为正统。
- 帝国宣传"十二王座维持世界平衡"，但实际只承认对帝国有利的王座合法性
- 贵族、教会、军团和炼金院共同维持统治，彼此制衡，也彼此出卖

**真实矛盾**：
- 灰烬皇帝伊格持有熔炉王座，操持创造以太却有着毁灭倾向；他以"锻造完美文明"为名反复清洗帝国
- 帝国内部有人想借进化教派的力量争夺万灵王座，来反抗伊格的独裁
- 进化教派残党躲在帝国炼金院和边境军团中，继续寻找能承载王座的试验体
- 铁血王座空置，多方势力暗中角逐，伊格也试图染指 |

### 进化方向

强大的生命在变强时可以选择自己的进化方向。其中部分人选择了克苏鲁神话方向：

| 角色 | 进化方向 | 说明 |
|------|---------|------|
| **观测者** | 尤格·索托斯方向 | 同时存在于所有时间和空间中，记录一切，渴望全知全能 |
| **千衍王座（维德）** | 尤格·索托斯方向 | 观测者的投影，时空记录者 |
| **虚无王座（诺缇、露希）** | 阿撒托斯方向 | 代表规则之外的混沌，是一切存在的终局 |

## 法则碎片与规则漏洞

### 上世代大战的遗产

在上世代三神大战和王座陨落中，世界法则被打散成无数碎片：

| 类型 | 特点 | 位置 |
|------|------|------|
| **法则碎片** | 携带部分法则能力的以太结晶 | 散落各地，可能埋藏在地底或生物体内 |
| **法则残留** | 打散的法则能量，形成以太异常 | 王座陨落地、战场遗址 |
| **规则漏洞** | 世界法则被打散后形成的漏洞 | 以太风暴带、古代遗迹、时空裂缝附近 |
| **因果裂隙** | 因果链被打断的地方 | 上世代战场、王座陨落地 |

### 法则碎片的价值

| 碎片类型 | 能力 | 获得方式 |
|---------|------|---------|
| **创造碎片** | 强化治愈、创造生命、提升以太亲和 | 探险遗迹、挖掘、击杀特定怪物 |
| **毁灭碎片** | 强化破坏、腐蚀、爆炸能力 | 探险危险区域、击杀特定怪物 |
| **观测碎片** | 增强感知、预知、记录能力 | 极难获得，可能隐藏在古代文献中 |
| **虚空碎片** | 获得规则外能力，暂时突破规则 | 极其危险，可能来自虚无法则本身 |

### 规则漏洞的利用

**规则漏洞的表现**：
- 某些地区或时期以太规则会失效
- 特定条件下术式效果会扭曲或增强
- 因果链可能被临时切断

**规则漏洞的风险**：
- 漏洞会被世界法则修复，时间不定
- 漏洞区域以太极不稳定，可能导致疯狂或变异
- 利用漏洞可能引起观测者的注意

## 以太异常区域

### 以太污染区

王座争斗或实验失败的地区，以太被扭曲：

| 类型 | 特点 | 危险等级 |
|------|------|----------|
| **轻度污染** | 以太颜色或性质改变，生物略微变异 | 低，可短期进入 |
| **中度污染** | 以太结构扭曲，生物产生异变 | 中，需要防护 |
| **重度污染** | 以太完全扭曲，形成以太怪物 | 高，极度危险 |
| **以太荒原** | 以太几乎消失，无法使用术式 | 特殊，只能靠肉体生存 |

### 以太怪物

被以太扭曲的生命或人造物：

| 类型 | 特点 | 危险性 |

### 进化系统

#### 进化的本质

**进化是生命本质的重塑，本质上是重塑生命中的以太结构**：

| 进化层级 | 本质 | 以太变化 |
|---------|------|---------|
| **基础进化** | 强化身体或能力 | 以太脉络的强化或扩展 |
| **领域进化** | 形成或强化领域 | 以太范围和密度的扩大 |
| **法则进化** | 触碰法则层面 | 以太与法则产生共鸣 |

#### 进化触发

当生命体的以太强度达到特定水平时，会触发进化机会：

| 位阶 | 进化触发条件 | 进化选择权 |
|------|------------|----------|
| **大师 → 传奇** | 精通某一系术式，创造独有术式 | 主动选择进化方向 |
| **传奇 → 王** | 开创独有术式，触碰因果律层面 | 主动选择进化方向 |
| **王 → 神** | 触碰因果律层面，获得王座或类似权限 | 主动选择进化方向 |

#### 原住民进化

- **进化方式**：被动进化
- **进化方向**：由生存环境决定，自动适应环境
- **进化过程**：
  1. 在特定环境中长期生存
  2. 身体和能力逐渐适应环境
  3. 达到进化触发条件时，自动获得与环境匹配的进化
- **限制**：原住民无法主动选择进化方向，只能被动适应

**例子**：
- 长期生活在以太风暴带的居民 → 进化为以太抗性体
- 长期在地下生存的矿工 → 进化为暗视、抗压体
- 长期在寒冷地区生存的居民 → 进化为耐寒体

#### 异界生命体进化

- **进化方式**：自选进化（在匹配度范围内）
- **进化方向**：在"匹配度合理"的范围内主动选择
- **匹配度机制**：
  1. 系统评估异界生命体的特质（共鸣能力、知识储备、性格倾向等）
  2. 系统列出"匹配度合理"的进化方向
  3. 异界生命体从列表中主动选择
  4. 进化完成后，获得该方向对应的能力和特性 |

**匹配度评估**：
| 评估维度 | 权重 | 说明 |
|---------|------|------|
| 共鸣能力 | 30% | 异界生命体的共鸣能力决定能承受的进化强度 |
| 知识储备 | 25% | 异界生命体的知识和经验影响可选择的进化方向 |
| 性格倾向 | 20% | 性格与进化方向的匹配程度 |
| 环境适应 | 15% | 当前环境与进化方向的兼容性 |
| 特殊资质 | 10% | 某些特殊资质会提高某些方向的匹配度 |

**进化选择界面**：

```
【进化匹配度评估】
当前位阶：行者
共鸣能力：S+（穿越者特质加成）
知识储备：A（异界知识）
性格倾向：B
环境适应：C
特殊资质：A

可选择的进化方向：
  基础进化：
    [A] 强化身体素质（匹配度95%）→ 力量+10，敏捷+10
    [B] 提高以太亲和（匹配度90%）→ 以太操控+15%
    [C] 增强感知能力（匹配度88%）→ 感知范围+20%
  领域进化：
    [D] 战斗直觉领域（匹配度72%）→ 战斗中时间变慢
  法则进化：
    [E] 吞噬法则（匹配度68%）→ 击杀生命后汲取力量

注意：选择匹配度不足的进化方向可能导致进化失败或产生副作用。
```

**进化方向分类**：

| 类型 | 特点 | 匹配度要求 |
|------|------|----------|
| **基础进化** | 强化身体能力、元素亲和 | 低，大部分人可获得 |
| **领域进化** | 获得某个领域的掌控能力 | 中，需要相关知识储备 |
| **法则进化** | 触碰世界法则，获得法则碎片亲和 | 高，需要极高共鸣能力 |
| **神明进化** | 模仿某个神明或神话存在的进化方向 | 极高，需要特殊条件和机缘 |
| **虚空进化** | 超越本世界规则的进化 | 未知，需要获得王座或类似权限 |

#### 世界法则系统

所有穿越者都拥有"系统"，这是世界法则对异界意识的自动响应机制。

**系统的来源**：
- 系统来自世界法则本身，不是观测者或其他神创造的
- 当异界意识进入这个世界时，世界法则会自动生成一个"适配器"——即系统
- 系统的本质是：**帮助异界意识适应世界法则，防止世界因异界意识而崩溃**

**系统的功能**：
1. **进化匹配度评估**：显示可选择的进化方向和匹配度
2. **状态监控**：显示身体、灵魂、能力的状态
3. **任务指引**：根据世界法则的引导提供任务（不是强制执行）
4. **知识整合**：帮助整合异界知识和本世界知识
5. **法则保护**：在极端情况下，系统可以提供临时的法则保护

**系统的限制**：
1. **无法创造法则**：系统只能利用现有法则，无法创造新的
2. **无法干涉王座**：系统无法直接干涉王座争夺
3. **无法对抗神明**：系统的能力远低于神明
4. **会被观测者看到**：观测者可以看到系统，但不能直接修改
5. **可以被抹除**：持有王座或达到足够实力后，可以主动抹除系统

**所有穿越者都有系统**：

| 穿越者类型 | 系统状态 | 特点 |
|-----------|---------|------|
| **进化教派实验体** | 完整系统 | 系统会标注试验体编号，提供实验相关信息 |
| **神明召唤者** | 完整系统 | 系统会显示召唤者的神明信息 |
| **时空裂缝意外者** | 完整系统 | 系统会显示穿越来源的不确定性 |
| **古代仪式召唤者** | 完整系统 | 系统会显示仪式相关信息 |

**系统抹除**：

当穿越者达到足够实力后，可以选择抹除系统：

| 条件 | 抹除方式 | 结果 |
|------|---------|------|
| **登上王座** | 利用王座力量主动抹除 | 系统消失，但保留已获得的能力和知识 |
| **达到神阶** | 超越世界法则后自动消失 | 系统消失，已经不需要系统 |
| **特殊仪式** | 执行某些特殊仪式 | 系统消失，但可能有副作用 |

**系统与观测者的关系**：

| 关系 | 说明 |
|------|------|
| **可见性** | 观测者可以看到系统，但不能直接修改 |
| **干扰性** | 观测者可以通过梦境、直觉等手段干扰系统显示的信息 |
| **利用性** | 观测者可以利用系统信息来引导穿越者 |
| **防御性** | 系统无法防御观测者的观测，但可以提示穿越者 |

| **进化教派实验** | 主动或被动参与实验 | 身体可能不稳定，能力可能受限 | 害怕被视为"试验品" |

#### 穿越者的共同特点

- **隐瞒身份**：绝大多数穿越者不会向外人提及自己的来历
- **隐瞒原因**：
  1. 被视为"异常"，可能被追捕或利用
  2. 被当作"试验品"，可能被再次关押
  3. 被视为"天选者"，可能引起不必要的注意
  4. 被当作"异端"，可能被审判
- **相互识别**：穿越者之间存在某种"共鸣"，可以互相识别
- **共守秘密**：穿越者之间有一条不成文的规则：不主动暴露其他穿越者

#### 穿越者的分布

| 地区 | 穿越者密度 | 主要类型 |
|------|----------|---------|
| **黑岛** | 较高 | 进化教派失败体、意外穿越者 |
| **灰烬帝国** | 中等 | 神明召唤者、古代仪式召唤者 |
| **亡者国度周边** | 较低 | 时空裂缝意外者 |
| **以太风暴带边缘** | 极低 | 时空裂缝意外者 |

#### 穿越者与原住民的差异

| 特征 | 穿越者 | 原住民 |
|------|-------|-------|
| **进化方式** | 可自选（在匹配度范围内） | 被动，由环境决定 |
| **知识来源** | 来自异界的知识 | 来源于本世界的学习 |
| **能力体系** | 可能混合两套体系 | 只能使用本世界体系 |
| **世界观** | 可能有不同世界观 | 认可本世界的世界观 |
| **灵魂特质** | 异界特质，可能有特殊效果 | 本世界特质 |

### 术式分类

- **下位**：基础强化、元素操控
- **中位**：复合术式、领域展开
- **上位**：因果干涉、时间扭曲
- **神级**：创造/毁灭/观测 法则
