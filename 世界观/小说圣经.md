# 伊瑟尔大陆世界设定

## 核心概念：以太（Ether）

以太是世界的血液，是因果的载体，掌控者可以依靠以太，根据自己的主观意识引动世界法则。

### 三种以太

| 类型 | 特性 | 代表 |
|------|------|------|
| **创造之以太** | 温和、滋养、有序 | 人类主流 |
| **毁灭之以太** | 狂暴、破坏、混乱 | 魔物、兽人 |
| **观测之以太** | 中性、渗透、记录 | 混血、源晶 |

### 以太风暴带

环绕大陆边缘的狂暴以太区域，是世界的"边界"。穿越风暴带会到达"另一边"——一个被观测者支配的领域。

黑岛位于风暴带边缘，风暴不是偶发天气，而是会持续数日到数月的边界期。风暴持续时，晶纹、源晶和观测反馈都会被逼醒；白铃罗盘会变慢，因此半成品常利用风暴期藏匿、训练或逃亡。

## 种族与混血

### 主要种族

- **人类**：创造之以太亲和，社会结构完整，歧视混血
- **哥布林**：适应性强，杂食，社会地位低下
- **暗精灵**：毁灭之以太亲和，夜视，被主流社会排斥
- **矮人**：工业种族，以太矿石开采者

## 势力分布

### 进化教派（The Cult of Evolution）

**核心信仰**：制造"完美生命"

**历史**：
- 起源：古代炼金术师试图人工制造"神"，却远达不到神的层次，直到三神开始对世界进行干预，真正授予 6 个因果席位给当世最强者
- 鼎盛："成功体"中有人成功进入王阶
- 衰落：成功体反叛，教派分裂
- 现状：地下活动，继续进行人体实验，试图再次造神，却意外引渡了很多异界意识体，也造出了很多天灾级的怪物，因此被大部分势力排斥

**实验体分类**：
- **成品**：稳定融合源晶，获得超常能力，并被教派精神约束
- **半成品**：融合部分成功，但精神控制失败或残缺；拥有价值，也是不稳定因素，教派中对这类实验体的处置方式存在分歧
- **失败**：排斥反应死亡，或异常融合成为怪物
- **异常**：外来意识占据失败体身体，或意识融合

### 六王座（The Six Thrones）

六王不是单纯的六个强者，而是六个被世界承认的"因果席位"。

**王座规则**：
- 王座不是王国的椅子，而是一段被世界承认的规则权限；坐上王座的人会获得对应领域的因果优先级
- 王座可以被继承、篡夺、空置或污染；持有者死亡不等于王座消失
- 每个王座代表一种极端规则，六席互相制衡，维持世界不被某一种理念吞没
- 王座会选择能把对应规则推到极端的人；血统、种族、国家只是外壳

**六王座结构**：

| 层次 | 王座 | 核心规则 | 持有者 | 状态 |
|------|------|---------|--------|------|
| **基础层** | 万灵王座 | 生命必须延续，万生万物生生不息 | 无 | 争夺中 |
| | 白骨王座 | 生命的终结是白骨，万物归死 | 莫特 | 亡者国度 |
| **构建层** | 熔炉王座 | 世界必须有序，熔炉锻造一切 | 伊格 | 灰烬帝国 |
| | 铁血王座 | 力量与征服，铁血毁灭一切 | 无 | 争夺中 |
| **超越层** | 千衍王座 | 时空记录，见证一切，但无法干涉 | 维德 | 无固定势力 |
| | 虚无王座 | 规则之外，一切都被取消 | 诺缇 | 刚登座 |

**王座争夺的意义**：

王座争夺不是为了单纯的权力或统治，而是为了**争夺理念的极致**：

- 每个王座代表一个极致的理念
- 争夺者必须证明自己的理念"最正确"
- 王座会选择能把对应理念推到极端的人
- 争夺方式是战争——战争是证明理念的最终方式

**阵营分布**：
- **创造阵营**（2席）：万灵王座（生）、熔炉王座（创造）
- **毁灭阵营**（2席）：白骨王座（死）、铁血王座（毁灭）
- **超越阵营**（2席）：千衍王座（时空记录）、虚无王座（规则之外）

**王座争夺的格局**：

王座争夺不是简单的势力对抗，而是一个复杂的博弈：

| 夺杂的关系 | 说明 |
|-----------|------|
| **伊格与莫特的秘密合作** | 表面：伊格派遣锻火军团对抗亡灵，莫特劫持战场；秘密：伊格为莫特提供战场，莫特为伊格提供亡灵情报 |
| **瓦伦的背叛** | 争夺铁血王座的多个势力中，有一个是伊格的亲信，但他是创造者选中的种子，最终会背叛伊格 |
| **三神干预** | 创造者和毁灭者在暗中引导——创造者暗中帮助瓦伦，毁灭者暗中帮助某个争夺者 |
| **观测者的干扰** | 观测者通过千衍王座干扰争夺，试图补全自己的本质 |
| **虚无王座的变量** | 诺缇和露希在暗中破坏双方的计划，让争夺局势不可预测 |

**王座争夺的真相**：

王座争夺的真相是：**创造者和毁灭者正在暗中引导战争，为击杀观测者做准备。**

| 阶段 | 创造者的暗中引导 | 毁灭者的暗中引导 |
|------|-----------------|----------------|
| **争夺万灵王座** | 暗中支持某个争夺者，让万灵王座归属于创造者阵营的种子 | 暗中破坏其他争夺者，确保万灵王座不会落入毁灭者阵营 |
| **争夺铁血王座** | 暗中帮助瓦伦，让他成为双王持有者（熔炉+铁血） | 暗中支持某个争夺者，制造与瓦伦的对立 |
| **最终目的** | 创造者和毁灭者的种子联手，击杀观测者 | 同左 |

**虚无王座的异常**：

虚无王座原本被归入毁灭阵营，却不是稳定的统治权，而是世界给"规则之外的变量"预留的伤口。多年前虚无王座持有者（真名"露希"，自抹除后无人知晓）将自身存在抹除，隐匿于因果之外，实力达到准神层次。各国史书对虚无王座的记录互相矛盾：有人说虚无王座已死，有人说虚无王座从未存在。实则是因为持有者一直在暗中筹备击杀三神的计划，但实力始终差一线，因此只能隐匿以防被三神注视。

近期，虚无王座有了新的持有者——诺缇（Noctis，意为"属于夜的"），他登上王座后利用虚无法则抹除了身上的观测者标记，开始与前任持有者露希（Lucis，意为"属于光的"）合作。

### 人类国度：灰烬帝国

灰烬帝国是大陆最大的人类政权，也是主角离开风暴带后进入的主要舞台。

**表面秩序**：
- 以人类为统治主体，创造之以太被视为正统。
- 帝国宣传"十二王座维持世界平衡"，但实际只承认对帝国有利的王座合法性
- 贵族、教会、军团和炼金院共同维持统治，彼此制衡，也彼此出卖

**真实矛盾**：
- 灰烬皇帝伊格持有熔炉王座，操持创造以太却有着毁灭倾向；他以"锻造完美文明"为名反复清洗帝国
- 帝国内部有人想借进化教派的力量争夺万灵王座，来反抗伊格的独裁
- 进化教派残党躲在帝国炼金院和边境军团中，继续寻找能承载王座的试验体
- 铁血王座空置，多方势力暗中角逐，伊格也试图染指

### 三神（The Three Gods）

| 神明 | 属性 | 状态 |
|------|------|------|
| **创造者** | 创造之以太 | 沉睡，力量衰退 |
| **毁灭者** | 毁灭之以太 | 沉睡，力量衰退 |
| **观测者** | 观测之以太 | 活跃，相对最强 |

## 地理

### 黑岛（Black Isle）

位于以太风暴带边缘的岛屿，是"被世界抛弃者"的聚集地。

- **主城**：人类统治，歧视混血
- **城外**：混血、流浪者、逃犯的营地
- **地下**：进化教派废弃试验场（诺缇醒来的地方）
- **悬崖**：通往以太风暴带的入口

### 以太风暴带（Ether Storm Belt）

环绕大陆的海洋区域，以太浓度极高，普通人进入会疯狂或死亡。

## 能力体系

### 位阶

| 位阶 | 描述 | 数量 |
|------|------|------|
| **学徒** | 刚觉醒以太感知 | 大量 |
| **行者** | 能使用基础术式 | 较多 |
| **大师** | 精通某一系术式 | 较少 |
| **传奇** | 开创独有术式 | 极少 |
| **王** | 触碰因果律层面 | 6人 |
| **神** | 超越因果律 | 3人 |

### 术式分类

- **下位**：基础强化、元素操控
- **中位**：复合术式、领域展开
- **上位**：因果干涉、时间扭曲
- **神级**：创造/毁灭/观测 法则
