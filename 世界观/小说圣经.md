# 小说圣经

## 一、核心设定

### 1. 世界观真相
- 《伊瑟尔》不是游戏，是上古者创造的"筛选器"
- 玩家以为是游戏，实际是意识投影到试炼场
- 陈诺通关时被判定为"高适格者"，肉身+意识全部拉入
- 成为第427号观测者，前426人大多死亡或疯狂

### 2. 力量体系

#### 三大进化阵营
| 阵营 | 特点 | 代表种族 | 进化方向 | 限制 |
|------|------|----------|----------|------|
| **创世者** | 以太温和，易于操控 | 精灵、矮人、龙族 | 智力↑、魔法亲和↑ | 进化慢，易结晶化 |
| **毁灭者** | 以太狂暴，难以控制 | 哥布林、兽人、魔物 | 肉体强化↑、进化快 | 理智下降，寿命短 |
| **观测者** | 以太中性，可塑性强 | 人类、混血种 | 多样性↑、潜力无限 | 需引导才能突破 |

#### 八元素
| 元素 | 特点 | 代表颜色 |
|------|------|----------|
| 水 | 治愈、变化 | 蓝 |
| 火 | 破坏、爆发 | 红 |
| 风 | 速度、感知 | 青 |
| 土 | 防御、坚韧 | 褐 |
| 光 | 神圣、净化 | 金 |
| 暗 | 隐匿、诅咒 | 紫 |
| 生命 | 生长、恢复 | 绿 |
| 时空 | 稀有、禁忌 | 银 |

#### 以太侵蚀
| 阶段 | 症状 | 结果 |
|------|------|------|
| 初期 | 皮肤结晶化，微光流动 | 可逆，需特殊药剂 |
| 中期 | 内脏结晶化，力量增强 | 难以治愈，可能失控 |
| 后期 | 全身晶体化，意识模糊 | 死亡或转化为晶灵 |

#### 进化触发条件
| 类型 | 方式 | 结果 |
|------|------|------|
| 自然进化 | 长期吸收以太、生死突破 | 可预测，稳定 |
| 引导进化 | 观测者介入、源晶催化 | 不可预测，可能突破限制 |
| 强制进化 | 进化教派实验、暴力灌注 | 大概率失控或死亡 |

### 3. 混血禁忌
```
创世者以太（温和） + 毁灭者以太（狂暴） = 体内战争

结果：
- 一方胜利，偏向某阵营（加尔）
- 脆弱平衡，随时崩溃（诺提出生时结晶化）
- 产生"新东西"（诺提的观测者引导）
```

### 4. 能力体系：因果观测
- **既视感**：关键时刻感到"这一幕我见过"
- **因果收束**：直觉感知选择的大致结果
- **残响（死亡回溯）**：死亡后因果倒带，以为是噩梦
- **观测污染**：过度使用会看到"不该看的"

### 5. 理智值系统（隐性）
```
100-80：正常
79-50：偶尔幻觉（噩梦细节在现实中出现）
49-20：频繁幻觉，分不清既视感与真实
19-0：分不清现实，可能伤害盟友
```

### 6. 称号体系
| 等级 | 称号 | 人数 | 描述 |
|------|------|------|------|
| 凡阶 | 学徒/正式/精英 | 数百万 | 普通冒险者、士兵 |
| 中位 | 队长/大师 | 数十万 | 一城之精英 |
| 上位 | 传奇/英雄 | 数千人 | 一国支柱 |
| 将阶 | 半神/天灾 | 数百人 | 大陆知名 |
| **王阶** | **王者/霸主** | **12人** | **世界规则制定者** |
| **神阶** | **上古者/彼方** | **3人** | **世界运行核心** |

## 四、神阶三人（传说）

### 创世者
- 创造了世界，但已"沉睡"
- 世界是其"梦境"

### 毁灭者
- 创世者的"影子"
- 不是敌人，是"必要的终结"
- 当世界"腐败"到某个程度，会醒来

### 观测者
- 不是创世也不是毁灭，只是"看着"
- 据说选择了诺提作为"代行者"
- 但可能只是因果的幻觉

## 七、地理设定
- **黑岛**：被诅咒的海域，暗精灵不敢靠近，主角避难所

## 八、进化教派
- 本质：崇拜进化本身，认为生命应突破限制
- 方法：强制灌注以太、混血实验、牺牲个体
- 与观测者关系：误解为选择强者，实际观测者选择可能性

## 十、关键剧情节点

### 第一卷：觉醒
- 穿越、苏醒、适应
- 建立与加尔、格林的羁绊
- 发现"既视感"的存在
- 被黑城接纳，开始训练
- **结尾**：听到加尔要杀自己的对话（或幻觉）

### 第二卷：崩坏
- 诺提开始怀疑一切
- 既视感越来越清晰，越来越恐怖
- **弑父事件**
- 诺提的冷血化完成
- **结尾**：发现"进化教派"的存在，凯恩出现

### 第三卷：观测者
- 凯恩训练诺提控制"能力"
- 诺提发现能力真相
- 面对选择：复仇还是保护他人
- **结尾**：第一次主动使用能力修改因果，付出巨大代价
