# 世界观核心设定 v4.0

## 一、以太源晶是什么？

**本质**：观测者（三神之一）创造的"锚点"，用于连接不同世界

**来源**：
- 远古时代，观测者为了"观察"其他世界，创造了源晶作为"眼睛"
- 源晶散落在各个世界（包括地球），以不同形式存在
- 在《伊瑟尔》游戏中，源晶是"通关奖励"——实际上是观测者设置的"陷阱"

**作用**：
- 捕获濒死或死亡的灵魂
- 将灵魂投射到伊瑟尔世界
- 与宿主身体融合，赋予特殊能力

**关键设定**：
- 源晶不是"系统"，没有界面、没有任务、没有奖励
- 它是"被动的"——只在关键时刻触发（濒死、极端情绪）
- 宿主会获得"因果观测"能力，但不知道如何使用

## 二、陈诺死前在干什么？

**时间**：2024年，地球

**身份**：22岁大学生，沉迷《伊瑟尔》游戏

**死前状态**：
- 连续通宵3天，身体虚弱
- 通关最后副本，获得"以太源晶"
- 心脏骤停（过劳+兴奋+源晶触发）

**源晶触发过程**：
1. 陈诺拿到源晶（游戏内物品）
2. 源晶感应到宿主濒死，激活"投影协议"
3. 陈诺的灵魂被抽离，身体死亡
4. 灵魂被压缩、重组，投射到伊瑟尔世界

**关键**：陈诺不是"穿越到游戏角色身上"，而是"灵魂被塞进了试验体尸体"

## 三、试验场为什么会塌？

**背景**：
- 进化教派在黑岛地下建立了"第7试验场"，制造混血试验体
- 目的是创造"适格者"——能承载源晶碎片的容器
- 427号试验体（诺提的身体原主）植入源晶碎片后死亡
- 但陈诺的灵魂被源晶捕获，塞进了这具尸体

**坍塌原因**：
- 教派内部派系斗争："销毁派" vs "回收派"
- 销毁派认为试验失败，引爆炸药毁灭证据
- 回收派想保留数据，但来不及阻止
- 诺提醒来时，正是销毁派引爆炸药后，回收派撤离时

**尸体堆积原因**：
- 试验场长期丢弃失败试验体
- 通风管道连接"弃尸坑"——悬崖下的凹陷
- 教派定期清理，但最近因为派系斗争，无人处理

## 四、穿越到游戏中的什么时期？

**时间线**：
- 《伊瑟尔》游戏时间：2024年（地球时间）
- 伊瑟尔世界时间：**游戏剧情结束后300年**

**关键设定**：
- 游戏里的"现在"，是世界历史上的"古代"
- 诺提穿越到的，是"未来"——一个游戏里没有记载的时代
- 这个时代：
  - 12王已经存在
  - 三神中的创造者、毁灭者已经沉睡
  - 观测者相对最强
  - 进化教派是地下势力

**为什么不是游戏时期？**
- 源晶投射需要时间——300年是"缓冲期"
- 观测者需要等待"因果线"稳定
- 这个时代更适合"适格者"成长（混乱、无序、力量真空）

## 五、会遇到玩家吗？玩家们是什么形式？

**答案：会。但不是你想的形式。**

**玩家类型**：

### 1. 陈诺型（灵魂投射）
- 死亡/濒死时，被源晶捕获灵魂
- 投射到伊瑟尔世界，占据试验体/尸体/新生儿
- 保留前世记忆，但身体不是自己的
- **数量极少**——源晶选择条件苛刻

### 2. 觉醒型（本地觉醒）
- 伊瑟尔世界的原住民，某天突然"觉醒"前世记忆
- 以为自己穿越了，其实是源晶碎片触发了"记忆回溯"
- 他们的"前世"可能是另一个世界的投影
- **数量较多**——但大多数以为是精神病

### 3. 观测型（纯意识）
- 没有实体，以"幽灵"形式存在
- 可以附身、可以观察、不能长期控制
- 通常是源晶融合失败的产物
- **数量未知**——可能无处不在

### 4. 教造型（人工制造）
- 进化教派制造的"伪适格者"
- 植入源晶碎片，但灵魂不是外来的
- 能力不稳定，容易暴走
- **数量最多**——但大多数是失败品

**关键冲突**：
- 陈诺（427号）是"天然适格者"——灵魂+源晶+试验体，三者意外融合
- 教派想复制这个成功案例
- 其他"玩家"可能是竞争对手，也可能是盟友

## 六、主角为什么能快速适应？

**表面**：陈诺是游戏玩家，熟悉世界观

**实际**：
1. **源晶在帮他**——因果观测能力在潜意识层面引导他
2. **身体记忆**——试验体原主有生存本能，肌肉记忆残留
3. **极端环境**——不快速适应就死，没有缓冲期
4. **前世经验**——陈诺是《伊瑟尔》高玩，了解基础设定

**但适应是表面的**：
- 他能战斗、能逃跑、能计划
- 但他不知道自己为什么能——以为是"游戏经验"
- 实际上，很多"直觉"是源晶在引导
- 这种"被引导"的感觉，会在后期成为他的恐惧来源

## 七、核心悬念（未解之谜）

2. **源晶的最终目的是什么？** ——培养容器？还是制造武器？
3. **其他"玩家"在哪里？** ——陈诺不是唯一，但为什么没遇到？
4. **教派为什么制造混血？** ——混血和源晶的兼容性更好？
5. **风暴带另一边有什么？** ——观测者的"本体"？还是另一个世界？
