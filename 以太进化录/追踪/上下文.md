### 2026-06-03 诺缇×露希感情线重做

**核心基调**：伤痛文学，互相喜欢但从未可能在一起。光与夜无法共存的悲剧。

**露希侧**：
- 第一次见诺缇时他还是个孩子，她认出了观测者标记，本该离开却记住了他那一眼
- 数年暗中守护（地下城偏半寸、追杀者传讯失灵、以太暴走引导方向），从未现身
- 拒绝靠近的原因：她走的路没有回头路，不该让任何人跟着
- 三千年来唯一的私心是"想让他活着"

**诺缇侧**：
- 感知到有个看不见的人在注视，不是观测者的冰冷感，而是"远处窗户透出的灯光"
- 开始期待那个声音，压在很深的地方，告诉自己是对盟友的依赖
- 登上王座那一刻在因果缝隙中感知到她的轮廓，转头她已不在

**写法红线**：绝不直写"喜欢"或"爱"。通过行为暗示、感官细节、物理距离的精确控制来表达。青春懵懂而禁忌。

**变更文件**：露希-前虚无王座.md、诺缇-主角.md、关系.md

### 2026-06-03 金手指重做 + 双轨战力体系

**主角金手指重做**：删除"吞噬进化"，改为"以太操纵"——环境以太引导能力极强，但体内无法形成以太回路。前期只能走体修路线（肉搏），后期通过地下城寻找远古术式走出自己的路。

**双轨战力体系**：
- **位阶**（下位/中位/上位/传奇/王/神）= 战斗技巧和以太操控熟练度
- **生命强度**（数值）= 生命层次，由系统量化显示
- 两轨独立增长，生命强度差距产生压制效果（2倍/5倍/10倍）
- 进化方向是"容器"，有容量上限，触达上限必须再次进化

**体修与术士**：不常用以太术式的强者为体修，生命强度普遍高于同阶术士。诺缇被迫走体修路线，生命强度超过同阶术士。

**与拉佩斯的配合**：拉佩斯傀儡术式消耗恐怖，诺缇用环境以太补充其消耗。两人物理上难以分开。

**设计逻辑**：
- 金手指的弱点和强项是同一事物的两面（无法形成回路 → 以太只能外溢 → 环境操控强）
- 体修路线让"缺陷"变成"另一种优势"
- 双轨制让"位阶低但生命强度高"成为合理的战力描述
- 进化容器机制创造了"选弱方向短期强、选强方向长期强"的策略博弈

### 2026-06-04 生命强度增长机制重做

**核心问题**：旧机制"杀什么都能涨"导致逻辑漏洞——批量杀蚂蚁也能成神。

**新机制：以太洗礼**
- 生命强度增长的本质是身体在以太冲击下的适应性进化，不是"杀怪掉经验"
- **轻松击杀弱敌 = 零增长**（以太冲击不够，身体没有适应压力）
- **势均力敌的濒死战斗 = 中等增长**（对手生命强度需在3倍差距内）
- **本源碎片 = 极大增长**（最强洗礼形式）
- **系统任务 = 中等增长**（系统引导以太改造身体）
- 同类战斗有适应性衰减——连续打同一种怪，收益递减至零

**设计效果**：
- 杜绝"杀蚂蚁成神"
- 碎片是核心增长来源 → 系统引导探索地下城是主线
- 战斗需要势均力敌 → 主角必须不断面对新挑战
- 轻松击杀无意义 → 避免"刷怪升级"套路

**变更文件**：力量体系.md（重写生命强度增长段落）、细纲_第003-007章（统一使用"以太洗礼"框架）
