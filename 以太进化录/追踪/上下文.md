### 2026-06-03 诺缇×露希感情线重做

**核心基调**：伤痛文学，互相喜欢但从未可能在一起。光与夜无法共存的悲剧。

**露希侧**：
- 第一次见诺缇时他还是个孩子，她认出了观测者标记，本该离开却记住了他那一眼
- 数年暗中守护（地下城偏半寸、追杀者传讯失灵、以太暴走引导方向），从未现身
- 拒绝靠近的原因：她走的路没有回头路，不该让任何人跟着
- 三千年来唯一的私心是"想让他活着"

**诺缇侧**：
- 感知到有个看不见的人在注视，不是观测者的冰冷感，而是"远处窗户透出的灯光"
- 开始期待那个声音，压在很深的地方，告诉自己是对盟友的依赖
- 登上王座那一刻在因果缝隙中感知到她的轮廓，转头她已不在

**写法红线**：绝不直写"喜欢"或"爱"。通过行为暗示、感官细节、物理距离的精确控制来表达。青春懵懂而禁忌。

**变更文件**：露希-前虚无王座.md、诺缇-主角.md、关系.md

### 2026-06-03 金手指重做 + 双轨战力体系

**主角金手指重做**：删除"吞噬进化"，改为"以太操纵"——环境以太引导能力极强，但体内无法形成以太回路。前期只能走体修路线（肉搏），后期通过地下城寻找远古术式走出自己的路。

**双轨战力体系**：
- **位阶**（下位/中位/上位/传奇/王/神）= 战斗技巧和以太操控熟练度
- **生命强度**（数值）= 生命层次，由系统量化显示
- 两轨独立增长，生命强度差距产生压制效果（2倍/5倍/10倍）
- 进化方向是"容器"，有容量上限，触达上限必须再次进化

**体修与术士**：不常用以太术式的强者为体修，生命强度普遍高于同阶术士。诺缇被迫走体修路线，生命强度超过同阶术士。

**与拉佩斯的配合**：拉佩斯傀儡术式消耗恐怖，诺缇用环境以太补充其消耗。两人物理上难以分开。

**设计逻辑**：
- 金手指的弱点和强项是同一事物的两面（无法形成回路 → 以太只能外溢 → 环境操控强）
- 体修路线让"缺陷"变成"另一种优势"
- 双轨制让"位阶低但生命强度高"成为合理的战力描述
- 进化容器机制创造了"选弱方向短期强、选强方向长期强"的策略博弈

### 2026-06-03 设定补全与矛盾修复

**背景**：用户确认本书定位为"群体穿越 + 群像"，目标平台起点。世界观主线为"各国势力斗争 + 地下城攻克"，大背景为三神斗争。系统奖励主要来自系统任务。

**修改内容**：

1. **修正力量体系.md**：系统从"被动适配器"改为"任务驱动的成长引导器"，任务系统是穿越者成长的核心途径，与地下城攻克和世界本源回收直接挂钩。新增系统限制"任务有代价"。新增"穿越者的社会定位"段落，明确穿越者是"被争相招募但严加监控的战略资源"。

2. **新建地下城体系.md**：定义地下城为上古大战中世界本源沉入地底形成的异常空间。E~S 六级分类，五种遗迹类型，地下城结构（外围→核心→深层→本源室），探索流程与风险。

3. **新建各国势力.md**：六国势力设定——灰烬帝国（人类帝制）、铁脊矮人联邦（矮人技术联邦）、永夜王廷（暗精灵君主制）、风暴城邦联盟（商业自由城邦）、亡者国度（莫特的亡灵国度）、哥布林部落联邦（分散部落）。各国对地下城和穿越者的不同政策。

4. **更新背景设定.md**：新增"世界本源"概念（法则完整度，上古大战散落，沉入地下城），新增"世界法则的嘉奖"机制（以太强化/进化契机/法则碎片/法则感悟/因果印记）。

5. **更新进化教派.md**：新增"与地下城的关系"段落——教派实验失败产物成为地下城魔物来源之一，教派利用深层地下城进行禁忌实验，暗中回收本源用于"造神"。

6. **修正灰烬帝国.md**：原文"十二王座"改为"六王座"。

**设计逻辑**：
- "世界本源"概念将"地下城探索"与"三神大战背景"连接起来
- "系统任务驱动"使穿越者天然有探索地下城的动机，解决了"穿越者为什么行动"的问题
- 各国势力的不同政策为穿越者提供了多种选择，群像叙事有了不同的阵营舞台
- 进化教派与地下城的关联为"魔物来源"和"禁忌实验"提供了合理的叙事空间

### 2026-06-03 新增角色：拉佩斯

**新增角色**：拉佩斯·阿曼（Lapes Arman），灰烬帝国大公次子，穿越者（意识融合体），比诺缇早来约5年。

**角色定位**：前期核心角色，诺缇的第一位穿越者盟友。前世是黏土手办宅男（程序员梁培），猝死后被原身的禁忌炼金术召唤融合。岩石傀儡术式远超同阶，被视为帝国最有前途的年轻贵族。

**关键场景**：拉佩斯通过行为模式分析判断诺缇是穿越者，以贵族身份召见后用"你喜欢可口可乐还是百事"试探确认。随后传授自编《异世界生存守则》（十二条，槽点多），邀请诺缇加入帝国探索队。

**设计意图**：
- 作为诺缇的镜像——同为穿越者但类型不同（融合体vs占据体），身份稳定性形成对比
- 提供前期信息输入（生存守则、势力情报、穿越者生态）
- 群像叙事中代表"帝国体制内穿越者"的路径
- 黏土手办→岩石傀儡的能力设计，将前世爱好转化为异世界天赋

**文件**：`设定/角色/拉佩斯-穿越者挚友.md`，关系已更新至 `关系.md`

### 2026-06-03 开局核心设定确认

**诺缇在进化教派的经历**：
- 意识完全替换型穿越（意识占据体），穿越来时什么都不懂
- 在教派中装疯卖傻地活着，被判定为"失败实验体"
- 被安排清扫垃圾，靠捡垃圾残渣为食
- 实际上在利用这段时间学习本土语言、收集教派情报
- **文章开篇时已是穿越一年后**，诺缇仍在装傻
- 开篇当天：计划出逃

**开篇关键场景——露希第一次"看到"诺缇**：
- 诺缇最后一次捡垃圾时，感知到有什么东西在看他（露希）
- 露希认出了他身上的观测者标记，本该离开
- 但诺缇察觉到了她的注视——这让她觉得"这个家伙不对劲"
- 一个失败实验体，不应该能感知到准神级存在的窥视
- 露希因此加深了对他的印象（从"路过看到"变成"值得观察"）
- 这是"光与夜"关系的真正起点

**出逃后剧情**：诺缇出逃后被教派追捕

### 2026-06-03 以太操纵能力来源修正

**重要**：以太操纵不是生来就会的。诺缇穿越后发现能隐约感知以太流动，一直认为是教派源晶植入"失败"的残余副作用。一年来他偷偷练习对游离以太的引动——无法造成杀伤，但能顺着以太屏障的结构纹路找到薄弱点并撕开。他把这当作出逃计划的关键手段。

**开篇时的能力状态**：
- 能引动环境中的游离以太（一年练习的成果）
- 无法吸收以太（以太在他体内留不住）
- 对以太屏障/封印效果好，对活物几乎无杀伤力
- 他不知道这是真正的天赋，以为是"失败的副产品"

### 2026-06-04 生命强度增长机制重做

**核心问题**：旧机制"杀什么都能涨"导致逻辑漏洞——批量杀蚂蚁也能成神。

**新机制：以太洗礼**
- 生命强度增长的本质是身体在以太冲击下的适应性进化，不是"杀怪掉经验"
- **轻松击杀弱敌 = 零增长**（以太冲击不够，身体没有适应压力）
- **势均力敌的濒死战斗 = 中等增长**（对手生命强度需在3倍差距内）
- **本源碎片 = 极大增长**（最强洗礼形式）
- **系统任务 = 中等增长**（系统引导以太改造身体）
- 同类战斗有适应性衰减——连续打同一种怪，收益递减至零

**设计效果**：
- 杜绝"杀蚂蚁成神"
- 碎片是核心增长来源 → 系统引导探索地下城是主线
- 战斗需要势均力敌 → 主角必须不断面对新挑战
- 轻松击杀无意义 → 避免"刷怪升级"套路

**变更文件**：力量体系.md（重写生命强度增长段落）、细纲_第003-007章（统一使用"以太洗礼"框架）
