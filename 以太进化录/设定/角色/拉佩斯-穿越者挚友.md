## 核心设定

### 穿越经过

拉佩斯的原身是阿曼大公的次子，天资平庸但性格固执。原身偷偷研究禁忌炼金术，试图通过召唤异界灵魂来获得超越兄长的力量。实验"成功"了——梁培的灵魂被拉入这个世界，与原身意识融合。

融合后的拉佩斯保留了两方记忆：前世的程序员知识和黏土手办热情，今世的贵族礼仪和帝国常识。他没有诺缇那种"我是谁"的身份困惑，因为他明确知道"我是两个人融合后的新我"。

穿越后五年，拉佩斯凭借融合后的特殊资质，在岩石术式上展现出远超同阶的天赋。他没有急于暴露穿越者身份，而是按照自己编写的《异世界生存守则》稳步成长，最终成为帝国最被看好的年轻贵族之一。

### 岩石傀儡术式

拉佩斯的核心能力是**岩石傀儡**——一种特殊的以太术式，将岩石塑造成可操控的傀儡战斗体。

| 特性 | 说明 |
|------|------|
| **傀儡自主性** | 拉佩斯的傀儡不需要持续操控，可以独立执行简单指令，远超同阶术式的自主度 |
| **傀儡多样性** | 可以塑造不同形态的傀儡（人形、兽形、工具形），根据战斗需要切换 |
| **傀儡强度** | 岩石傀儡的物理强度远超同阶，被视为"中位阶的传奇级战力" |
| **弱点** | 傀儡怕高频振动（会碎裂），拉佩斯本人近战能力一般 |

**为什么这么强**：拉佩斯前世是黏土手办师，对"造型"和"结构"有超乎常人的理解。他用捏手办的思维来塑造岩石傀儡——每个关节怎么活动、重心怎么分布、结构怎么加固——这些前世的"手艺"在以太世界变成了战斗天赋。

**致命弱点：消耗**：傀儡术式的强大是有代价的——以太消耗极其恐怖。每维持一个傀儡就需要持续消耗以太，多个傀儡同时作战时消耗呈指数增长。拉佩斯中位阶的以太储量，最多维持三个傀儡全力战斗十分钟。这使他的傀儡术式成了"爆发型"手段——开几分钟就必须收手回气。

### 与诺缇的供能配合

拉佩斯发现诺缇的以太天赋后，意识到两人的能力可以完美互补：

| | 拉佩斯 | 诺缇 |
|--|--------|------|
| **强项** | 傀儡术式极其强大 | 环境以太引导能力极强 |
| **弱点** | 以太消耗恐怖，体内储量不够用 | 体内无法形成回路，无法使用任何术式 |
| **给对方的** | 提供战斗输出 | 从环境中抽取游离以太，补充拉佩斯的消耗 |

**配合机制**：诺缇从环境中引导游离以太，直接注入拉佩斯的以太回路。拉佩斯不需要自己吸收环境以太（他不擅长这个），只需要专注于傀儡操控。

**配合效果**：
- 拉佩斯的傀儡作战时间从十分钟延长到可持续作战
- 可以同时操控更多傀儡（从三个提升到五个甚至更多）
- 诺缇从"废物近卫"变成"团队核心供能者"

**戏剧性**：这个配合让两人在物理上难以分开——拉佩斯越依赖诺缇的供能，两人的关系就越紧密。拉佩斯对诺缇说的第一句吐槽是："你知道吗，你是我见过最贵的充电宝。"诺缇的回应："你知道吗，你是我见过最耗电的用电器。"

### 灰色戒指

拉佩斯左手无名指上的灰色戒指是唯一佩戴的饰品，是他自己用岩石魔法制成的，这枚戒指在以太环境中产生了微弱的变化，能储存以太。拉佩斯将其视为"前世的锚点"，从不摘下。

### 穿越者社交策略

拉佩斯乐于结交穿越者，但绝不冒进。他的策略是**先观察，再判断，最后试探**：

1. **识别阶段**：通过帝国情报网和自身观察，锁定可疑的穿越者候选
2. **评估阶段**：长期观察目标的行为模式、性格倾向、战斗力，判断是否构成威胁
3. **试探阶段**：只有确认对方不可能威胁到自己后，才会发起接触（如"可乐试探"）
4. **分流阶段**：根据对方的性格和能力，决定是深交还是安置

对于弱小、不喜争斗的穿越者，拉佩斯不会强迫他们加入自己的圈子，而是利用贵族身份帮他们找到合适的去处——某个偏远城镇的安静职位、一份不需要冒险的差事、一个安全的身份。他不求回报，只是觉得"既然都是穿越者，能帮一把就帮一把"。

这种做法源于他的守则第六条："找一个你信任的人。"他自己体会过孤立无援的痛苦，所以愿意成为别人"可以信任的那个人"——但前提是确认安全。

### 公众形象

拉佩斯在帝国臣民中很受爱戴。岩石术式天才、年轻有为的王子、待人温和的贵族——这些标签让他成为帝国上下的宠儿。民众喜欢他，贵族尊重他，连伊格都对他另眼相看。

但拉佩斯自己觉得这不算什么。他知道这一切的根基是"穿越者的优势"，而非真正的天赋。他有时会对诺缇说："他们喜欢的不是我，是这具身体的头衔和前世带来的见识。如果他们知道我是从另一个世界来的，态度会变的。"

## 异世界生存守则

拉佩斯根据自己的穿越经历编写了《异世界生存守则》，共十二条，在穿越者圈子中小范围流传：

> **拉佩斯的异世界生存守则（v3.2）**
>
> 第一条：永远不要暴露你是穿越者。我说永远。
> 第二条：这个世界没有电、没有网、没有外卖。接受它。
> 第三条：以太是万能的，但你不是。量力而行。
> 第四条：不要相信任何主动接近你的人，尤其是教派的。
> 第五条：系统任务必须做，但不要盲目做。活着比完成任务重要。
> 第六条：找一个你信任的人。一个就够了。没有的话，写日记。
> 第七条：尊重原住民。他们不是NPC，他们有家人、有梦想、会哭会笑。
> 第八条：多交朋友。在这个世界，人脉比技能更实用。
> 第九条：发展一个"安全的"爱好。它会救你的命（心理层面）。
> 第十条：永远保留一条退路。城市、国家、势力都可以换，命只有一条。
> 第十一条：记住你前世是谁，但不要被前世困住。你是新的你。
> 第十二条：如果以上守则互相矛盾，说明你遇到了需要随机应变的情况。祝你好运。

**评价**：穿越者们普遍认为这份守则"有用但槽点太多"——第一条和第八条矛盾（隐藏身份怎么交朋友？），第五条和第七条矛盾（不做任务怎么活？尊重原住民怎么利用人？），第十二条属于"说了等于没说"。拉佩斯对此的回应是："这不是教科书，这是经验总结。能帮你避开80%的坑，剩下20%靠你自己。"

## 与瓦伦的关系

瓦伦是拉佩斯的表哥——瓦伦的母亲是阿曼家族的人，嫁入帝国后生下瓦伦。从血缘上说，瓦伦和拉佩斯是表兄弟。

### 拉佩斯的视角

拉佩斯**害怕瓦伦**。

不是因为瓦伦对他不好——恰恰相反，瓦伦很喜欢这个表弟，经常主动找他玩、送他东西、在公开场合夸赞他。但瓦伦身上有一种让人本能畏惧的气场：传奇阶的压迫感、锻造征服的天赋气质、以及那双仿佛能看穿一切的眼睛。

拉佩斯每次和瓦伦相处都浑身不自在。瓦伦越热情，他越紧张。他不知道该怎么面对一个"明明很可怕但对你很好"的人。

| 状态 | 说明 |
|------|------|
| **表面** | 尊敬的表兄弟关系，帝国宴会上正常寒暄 |
| **拉佩斯的真实感受** | 每次瓦伦靠近都手心出汗，努力维持贵族微笑，内心疯狂祈祷"快走吧求你了" |
| **瓦伦的态度** | 真心喜欢这个表弟，觉得拉佩斯"有意思"，经常不请自来 |
| **最尴尬的时刻** | 瓦伦拍他肩膀说"我们是一家人"，拉佩斯差点原地石化 |

### 穿越者的双重负担

拉佩斯不仅要隐藏穿越者身份，还要应付瓦伦的热情。这对他来说是双重压力：
- 瓦伦的感知力极强，拉佩斯担心被看穿穿越者身份
- 瓦伦是传奇阶，拉佩斯的岩石傀儡在瓦伦面前不值一提
- 瓦伦的"亲近"让拉佩斯无法拒绝，但每次相处都如履薄冰

拉佩斯对诺缇吐槽过这件事："你知道最可怕的是什么吗？是一个对你太好的强者。你没法翻脸，没法逃跑，只能笑着承受。打游戏的时候好歹能关掉对话框，这个世界连个关闭按钮都没有。"

## 与诺缇的关系

### 初见

拉佩斯在帝国的情报网中注意到一个"异常"——进化教派实验区附近出现了一个不该存在的活动迹象。通过分析行为模式（极其谨慎、不与人接触、专门避开教派搜索队，对一切都充满疑惑与好奇），拉佩斯判断这个人**大概率是穿越者**。

他以帝国大公次子的身份"召见"了这个逃亡者。诺缇带着警惕赴约，做好了随时逃跑的准备。

召见的地点是拉佩斯的私人书房。拉佩斯屏退侍从，关上门，沉默了三秒——然后突然紧张地问：

"你喜欢可口还是百事？"

诺缇愣住了。

拉佩斯看到诺缇的反应，长舒一口气："好了，确认了。你是穿越者。原住民不可能有这种表情。"

这个试探方式后来被诺缇吐槽了无数次——"你就没想过万一我是个喜欢喝矿泉水的穿越者呢？"拉佩斯的回答是："那就再问你薯片喜欢什么口味。总有一款适合你。"

### 关系演变

| 阶段 | 状态 | 说明 |
|------|------|------|
| **初识** | 试探→确认→接纳 | "可口可乐还是百事"试探，确认穿越者身份，拉佩斯主动示好 |
| **信任建立** | 守则传授→共同探索 | 拉佩斯向诺缇介绍《异世界生存守则》，邀请诺缇参与地下城探索 |
| **深厚友谊** | 并肩作战→互相托付 | 经历多次地下城探索和势力斗争后，成为可以托付后背的挚友 |
| **分歧** | 路线分歧 | 拉佩斯倾向在帝国体制内生存，诺缇倾向独立行动 |
| **重逢** | 各自成长后的重逢 | 分歧不等于决裂，两人在更高层面重新理解彼此 |

### 两人的互补

| 维度 | 诺缇 | 拉佩斯 |
|------|------|--------|
| **穿越者类型** | 意识占据体（无原身记忆） | 意识融合体（双记忆） |
| **生存方式** | 独行、谨慎、不信任 | 社交、规划、建立人脉 |
| **战斗风格** | 近卫肉搏 + 环境以太供能 | 远程、岩石傀儡 |
| **性格** | 冷淡、内敛 | 温和、外向 |
| **弱点** | 信息不对称、孤立 | 过于依赖体系、低估阴谋 |

两人在一起时，诺缇负责战斗和直觉判断，拉佩斯负责情报分析和社交周旋。拉佩斯是诺缇第一个真正信任的人，也是唯一一个能让诺缇放下防备的人。

| 卷 | 拉佩斯的状态 |
