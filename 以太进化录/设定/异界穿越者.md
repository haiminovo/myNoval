# 异界穿越者设定

## 基本概念

异界穿越者是指来自"外界"（通常指现实世界或其他维度）的意识或生命体通过各种方式进入伊瑟尔大陆的存在。

---

## 穿越者的分类

### 按穿越来源分类

| 类型 | 占比 | 来源 | 特点 |
|------|------|------|------|
| **进化教派实验体** | 约60% | 主动或被动参与进化教派的实验 | 身体可能不稳定，能力可能受限 |
| **神明召唤者** | 约20% | 被神明或王座直接召唤 | 获得特殊能力，但也承担相应责任 |
| **时空裂缝意外者** | 约15% | 意外穿越时空裂缝 | 保留原世界能力，但需要适应新世界 |
| **古代仪式召唤者** | 约5% | 被古代仪式意外召唤 | 可能拥有被遗忘的知识或能力 |

### 按意识融合程度分类

| 类型 | 占比 | 特点 |
|------|------|------|
| **完全融合体** | 5% | 异界意识与身体完美融合，获得完整记忆和能力 |
| **部分融合体** | 30% | 异界意识与身体部分融合，记忆和能力部分保留 |
| **意识占据体** | 40% | 异界意识完全占据身体，获得异界者的全部能力 |
| **意识融合体** | 15% | 异界意识与原主意识融合，产生新的意识 |
| **失败体** | 10% | 排斥反应死亡，或成为无法控制的怪物 |

## 进化教派的实验

### 实验目的

- 制造"完美生命"
- 寻找能承载王座的试验体
- 探索异界意识的特殊性

### 实验过程

| 阶段 | 过程 | 成功率 |
|------|------|--------|
| **意识引渡** | 通过特殊仪式将异界意识引渡到试验体身体 | 约30% |
| **融合过程** | 异界意识与身体进行融合 | 约50%（在成功引渡后） |
| **稳定过程** | 稳定融合体，防止排斥反应 | 约70%（在成功融合后） |
| **控制过程** | 建立精神控制，确保实验体听话 | 约40%（在成功稳定后）|

### 实验结果

| 结果 | 占比 | 处理方式 |
|------|------|---------|
| **成功体（成品）** | 5% | 成为教派的核心战力，被精神控制 |
| **半成品** | 30% | 教派对其处置方式存在分歧，有的继续使用，有的清理 |
| **失败体（怪物）** | 10% | 清理或封印 |
| **逃亡者** | 5% | 逃出实验场，成为流浪者或被追捕者 |
| **死亡** | 50% | 作为废料处理 |

## 穿越者的共同特点

### 系统存在

所有穿越者都拥有"系统"，这是世界法则对异界意识的自动响应机制。

**系统的来源**：
- 系统来自世界法则本身，不是观测者或其他神创造的
- 当异界意识进入这个世界时，世界法则会自动生成一个"适配器"——即系统
- 系统的本质是：**帮助异界意识适应世界法则，防止世界因异界意识而崩溃**

**系统的功能**：
1. **进化匹配度评估**：显示可选择的进化方向和匹配度
2. **状态监控**：显示身体、灵魂、能力的状态
3. **任务指引**：根据世界法则的引导提供任务（不是强制执行）
4. **知识整合**：帮助整合异界知识和本世界知识
5. **法则保护**：在极端情况下，系统可以提供临时的法则保护

**系统的界面**：

```
【状态面板】
姓名：诺缇（Noctis）
穿越者类型：进化教派第427号试验体（意识占据体）
穿越时间：[已屏蔽]
身体状态：虚弱（需要营养）
观察推理能力：S+（穿越者特质加成） |

【进化评估】
当前位阶：行者
进化进度：45/100
可选择的进化方向：
  基础进化：
    [A] 强化身体素质（匹配度95%）→ 力量+10，敏捷+10
    [B] 提高以太亲和（匹配度90%）→ 以太操控+15%
    [C] 增强感知能力（匹配度88%）→ 感知范围+20%
  领域进化：
    [D] 战斗直觉领域（匹配度72%）→ 战斗中时间变慢
  法则进化：
    [E] 吞噬法则（匹配度68%）→ 击杀生命后汲取力量

【任务】
  [主线] 逃离废弃试验场（进度1/3）
    1. 找到离开试验场的路径
    2. 收集足够的食物和水
    3. 避免被发现

【知识整合】
  异界知识：[已整合]现代科技、[已整合]生存技能
  本世界知识：[未解锁]以太基础理论、[未解锁]术式使用
```

**系统的限制**：
1. **无法创造法则**：系统只能利用现有法则，无法创造新的
2. **无法干涉王座**：系统无法直接干涉王座争夺
3. **无法对抗神明**：系统的能力远低于神明
4. **会被观测者看到**：观测者可以看到系统，但不能直接修改
5. **可以被抹除**：持有王座或达到足够实力后，可以主动抹除系统
6. **任务导向生存**：系统任务不会给出"友善建议"，只会给出"生存压力"
7. **不鼓励合作**：系统不会鼓励合作，只会强调竞争和生存
8. **威胁感知**：系统会持续提示"其他异界意识体"的存在和威胁

**所有穿越者都有系统**：

| 穿越者类型 | 系统状态 | 特点 |
|-----------|---------|------|
| **进化教派实验体** | 完整系统 | 系统会标注试验体编号，提供实验相关信息 |
| **神明召唤者** | 完整系统 | 系统会显示召唤者的神明信息 |
| **时空裂缝意外者** | 完整系统 | 系统会显示穿越来源的不确定性 |
| **古代仪式召唤者** | 完整系统 | 系统会显示仪式相关信息 |

**系统抹除**：

当穿越者达到足够实力后，可以选择抹除系统：

| 条件 | 抹除方式 | 结果 |
|------|---------|------|
| **登上王座** | 利用王座力量主动抹除 | 系统消失，但保留已获得的能力和知识 |
| **达到神阶** | 超越世界法则后自动消失 | 系统消失，已经不需要系统 |
| **特殊仪式** | 执行某些特殊仪式 | 系统消失，但可能有副作用 |

**系统与观测者的关系**：

| 关系 | 说明 |
|------|------|
| **可见性** | 观测者可以看到系统，但不能直接修改 |
| **干扰性** | 观测者可以通过梦境、直觉等手段干扰系统显示的信息 |
| **利用性** | 观测者可以利用系统信息来引导穿越者 |
| **防御性** | 系统无法防御观测者的观测，但可以提示穿越者 |

### 隐瞒身份

绝大多数穿越者不会向外人提及自己的来历，原因如下：

1. **被视为"异常"**：穿越者被视为世界的"异常"，可能被追捕或利用
2. **被当作"试验品"**：如果暴露，可能被当作"试验品"再次关押
3. **被视为"天选者"**：被神明召唤的穿越者可能引起不必要的注意
4. **被当作"异端"**：可能被宗教组织视为异端进行审判

### 相互识别（黑暗森林法则）

- 诺缇无法通过任何特殊能力直接识别其他穿越者
- 诺缇只能通过观察他人的行为来判断对方是不是穿越者
- **判断穿越者的行为特征**：
  1. 对异界知识的反应（如看到现代科技时的奇怪反应）
  2. 对系统"提示"的反应（如突然查看虚空）
  3. 行为模式与原住民不同（如用"异界方式"解决问题）
  4. 对本世界的"陌生感"（如对常识的不了解）
  5. 系统任务完成后的异常反应（如获得知识后的突然沉默）
- **注意**：这种判断不是"能力"，而是"观察和推理"
- **黑暗森林法则**：系统不会提供任何穿越者的信息，诺缇只能自己观察和判断

### 系统任务（黑暗森林风格）

系统的任务列表不会给出友好的指引，而是生存本能的体现：

```
【任务】
  [生存] 击杀其他异界意识体（0/1）
    - 奖励：进化进度+20
    - 失败：你的位置会被其他异界意识体发现

  [生存] 击杀其他异界意识体（2/5）
    - 奖励：系统功能解锁
    - 失败：你的位置会被其他异界意识体发现
```

**系统任务的特点**：
- **生存导向**：所有任务都与生存有关，不是"帮助"而是"生存"
- **击杀任务**：系统会给出"击杀其他异界意识体"的任务
- **不提供位置**：系统不会提供其他穿越者的位置信息，诺缇需要自己去找
- **竞争压力**：系统不会鼓励合作，只会强调竞争和生存
- **任务失败后果**：系统不会提供关于其他异界意识体的任何信息

**系统界面风格（黑暗森林）**：

```
【状态面板】
姓名：诺缇（Noctis）
穿越者类型：进化教派第427号试验体（意识占据体）
穿越时间：[已屏蔽]
身体状态：虚弱（需要营养）

【任务】
  [生存] 击杀其他异界意识体（0/1）
    - 奖励：进化进度+20
    - 失败：你的位置会被其他异界意识体发现

【进化评估】
当前位阶：行者
进化进度：45/100
可选择的进化方向：
  基础进化：
    [A] 强化身体素质（匹配度95%）→ 力量+10，敏捷+10
    [B] 提高以太亲和（匹配度90%）→ 以太操控+15%
    [C] 增强感知能力（匹配度88%）→ 感知范围+20%
  领域进化：
    [D] 战斗直觉领域（匹配度72%）→ 战斗中时间变慢
  法则进化：
    [E] 吞噬法则（匹配度68%）→ 击杀生命后汲取力量
```

**系统的本质**：
- 系统不是"友好的助手"，而是"生存本能"
- 系统的任务不是"引导"，而是"生存压力"
- 系统的提示不是"友善建议"，而是"生存警告"
- 系统不会鼓励合作，只会强调竞争
- **系统不提供位置信息**：任务只说"击杀其他异界意识体"，不说"在哪里"
- **系统不提供识别能力**：诺缇无法通过系统直接识别其他穿越者

### 共守秘密

- 穿越者之间有一条不成文的规则：**不主动暴露其他穿越者**
- 这条规则的起源：
  1. 共同的风险：暴露一个穿越者可能引起连锁反应
  2. 互助的需要：穿越者需要互相帮助（但非常少）
  3. 保守秘密的默契：不暴露自己的同时，也不暴露他人
  4. **观察和判断**：不会直接询问对方是否是穿越者，而是通过观察对方的行为来判断
- **但并非所有穿越者都遵守**：
  1. 部分穿越者会主动残杀其他穿越者，消除竞争
  2. 部分穿越者会背叛其他穿越者，换取自身的安全
  3. 共守秘密只是"默认规则"，不是"绝对规则"
- **判断对方是否是穿越者的方法**：
  1. 对异界知识的反应（如看到现代科技时的奇怪反应）
  2. 对系统"提示"的反应（如突然查看虚空）
  3. 行为模式与原住民不同（如用"异界方式"解决问题）
  4. 对本世界的"陌生感"（如对常识的不了解）
  5. 系统任务完成后的异常反应（如获得知识后的突然沉默）

### 穿越者之间的敌对关系

穿越者之间的关系并非全是友好的：

| 关系类型 | 占比 | 特点 |
|---------|------|------|
| **隐藏合作** | 约30% | 互相提供隐秘帮助，共同保守秘密 |
| **敌对残杀** | 约25% | 主动残杀其他穿越者，消除竞争 |
| **冷漠无视** | 约25% | 互不干涉，各自为战 |
| **利用关系** | 约15% | 互相利用，最终可能背叛 |
| **友好联盟** | 约5% | 真正的盟友，共同面对挑战 |

**敌对残杀的原因**：
1. **消除竞争**：穿越者被视为"变量"，数量越少越好
2. **系统限制**：某些任务或能力需要"消除其他异界意识体"
3. **自身安全**：认为其他穿越者是威胁
4. **获得能力**：杀死穿越者可能获得某些特殊能力或收益

**残杀穿越者的收益**：
| 收益 | 说明 |
|------|------|
| **灵魂碎片** | 被杀穿越者的灵魂碎片可以用于强化自身 |
| **知识吸收** | 可能获得被杀穿越者的异界知识 |
| **系统奖励** | 系统可能提供特殊奖励（但极少） |
| **消除竞争** | 减少"变量"数量，降低暴露风险 |

**穿越者的认知阶段**：

大多数穿越者都会经历以下认知阶段：

| 阶段 | 认知 | 典型行为 |
|------|------|---------|
| **初期** | 以为自己是唯一的穿越者 | 不主动寻找其他穿越者，隐瞒身份 |
| **中期** | 通过观察发现对方行为异常，怀疑对方也是穿越者 | 保持警惕，谨慎接触，可能敌对 |
| **后期** | 明白穿越者之间的关系复杂 | 选择合作、敌对或无视，形成自己的立场 |

**判断对方是否是穿越者的方法**：

1. **观察对方对异界知识的反应**
   - 原住民对现代科技无反应
   - 穿越者可能表现出好奇或困惑

2. **观察对方对系统"提示"的反应**
   - 原住民没有系统，不会查看虚空
   - 穿越者可能突然查看虚空，然后恢复正常

3. **观察对方的行为模式**
   - 原住民的行为符合本世界逻辑
   - 穿越者可能用"异界方式"解决问题

4. **观察对方对本世界的"陌生感"**
   - 原住民对常识很熟悉
   - 穿越者可能对本世界的常识感到陌生

5. **观察对方完成任务后的反应**
   - 原住民完成任务后没有特殊反应
   - 穿越者完成任务后可能突然沉默或查看系统

**注意事项**：
- 这些判断方法不是100%准确，可能误判
- 原住民可能也有"奇怪行为"，需要仔细观察
- 穿越者可能伪装成原住民，需要深入观察

**主角诺缇的认知**：

| 阶段 | 认知 | 行为 |
|------|------|------|
| **初期** | 以为自己是唯一的穿越者 | 不主动寻找，完全隐瞒身份 |
| **中期** | 发现还有其他穿越者，但极少 | 保持警惕，不主动接触 |
| **后期** | 明白穿越者之间存在敌对关系 | 选择谨慎的盟友，避免被残杀 |

## 穿越者的能力体系

### 能力来源

| 来源 | 占比 | 特点 |
|------|------|------|
| **原世界能力** | 30% | 保留原世界的某些能力或知识 |
| **本世界能力** | 50% | 学习本世界的以太能力和术式 |
| **混合能力** | 20% | 两套能力体系混合使用 |

### 异界特质

穿越者通常拥有某些"异界特质"，这些特质是原住民不具备的：

| 特质 | 表现 | 影响 |
|------|------|------|
| **观察推理能力** | 能通过观察判断对方是否是穿越者 | 可以识别其他穿越者（需要仔细观察） |
| **知识保留** | 保留原世界的某些知识 | 可能对这个世界有不同视角 |
| **特殊直觉** | 对某些事情有特殊的直觉或预感 | 可能在关键时刻做出正确选择 |
| **适应性增强** | 对新环境的适应能力更强 | 能更快地学习本世界的技能 |
| **以太整合** | 将原世界的能量概念映射到以太 | 可以融合不同能量体系的理念，开创新的以太术式 |

### 穿越者与本源能量的关系

**平行宇宙穿越**：

穿越是"平行宇宙"之间的意识转移，而非空间穿越：

| 宇宙 | 本源能量 | 法则体系 | 特点 |
|------|---------|---------|------|
| **伊瑟尔大陆** | 以太 | 以太法则、王座规则 | 以太是世界的血液 |
| **穿越者原世界** | 各有不同 | 各有不同 | 每个宇宙都有自己的本源能量 |

**穿越的本质**：
- 意识从原宇宙转移到伊瑟尔大陆
- 原宇宙的记忆和知识会保留
- 但身体必须适应新的本源能量（以太）
- 系统是世界法则生成的"适配器"，帮助异界意识适应以太

**以太适应阶段**：

| 阶段 | 适应状态 | 身体反应 | 系统帮助 |
|------|---------|---------|---------|
| **穿越初期** | 不适应以太 | 结晶纹路、虚弱、排斥反应 | 生成系统，提供适应指引 |
| **适应期** | 逐渐适应以太 | 结晶纹路消失，开始感知以太 | 提供以太整合，减轻排斥 |
| **掌控期** | 开始操控以太 | 形成以太脉络，可以使用术式 | 显示可进化方向 |
| **精通期** | 熟练操控以太 | 可以开创新术式，形成领域 | 提供法则进化选择 |

**力量形式的以太本质**：

| 力量形式 | 以太本质 | 穿越（者理解 |
|---------|---------|----------|
| **斗气** | 以太凝聚在身体中 | 强化身体、爆发力量 |
| **魔法** | 以太由精神引导 | 元素操控、术式释放 |
| **进化** | 以太在存在层面的重塑 | 改变生命本质、获得特殊能力 |

**穿越者的进化优势**：

由于原世界可能有不同的本源能量体系，穿越者在适应以太后可能具有特殊优势：

| 优势 | 说明 |
|------|------|
| **多元视角**：见过不同的能量体系，对以太有不同视角的理解 |

## 穿越者与进化

### 进化选择权

穿越者与原住民最大的差异在于：穿越者可以在匹配度合理的范围内自选进化方向。

**匹配度评估**：
| 评估维度 | 权重 | 说明 |
|---------|------|------|
| 观察推理能力 | 30% | 穿越者的观察和推理能力通常高于原住民，可以承受更强的进化 |
| 知识储备 | 25% | 来自异界的知识可以拓宽可选择的进化方向 |
| 性格倾向 | 20% | 性格与进化方向的匹配程度 |
| 环境适应 | 15% | 当前环境与进化方向的兼容性 |
| 特殊资质 | 10% | 某些特殊资质（如穿越者的异界特质）会提高匹配度 |

### 进化方向的优势

| 优势 | 说明 |
|------|------|
| **可自选** | 在匹配度合理的范围内，穿越者可以主动选择进化方向 |
| **更多选择** | 由于异界知识的存在，穿越者的可选择进化方向通常更多 |
| **更快适应** | 穿越者的观察和推理能力通常更强，可能更快适应进化 |
| **更强效果** | 穿越者的进化效果通常比原住民更强 |

### 进化方向的风险

| 风险 | 说明 |
|------|------|
| **匹配度不足** | 如果选择了匹配度不足的进化方向，可能进化失败或产生副作用 |
| **不稳定** | 穿越者的身体可能不稳定，进化后可能更加不稳定 |
| **暴露风险** | 进化后可能引起注意，暴露穿越者身份 |

## 穿越者之间的互动

### 互动规则

1. **初次接触**：通过观察和推理判断对方可能是穿越者
2. **试探**：不会直接询问对方来历，而是通过对话试探
3. **确认**：如果确认对方是穿越者，会产生某种默契
4. **互助**：在不暴露自己的前提下，互相提供帮助
5. **不暴露**：不会向第三者透露对方是穿越者

### 常见对话

穿越者之间的对话通常带有某种"暗示"，不会直接谈论穿越：

- "你以前也是'那边'的吗？"（询问来源）
- "我之前也遇到过'和你类似的人'。"（暗示对方是穿越者）
- "有些东西，只有'懂的人'才懂。"（暗示共同的经历）
- "得小心，'他们'在找我们。"（指教派或其他追捕者）

### 冲突与合作

| 情况 | 处理方式 |
|------|---------|
| **利益冲突** | 不以穿越者身份施压，用常规方式解决 |
| **共同威胁** | 在不暴露身份的前提下合作 |
| **互相帮助** | 提供隐秘的帮助，避免引起注意 |
| **背叛** | 极少发生，因为背叛者会暴露所有人的秘密 |
