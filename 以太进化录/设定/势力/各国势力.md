# 各国势力

## 大陆势力格局

伊瑟尔大陆存在多个国家和势力，彼此之间围绕**地下城控制权、王座争夺、穿越者资源**展开复杂博弈。

## 主要国家

### 灰烬帝国

| 项目 | 内容 |
|------|------|
| **位置** | 大陆中央平原 |
| **主体种族** | 人类 |
| **统治者** | 灰烬皇帝伊格（熔炉王座持有者） |
| **政治体制** | 帝制，贵族-教会-军团-炼金院四权分立 |
| **核心力量** | 锻火军团、帝国炼金院 |
| **对以太的态度** | 创造之以太为正统，压制毁灭之以太 |

灰烬帝国是大陆最强的人类政权。伊格以"锻造完美文明"为名，实际上在为王座争夺布局。帝国拥有最多数量的注册穿越者，穿越者在帝国的地位是"被招募的特殊战力"——有编制、有待遇，但也有监控。

**地下城政策**：帝国实行"官方探索制"，所有地下城探索必须经帝国批准，穿越者编入国家探索队，战利品按比例上缴。

### 磐石国

| 项目 | 内容 |
|------|------|
| **位置** | 帝国东部山区 |
| **主体种族** | 人类 |
| **统治者** | 阿曼国王（灰烬皇帝伊格的亲兄弟） |
| **政治体制** | 王权制，名义独立，实际为帝国附庸 |
| **核心力量** | 岩石术式传承、山区地下城群 |
| **对以太的态度** | 偏重创造之以太的大地系分支 |

磐石国是灰烬帝国的附庸国，名义上独立，实际上国王是皇帝的亲兄弟。磐石国以岩石术式闻名，国内多山地和地下城群，是最活跃的地下城探索区域之一。国王的次子拉佩斯是穿越者（身份未公开），被视为磐石国最耀眼的天才。

**地下城政策**：磐石国拥有帝国最多的未攻克地下城，实行"自主探索+帝国监管"的混合制。王室成员有优先探索权。

### 铁脊矮人联邦

| 项目 | 内容 |
|------|------|
| **位置** | 大陆北部山脉 |
| **主体种族** | 矮人 |
| **统治者** | 铁王议会（七大家族轮值） |
| **政治体制** | 联邦制，家族自治 |
| **核心力量** | 以太锻造技术、地下城工程部队 |
| **对以太的态度** | 实用主义，三种以太都是资源 |

矮人是大陆最先进的以太技术种族。他们不关心以太的哲学意义，只关心如何用以太锻造更强的武器和装备。矮人拥有最成熟的地下城工程技术——他们不只是探索地下城，他们在**开采**地下城。

**地下城政策**：矮人实行"工程化开采"，将地下城视为矿产资源，系统性地挖掘、提炼、出售。矮人是最大的以太结晶出口国。

### 永夜王廷

| 项目 | 内容 |
|------|------|
| **位置** | 大陆西部森林 |
| **主体种族** | 暗精灵 |
| **统治者** | 永夜女王（世袭） |
| **政治体制** | 君主制，长老院辅政 |
| **核心力量** | 暗影术式、毁灭之以太操控 |
| **对以太的态度** | 毁灭之以太是自然的一部分，不应被歧视 |

暗精灵长期被人类社会排斥，退守西部森林建立了自己的国家。永夜王廷信奉毁灭之以太的自然循环——毁灭不是邪恶，而是新生的前提。他们与灰烬帝国长期对峙。

**地下城政策**：暗精灵倾向于"共生式探索"——不完全清除魔物，而是与地下城生态达成某种平衡，只取所需。这使他们的探索效率较低，但对地下城的破坏最小。

### 风暴城邦联盟

| 项目 | 内容 |
|------|------|
| **位置** | 大陆南部海岸 |
| **主体种族** | 人类、混血、各族混居 |
| **统治者** | 城主议会（各城自治） |
| **政治体制** | 松散联盟，商业驱动 |
| **核心力量** | 商业网络、探索者公会总部 |
| **对以太的态度** | 中立，以太是商品 |

风暴城邦联盟不是一个真正的国家，而是一群商业城邦的松散联合。这里没有统一的法律，只有商业规则。探索者公会总部设在这里，所有地下城探索的佣金、装备、情报交易都在此进行。

**地下城政策**：自由探索。任何人可以探索任何地下城，只要向公会注册并缴纳佣金。这里是穿越者最自由的地方，也是最危险的地方——没有法律保护，一切靠实力说话。

### 亡者国度

| 项目 | 内容 |
|------|------|
| **位置** | 大陆东部荒原 |
| **主体种族** | 亡灵、死灵法师 |
| **统治者** | 莫特（白骨王座持有者） |
| **政治体制** | 王权制，莫特的意志即法律 |
| **核心力量** | 白骨法则、亡灵军团 |
| **对以太的态度** | 毁灭之以太的终极形态——死亡即回归 |

亡者国度是大陆上最被恐惧的势力。莫特持有白骨王座，掌控死亡法则。亡者国度的居民大多是亡灵和死灵法师，他们不惧怕地下城的魔物——对他们来说，死亡不是终结，而是转化。

**地下城政策**：亡者国度将地下城视为"死亡资源"——地下城中死亡的魔物和探索者都可以被转化为亡灵战力。莫特与伊格有秘密合作，伊格为莫特提供战场，莫特为伊格提供亡灵情报。

### 哥布林部落联邦

| 项目 | 内容 |
|------|------|
| **位置** | 大陆各地（分散居住） |
| **主体种族** | 哥布林 |
| **统治者** | 无统一领导，各部落自治 |
| **政治体制** | 部落制 |
| **核心力量** | 数量优势、适应性 |
| **对以太的态度** | 生存本能驱动 |

哥布林是大陆上数量最多但地位最低的种族。他们分散居住在各地，没有统一的国家。但哥布林的适应性极强——在地下城环境中，哥布林的进化速度远超其他种族。

**地下城政策**：哥布林没有统一的地下城政策。有些部落在地下城外围捡拾残余资源，有些部落深入地下城与魔物共生，有些哥布林被各国雇佣作为地下城探索的炮灰。

## 势力关系

### 核心矛盾

| 矛盾 | 说明 |
|------|------|
| **帝国 vs 永夜王廷** | 人类与暗精灵的种族对立，创造之以太与毁灭之以太的哲学之争 |
| **帝国 vs 亡者国度** | 表面敌对，伊格与莫特秘密合作 |
| **矮人联邦 vs 所有人** | 矮人只做生意不站队，但以太技术使所有人忌惮 |
| **风暴城邦 vs 各国** | 城邦的自由政策威胁各国对穿越者的控制 |
| **哥布林 vs 所有人** | 数量庞大但被歧视，地下城探索中的廉价消耗品 |

### 穿越者争夺

各国争夺穿越者的核心逻辑：
- 穿越者有系统任务驱动，是最高效的地下城探索者
- 穿越者的自选进化能力使其成长速度远超原住民
- 穿越者完成系统任务获得的法则馈赠，部分会惠及所属势力

| 势力 | 穿越者政策 | 优势 | 劣势 |
|------|-----------|------|------|
| **灰烬帝国** | 登记制，编入国家探索队 | 资源充足、有编制 | 监控严格、自由度低 |
| **矮人联邦** | 雇佣制，按项目合作 | 报酬丰厚、装备精良 | 只看技术不看人 |
| **永夜王廷** | 少量接纳，平等对待 | 自由度高、尊重个体 | 数量少、资源有限 |
| **风暴城邦** | 自由注册，无约束 | 最自由 | 无保护、竞争残酷 |
| **亡者国度** | 强制转化或驱逐 | 莫特的力量 | 非亡灵者不愿前往 |
