# 进化教派

## 核心信仰

制造"完美生命"

## 历史

- 起源：古代炼金术师试图人工制造"神"，却远达不到神的层次，直到三神开始对世界进行干预，真正授予 6 个因果席位给当世最强者
- 鼎盛："成功体"中有人成功进入王阶
- 衰落：成功体反叛，教派分裂
- 现状：地下活动，继续进行人体实验，试图再次造神，却意外引渡了很多异界意识体，也造出了很多天灾级的怪物，因此被大部分势力排斥

## 与地下城的关系

进化教派与地下城有着深层关联：

| 关系 | 说明 |
|------|------|
| **魔物来源** | 教派实验失败的产物——失控的融合体、异变的怪物——被丢弃后流入地下城，成为魔物种群的一部分 |
| **实验场所** | 教派利用深层地下城的封闭环境和法则异常区域进行禁忌实验，躲避各国监控 |
| **本源争夺** | 教派暗中回收世界本源，不用于修复法则，而是用于"造神"实验——将本源力量注入实验体 |
| **情报网络** | 教派在各国地下城探索队中安插眼线，掌握最新的地下城情报和法则碎片流向 |

进化教派的地下城活动使其成为各国的**灰色地带**——各国明面上排斥教派，暗中却可能与教派交易地下城情报和实验成果。

## 实验体分类

- **成品**：稳定融合源晶，获得超常能力，并被教派精神约束
- **半成品**：融合部分成功，但精神控制失败或残缺；拥有价值，也是不稳定因素，教派中对这类实验体的处置方式存在分歧
- **失败**：排斥反应死亡，或异常融合成为怪物
- **异常**：外来意识占据失败体身体，或意识融合

## 异界意识实验

进化教派通过某种方式"引渡"异界意识进入这个世界：

| 实验体类型 | 占比 | 特点 |
|-----------|------|------|
| **完全融合体** | 5% | 异界意识与身体完美融合，获得完整记忆和能力 |
| **部分融合体** | 30% | 异界意识与身体部分融合，记忆和能力部分保留 |
| **意识占据体** | 40% | 异界意识完全占据身体，获得异界者的全部能力 |
| **意识融合体** | 15% | 异界意识与原主意识融合，产生新的意识 |
| **失败体** | 10% | 排斥反应死亡，或成为无法控制的怪物 |
