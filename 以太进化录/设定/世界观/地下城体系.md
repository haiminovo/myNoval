# 地下城体系

## 地下城的本质

地下城不是天然存在的洞穴，而是**上古大战中法则碎片和世界本源沉入地底后形成的异常空间**。本源力量不断吸引以太扭曲体聚集，经过漫长岁月，这些空间被魔物占据，形成了层层递进的危险区域。

- 地下城的核心是**沉睡的世界本源**
- 本源吸引以太变异体，魔物以本源为食，不断进化
- 攻克地下城 = 清除魔物 = 解放本源 = 获得世界法则嘉奖
- 本源被解放后，地下城不会消失，但会逐渐衰弱（魔物减少、危险等级下降）
- 被解放的本源会缓慢回归世界法则，但这个过程可能需要数十年

## 地下城分类

### 按危险等级

| 等级 | 名称 | 深度 | 危险程度 | 适合探索者 | 典型奖励 |
|------|------|------|---------|-----------|---------|
| **E** | 浅层遗迹 | 地表~50米 | 低 | 下位~中位 | 低纯度以太结晶、少量本源碎片 |
| **D** | 地下迷宫 | 50~200米 | 中低 | 中位 | 中纯度结晶、基础法则碎片 |
| **C** | 深层洞穴 | 200~500米 | 中 | 中位~上位 | 高纯度结晶、法则碎片 |
| **B** | 远古遗迹 | 500~1000米 | 高 | 上位~传奇 | 法则感悟、进化契机 |
| **A** | 禁地 | 1000米+ | 极高 | 传奇+ | 因果印记、法则馈赠 |
| **S** | 神域遗迹 | 未知 | 灾厄级 | 王阶 | 未知（从未被完全攻克） |

### 按遗迹类型

| 类型 | 来源 | 特点 | 常见魔物 |
|------|------|------|---------|
| **战场遗址** | 上古三神大战的战场 | 以太紊乱、法则碎片密集 | 战场亡灵、以太鬼 |
| **古代城市** | 上古文明的城市废墟 | 结构复杂、陷阱机关多 | 以太兽群、变异守卫 |
| **神殿遗迹** | 上古文明供奉王座的建筑 | 法则异常、幻象丛生 | 神殿守卫、法则傀儡 |
| **矿坑深渊** | 上古以太采矿场 | 以太浓度极高、变异生物多 | 以太元素、矿脉兽 |
| **封印之地** | 上古封印危险存在的场所 | 极度危险、禁忌知识 | 被封印的存在、守封者 |

## 地下城结构

每个地下城都有类似的结构：

```
地表入口
    ↓
外围区（E-D级）：魔物稀疏，资源有限
    ↓
核心区（C-B级）：魔物密集，有法则碎片
    ↓
深层区（A级）：法则异常，有远古守卫
    ↓
本源室（S级）：世界本源沉睡之处，有最强魔物守护
```

### 本源室

每个地下城的最深处都有一个"本源室"，是世界本源沉睡之处：
- 本源室被最强大的魔物守护
- 解放本源需要击败守护者或完成特殊条件
- 本源被解放后，该地下城的危险等级会逐渐下降
- 但解放过程可能引发连锁反应（以太风暴、魔物暴走等）

## 地下城探索

### 探索组织

| 组织形式 | 说明 | 特点 |
|---------|------|------|
| **国家探索队** | 各国官方组织的地下城探索队 | 装备精良、纪律严明，但受政治制约 |
| **自由探索者** | 独立行动的探索者或小队 | 灵活高效，但资源有限 |
| **探索者公会** | 跨国的探索者组织 | 提供任务、装备和情报，抽取佣金 |
| **黑市探索者** | 未注册的非法探索者 | 不受规则约束，但无法律保护 |

### 探索流程

1. **情报收集**：了解地下城等级、魔物类型、地形结构
2. **队伍组建**：根据地下城特点搭配人员（战斗、治疗、侦查、后勤）
3. **外围探索**：清理外围魔物，建立前进基地
4. **核心推进**：逐步深入，收集资源和法则碎片
5. **本源解放**：击败守护者，解放世界本源
6. **撤退回收**：携带战利品安全撤出

### 探索风险

| 风险 | 说明 |
|------|------|
| **魔物暴走** | 深层魔物感知到入侵者后可能集体暴走 |
| **以太风暴** | 地下城内部的以太可能突然紊乱，导致术式失控 |
| **法则异常** | 某些区域的法则可能扭曲，物理规则不再适用 |
| **队伍背叛** | 争夺战利品时队友可能反目（尤其在穿越者之间） |
| **迷路困死** | 地下城结构可能动态变化，不熟悉地形会迷失 |
| **封印崩塌** | 触碰禁忌可能导致被封印的存在苏醒 |

## 地下城与世界的关系

### 本源回收循环

```
世界法则完整 → 以太稳定 → 魔物减少 → 地下城衰弱
        ↑                                    ↓
  本源回归 ← 解放本源 ← 击败魔物 ← 探索地下城
```

这个循环是世界自我修复的机制。但三神大战造成的裂隙太深，仅靠自然修复需要数万年。穿越者的出现加速了这个过程——世界法则生成系统，引导穿越者主动回收本源。

### 地下城的"活"性

地下城不是静态的：
- **会生长**：新的本源碎片可能沉入地底，形成新的地下城
- **会变化**：魔物的活动会改变地下城的结构
- **会联动**：某些地下城之间存在地下通道，攻克一个可能影响另一个
- **会衰弱**：本源被解放后，地下城会逐渐失去活力

### 各势力对地下城的态度

| 势力 | 态度 | 目的 |
|------|------|------|
| **各国政府** | 积极探索，但控制节奏 | 获取资源、增强国力、控制本源分配 |
| **探索者公会** | 中立，提供服务 | 赚取佣金、维持秩序 |
| **进化教派** | 暗中渗透 | 获取本源用于实验、在地下城进行禁忌研究 |
| **穿越者** | 最活跃的探索者 | 完成系统任务、获取法则馈赠、生存竞争 |
