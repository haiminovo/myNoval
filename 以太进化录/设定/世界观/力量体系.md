# 力量体系

## 核心概念：以太（Ether）

以太是世界的血液，是因果的载体，掌控者可以依靠以太，根据自己的主观意识引动世界法则。

### 三种以太

| 类型 | 特性 | 代表 |
|------|------|------|
| **创造之以太** | 温和、滋养、有序 | 人类主流 |
| **毁灭之以太** | 狂暴、破坏、混乱 | 魔物、兽人 |
| **观测之以太** | 中性、渗透、记录 | 混血、源晶 |

### 以太与力量的关系

所有力量形式本质上都是以太的不同表现：

| 力量形式 | 本质 | 特点 |
|---------|------|------|
| **斗气** | 以太在身体中的凝聚 | 强化身体、爆发力量 |
| **魔法** | 以太在精神中的引导 | 元素操控、术式释放 |
| **进化** | 以太在存在层面的重塑 | 改变生命本质、获得特殊能力 |
| **领域** | 以太在空间的展开 | 控制一定范围内的以太流动 |
| **法则** | 以太的终极形态 | 直接干涉世界法则，触碰因果 |

**以太的作用机制**：
1. **斗气**：以太凝聚在身体各处，形成"以太脉络"，强化肉体
2. **魔法**：以太由精神引导，形成"以太回路"，释放术式
3. **进化**：以太渗透到存在层面，重塑生命本质
4. **领域**：以太以自身为中心向外展开，控制范围内以太流动
5. **法则**：以太达到极高密度，可以直接干涉世界法则

### 原住民的以太适应

原住民天生适应以太：

| 阶段 | 适应程度 | 能力 |
|------|---------|------|
| **出生** | 天然适应以太环境 | 可以感知以太，但无法操控 |
| **觉醒** | 开始操控以太 | 学习斗气或魔法的基础 |
| **掌控** | 熟练操控以太 | 可以使用中下位术式，形成以太脉络 |
| **精通** | 精通某种以太操控 | 可以开创新术式，形成领域 |
| **王阶** | 触碰法则层面 | 可以直接干涉世界法则，获得王座权限 |

### 穿越者的以太适应

穿越者的原世界可能有不同的本源能量（如元素力），需要进行适应：

**穿越者的适应阶段**：

| 阶段 | 适应状态 | 系统作用 | 身体反应 |
|------|---------|---------|---------|
| **穿越初期** | 不适应以太 | 生成系统，提供适应指引 | 身体虚弱，排斥反应（结晶纹路） |
| **适应期** | 逐渐适应以太 | 整合异界知识与本世界以太 | 结晶纹路消失，开始感知以太 |
| **掌控期** | 开始操控以太 | 提供进化方向选择 | 可以学习斗气或魔法 |
| **精通期** | 熟练操控以太 | 显示可进化方向 | 可以开创新术式 |
| **法则期** | 触碰法则层面 | 提供法则进化选择 | 可以干涉世界法则 |

**系统的适应机制**：

| 功能 | 说明 |
|------|------|
| **以太整合** | 帮助异界意识适应本世界以太，减轻排斥反应 |
| **知识转换** | 将异界知识转换为适应以太的知识 |
| **能力映射** | 将原世界的力量概念映射到以太体系 |
| **进化引导** | 基于以太本质，提供适合的进化方向 |
| **法则保护** | 在极端情况下，系统可以提供临时的法则保护 |

**穿越者的优势**：

由于原世界可能有不同的本源能量体系，穿越者在适应以太后可能具有特殊优势：

| 优势 | 说明 |
|------|------|
| **多元视角**：见过不同的能量体系，对以太有不同视角的理解 |
| **创新术式**：可以融合不同能量体系的理念，开创新的以太术式 |
| **独特进化**：异界知识可以拓宽可选择的进化方向 |
| **系统辅助**：系统帮助适应，比原住民更容易掌控以太 |

### 以太风暴带

环绕大陆边缘的狂暴以太区域，是世界的"边界"。穿越风暴带会到达"另一边"——一个被观测者支配的领域。

## 能力体系

### 位阶

| 位阶 | 描述 | 数量 | 以太状态 |
|------|------|------|---------|
| **下位** | 刚觉醒以太感知 | 大量 | 开始感知以太，体内形成初步以太脉络，无法引动外界以太 |
| **中位** | 能使用基础术式 | 较多 | 以太回路完善，可引导外界以太以太释放术式 |
| **上位（大师）** | 精通某一系术式 | 较少 | 以太密度大幅提升，可高速施法 |
| **传奇** | 开创独有术式 | 极少 | 以太达到临界密度，自发引动游离以太，防御型术式自发形成，可开创新术式 |
| **王** | 触碰因果律层面 | 6人 | 以太与法则直接共鸣，可干涉世界法则，触碰法则层面，术式附带部分规则型力量或判定|
| **神** | 超越因果律 | 3人 | 超越法则，创造法则 |

### 力量形式的以太本质

所有力量形式本质上都是以太的不同表现：

| 力量形式 | 以太本质 | 表现 | 形成条件 |
|---------|---------|------|---------|
| **斗气** | 以太凝聚在身体中 | 强化身体、爆发力量 | 通过呼吸吐纳，以太在身体中形成循环 |
| **术式** | 以太的特定排列 | 元素操控、复合效果 | 将以太按特定排列，形成稳定的效果 |
| **领域** | 以太向外展开 | 控制范围内以太流动 | 将以太以自身为中心向外扩散 |
| **法则** | 以太与法则共鸣 | 干涉世界法则、触碰因果 | 以太达到极高密度，直接与法则共鸣 |

### 进化系统

#### 进化的本质

**进化是生命本质的重塑，本质上是重塑生命中的以太结构**：

| 进化层级 | 本质 | 以太变化 |
|---------|------|---------|
| **基础进化** | 强化身体或能力 | 以太脉络的强化或扩展 |
| **领域进化** | 形成或强化领域 | 以太范围和密度的扩大 |
| **法则进化** | 触碰法则层面 | 以太与法则产生共鸣 |

#### 原住民进化

- **进化方式**：被动进化
- **进化方向**：由生存环境决定，自动适应环境
- **进化过程**：
  1. 在特定环境中长期生存
  2. 身体和能力逐渐适应环境
  3. 达到进化触发条件时，自动获得与环境匹配的进化
- **限制**：原住民无法主动选择进化方向，只能被动适应

**例子**：
- 长期生活在以太风暴带的居民 → 进化为以太抗性体
- 长期在地下生存的矿工 → 进化为暗视、抗压体
- 长期在寒冷地区生存的居民 → 进化为耐寒体

#### 异界生命体进化

- **进化方式**：自选进化（在匹配度范围内）
- **进化方向**：在"匹配度合理"的范围内主动选择
- **匹配度机制**：
  1. 系统评估异界生命体的特质（观察推理能力、知识储备、性格倾向等）
  2. 系统列出"匹配度合理"的进化方向
  3. 异界生命体从列表中主动选择
  4. 进化完成后，获得该方向对应的能力和特性 |

**匹配度评估**：
| 评估维度 | 权重 | 说明 |
|---------|------|------|
| 观察推理能力 | 30% | 异界生命体的观察和推理能力决定能承受的进化强度 |
| 知识储备 | 25% | 异界生命体的知识和经验影响可选择的进化方向 |
| 性格倾向 | 20% | 性格与进化方向的匹配程度 |
| 环境适应 | 15% | 当前环境与进化方向的兼容性 |
| 特殊资质 | 10% | 某些特殊资质会提高某些方向的匹配度 |

#### 世界法则系统

所有穿越者都拥有"系统"，这是世界法则对异界意识的自动响应机制。

**系统的来源**：
- 系统来自世界法则本身，不是观测者或其他神创造的
- 当异界意识进入这个世界时，世界法则会自动生成一个"适配器"——即系统
- 系统的本质是：**帮助异界意识适应世界法则，防止世界因异界意识而崩溃**

**系统的功能**：
1. **进化匹配度评估**：显示可选择的进化方向和匹配度
2. **状态监控**：显示身体、灵魂、能力的状态
3. **任务指引**：根据世界法则的引导提供任务（不是强制执行）
4. **知识整合**：帮助整合异界知识和本世界知识

**系统的限制**：
1. **无法创造法则**：系统只能利用现有法则，无法创造新的
2. **无法干涉王座**：系统无法直接干涉王座争夺
3. **无法对抗神明**：系统的能力远低于神明
4. **会被观测者看到**：观测者可以看到系统，但不能直接修改

#### 穿越者的共同特点

- **隐瞒身份**：绝大多数穿越者不会向外人提及自己的来历
- **隐瞒原因**：
  1. 被视为"异常"，可能被追捕或利用
  2. 被当作"试验品"，可能被再次关押
  3. 被视为"天选者"，可能引起不必要的注意
  4. 被当作"异端"，可能被审判
- **相互识别**：穿越者无法直接识别彼此，只能通过观察和推理判断
- **共守秘密**：穿越者之间有一条不成文的规则：不主动暴露其他穿越者

#### 穿越者与原住民的差异

| 特征 | 穿越者 | 原住民 |
|------|-------|-------|
| **进化方式** | 可自选（在匹配度范围内） | 被动，由环境决定 |
| **知识来源** | 来自异界的知识 | 来源于本世界的学习 |
| **能力体系** | 可能混合两套体系 | 只能使用本世界体系 |
| **世界观** | 可能有不同世界观 | 认可本世界的世界观 |
| **灵魂特质** | 异界特质，可能有特殊效果 | 本世界特质 |

## 法则碎片与规则漏洞

### 上世代大战的遗产

在上世代三神大战和王座陨落中，世界法则被打散成无数碎片：

| 类型 | 特点 | 位置 |
|------|------|------|
| **法则碎片** | 携带部分法则能力的以太结晶 | 散落各地，可能埋藏在地底或生物体内 |
| **法则残留** | 打散的法则能量，形成以太异常 | 王座陨落地、战场遗址 |
| **规则漏洞** | 世界法则被打散后形成的漏洞 | 以太风暴带、古代遗迹、时空裂缝附近 |
| **因果裂隙** | 因果链被打断的地方 | 上世代战场、王座陨落地 |

### 法则碎片的价值

| 碎片类型 | 能力 | 获得方式 |
|---------|------|---------|
| **创造碎片** | 强化治愈、创造生命、提升以太亲和 | 探险遗迹、挖掘、击杀特定怪物 |
| **毁灭碎片** | 强化破坏、腐蚀、爆炸能力 | 探险危险区域、击杀特定怪物 |
| **观测碎片** | 增强感知、预知、记录能力 | 极难获得，可能隐藏在古代文献中 |
| **虚空碎片** | 获得规则外能力，暂时突破规则 | 极其危险，可能来自虚无法则本身 |

### 规则漏洞的利用

**规则漏洞的表现**：
- 某些地区或时期以太规则会失效
- 特定条件下术式效果会扭曲或增强
- 因果链可能被临时切断

**规则漏洞的风险**：
- 漏洞会被世界法则修复，时间不定
- 漏洞区域以太极不稳定，可能导致疯狂或变异
- 利用漏洞可能引起观测者的注意

## 王座的诅咒与代价

### 王座诅咒

每个王座都有对应的诅咒：

| 王座 | 诅咒 | 表现 |
|------|------|------|
| **万灵王座** | "生生不息" | 持有者逐渐失去对死亡的感知，无法理解"终结" |
| **白骨王座** | "万死即归" | 持有者逐渐失去对生命的感知，冷漠化 |
| **熔炉王座** | "燃烧一切" | 持有者的身体逐渐被火焰侵蚀 |
| **铁血王座** | "征服即永生" | 持有者逐渐陷入永恒的征服欲望 |
| **千衍王座** | "见证一切" | 持有者逐渐失去对"参与"的渴望 |
| **虚无王座** | "抹除一切" | 持有者逐渐失去与世界的连接 |

### 王座的"觉醒"

王座可能逐渐"觉醒"：

| 阶段 | 王座觉醒程度 | 持有者状态 |
|------|----------|----------|
| **沉睡期** | 王座处于沉睡，赋予持有者力量 | 持有者可以控制王座 |
| **苏醒期** | 王座开始产生自我意识 | 持有者仍能控制，但王座开始"暗示" |
| **觉醒期** | 王座产生强烈自我意识 | 持有者开始被王座"影响" |
| **吞噬期** | 王座吞噬持有者意识 | 持有者失去自我，被王座控制 |
