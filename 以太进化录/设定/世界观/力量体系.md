# 力量体系

## 核心概念：以太（Ether）

以太是世界的血液，是因果的载体，掌控者可以依靠以太，根据自己的主观意识引动世界法则。

### 三种以太

| 类型 | 特性 | 代表 |
|------|------|------|
| **创造之以太** | 温和、滋养、有序 | 人类主流 |
| **毁灭之以太** | 狂暴、破坏、混乱 | 魔物、兽人 |
| **观测之以太** | 中性、渗透、记录 | 混血、源晶 |

### 以太与力量的关系

所有力量形式本质上都是以太的不同表现：

| 力量形式 | 本质 | 特点 |
|---------|------|------|
| **斗气** | 以太在身体中的凝聚 | 强化身体、爆发力量 |
| **魔法** | 以太在精神中的引导 | 元素操控、术式释放 |
| **进化** | 以太在存在层面的重塑 | 改变生命本质、获得特殊能力 |
| **领域** | 以太在空间的展开 | 控制一定范围内的以太流动 |
| **法则** | 以太的终极形态 | 直接干涉世界法则，触碰因果 |

**以太的作用机制**：
1. **斗气**：以太凝聚在身体各处，形成"以太脉络"，强化肉体
2. **魔法**：以太由精神引导，形成"以太回路"，释放术式
3. **进化**：以太渗透到存在层面，重塑生命本质
4. **领域**：以太以自身为中心向外展开，控制范围内以太流动
5. **法则**：以太达到极高密度，可以直接干涉世界法则

### 以太回路

以太回路是以太操控的基础结构。所有现代术式都依赖体内以太回路来引导和释放以太。

| 概念 | 说明 |
|------|------|
| **以太回路** | 体内形成的以太流通通道，类似电路。术式通过回路引导以太形成特定效果 |
| **回路形成** | 正常人在觉醒以太后逐渐形成回路，回路越多、越精密，能使用的术式越复杂 |
| **回路类型** | 不同类型的以太对应不同回路（创造系回路、毁灭系回路等） |
| **回路与术式** | 现代基础术式全部依赖体内回路。没有回路 = 无法使用任何现代术式 |

### 环境以太操控

以太不仅存在于体内，也弥漫在环境中。环境中的以太被称为"游离以太"。

| 概念 | 说明 |
|------|------|
| **游离以太** | 弥漫在空气、土地、水中的以太，密度因地点而异 |
| **体外引导** | 不通过体内回路，直接用精神力引导环境中的游离以太。极少数人具备这种天赋 |
| **与回路术式的区别** | 回路术式：从体内输出以太 → 形成效果。体外引导：从环境中抽取以太 → 形成效果 |
| **体外引导的困难** | 游离以太浓度不稳定、难以精确控制、速度比体内回路慢。正常情况下，没有人会走这条路 |
| **远古术式** | 上古文明的术式不依赖体内回路，而是直接引导环境以太。但这些术式需要极高的以太密度和掌控力才能发动，已基本失传 |

**关键事实**：体内无法形成以太回路的人，在现代术式体系中几乎是"废物"。但如果此人体外引导能力极强，理论上可以通过引导游离以太来弥补——只是没有现成的术式可用，需要自己摸索或找到远古术式。

### 原住民的以太适应

原住民天生适应以太：

| 阶段 | 适应程度 | 能力 |
|------|---------|------|
| **出生** | 天然适应以太环境 | 可以感知以太，但无法操控 |
| **觉醒** | 开始操控以太 | 学习斗气或魔法的基础 |
| **掌控** | 熟练操控以太 | 可以使用中下位术式，形成以太脉络 |
| **精通** | 精通某种以太操控 | 可以开创新术式，形成领域 |
| **王阶** | 触碰法则层面 | 可以直接干涉世界法则，获得王座权限 |

### 穿越者的以太适应

穿越者的原世界可能有不同的本源能量（如元素力），需要进行适应：

**穿越者的适应阶段**：

| 阶段 | 适应状态 | 系统作用 | 身体反应 |
|------|---------|---------|---------|
| **穿越初期** | 不适应以太 | 生成系统，提供适应指引 | 身体虚弱，排斥反应（结晶纹路） |
| **适应期** | 逐渐适应以太 | 整合异界知识与本世界以太 | 结晶纹路消失，开始感知以太 |
| **掌控期** | 开始操控以太 | 提供进化方向选择 | 可以学习斗气或魔法 |
| **精通期** | 熟练操控以太 | 显示可进化方向 | 可以开创新术式 |
| **法则期** | 触碰法则层面 | 提供法则进化选择 | 可以干涉世界法则 |

**系统的适应机制**：

| 功能 | 说明 |
|------|------|
| **以太整合** | 帮助异界意识适应本世界以太，减轻排斥反应 |
| **知识转换** | 将异界知识转换为适应以太的知识 |
| **能力映射** | 将原世界的力量概念映射到以太体系 |
| **进化引导** | 基于以太本质，提供适合的进化方向 |
| **法则保护** | 在极端情况下，系统可以提供临时的法则保护 |

**穿越者的优势**：

由于原世界可能有不同的本源能量体系，穿越者在适应以太后可能具有特殊优势：

| 优势 | 说明 |
|------|------|
| **多元视角**：见过不同的能量体系，对以太有不同视角的理解 |
| **创新术式**：可以融合不同能量体系的理念，开创新的以太术式 |
| **独特进化**：异界知识可以拓宽可选择的进化方向 |
| **系统辅助**：系统帮助适应，比原住民更容易掌控以太 |

### 以太风暴带

环绕大陆边缘的狂暴以太区域，是世界的"边界"。穿越风暴带会到达"另一边"——一个被观测者支配的领域。

## 能力体系

### 位阶

| 位阶 | 描述 | 数量 | 以太状态 |
|------|------|------|---------|
| **下位** | 刚觉醒以太感知 | 大量 | 开始感知以太，体内形成初步以太脉络，无法引动外界以太 |
| **中位** | 能使用基础术式 | 较多 | 以太回路完善，可引导外界以太以太释放术式 |
| **上位（大师）** | 精通某一系术式 | 较少 | 以太密度大幅提升，可高速施法 |
| **传奇** | 开创独有术式 | 极少 | 以太达到临界密度，自发引动游离以太，防御型术式自发形成，可开创新术式 |
| **王** | 触碰因果律层面 | 6人 | 以太与法则直接共鸣，可干涉世界法则，触碰法则层面，术式附带部分规则型力量或判定|
| **神** | 超越因果律 | 3人 | 超越法则，创造法则 |

### 力量形式的以太本质

所有力量形式本质上都是以太的不同表现：

| 力量形式 | 以太本质 | 表现 | 形成条件 |
|---------|---------|------|---------|
| **斗气** | 以太凝聚在身体中 | 强化身体、爆发力量 | 通过呼吸吐纳，以太在身体中形成循环 |
| **术式** | 以太的特定排列 | 元素操控、复合效果 | 将以太按特定排列，形成稳定的效果 |
| **领域** | 以太向外展开 | 控制范围内以太流动 | 将以太以自身为中心向外扩散 |
| **法则** | 以太与法则共鸣 | 干涉世界法则、触碰因果 | 以太达到极高密度，直接与法则共鸣 |

### 进化系统

#### 进化的本质

**进化是生命本质的重塑，本质上是重塑生命中的以太结构**：

| 进化层级 | 本质 | 以太变化 |
|---------|------|---------|
| **基础进化** | 强化身体或能力 | 以太脉络的强化或扩展 |
| **领域进化** | 形成或强化领域 | 以太范围和密度的扩大 |
| **法则进化** | 触碰法则层面 | 以太与法则产生共鸣 |

#### 原住民进化

- **进化方式**：被动进化
- **进化方向**：由生存环境决定，自动适应环境
- **进化过程**：
  1. 在特定环境中长期生存
  2. 身体和能力逐渐适应环境
  3. 达到进化触发条件时，自动获得与环境匹配的进化
- **限制**：原住民无法主动选择进化方向，只能被动适应

**例子**：
- 长期生活在以太风暴带的居民 → 进化为以太抗性体
- 长期在地下生存的矿工 → 进化为暗视、抗压体
- 长期在寒冷地区生存的居民 → 进化为耐寒体

#### 异界生命体进化

- **进化方式**：自选进化（在匹配度范围内）
- **进化方向**：在"匹配度合理"的范围内主动选择
- **匹配度机制**：
  1. 系统评估异界生命体的特质（观察推理能力、知识储备、性格倾向等）
  2. 系统列出"匹配度合理"的进化方向
  3. 异界生命体从列表中主动选择
  4. 进化完成后，获得该方向对应的能力和特性 |

**匹配度评估**：
| 评估维度 | 权重 | 说明 |
|---------|------|------|
| 观察推理能力 | 30% | 异界生命体的观察和推理能力决定能承受的进化强度 |
| 知识储备 | 25% | 异界生命体的知识和经验影响可选择的进化方向 |
| 性格倾向 | 20% | 性格与进化方向的匹配程度 |
| 环境适应 | 15% | 当前环境与进化方向的兼容性 |
| 特殊资质 | 10% | 某些特殊资质会提高某些方向的匹配度 |

#### 世界法则系统

所有穿越者都拥有"系统"，这是世界法则对异界意识的自动响应机制。

**系统的来源**：
- 系统来自世界法则本身，不是观测者或其他神创造的
- 当异界意识进入这个世界时，世界法则会自动生成一个"适配器"——即系统
- 系统的本质是：**帮助异界意识适应世界法则，同时引导其回收散落的世界本源**
- 系统是穿越者与世界法则之间的"契约接口"——穿越者完成法则发布的任务，法则给予对应的馈赠

**系统的功能**：
1. **任务系统（核心驱动）**：系统任务是穿越者成长的核心途径。任务由世界法则自动生成，与地下城攻克、世界本源回收直接挂钩。完成任务获得法则馈赠（以太强化、进化机会、法则碎片等）。任务难度与穿越者位阶匹配，越危险的任务回报越高。系统不会强制执行任务，但拒绝任务意味着放弃成长机会——在黑暗森林般的穿越者竞争中，停滞等于死亡。
2. **进化匹配度评估**：显示可选择的进化方向和匹配度
3. **状态监控**：显示身体、灵魂、能力的状态
4. **知识整合**：帮助整合异界知识和本世界知识
5. **法则馈赠转译**：将世界法则的奖励转化为穿越者可理解、可吸收的形式

**系统的限制**：
1. **无法创造法则**：系统只能利用现有法则，无法创造新的
2. **无法干涉王座**：系统无法直接干涉王座争夺
3. **无法对抗神明**：系统的能力远低于神明
4. **会被观测者看到**：观测者可以看到系统，但不能直接修改
5. **任务有代价**：失败的任务可能有惩罚（以太逆流、进化锁定等），系统不保证穿越者的安全

#### 穿越者的共同特点

- **隐瞒身份**：绝大多数穿越者不会向外人提及自己的来历
- **隐瞒原因**：
  1. 被视为"异常"，可能被追捕或利用
  2. 被当作"试验品"，可能被再次关押
  3. 被视为"天选者"，可能引起不必要的注意
  4. 被当作"异端"，可能被审判
- **相互识别**：穿越者无法直接识别彼此，只能通过观察和推理判断
- **共守秘密**：穿越者之间有一条不成文的规则：不主动暴露其他穿越者

#### 穿越者的社会定位

穿越者在社会中的定位是**矛盾的**——各国争相招募，同时严加防范：

| 身份 | 说明 |
|------|------|
| **地下城探索者** | 穿越者因系统任务驱动，是最活跃的地下城探索者，各国争相招募 |
| **战略资源** | 穿越者的自选进化能力使其成长速度远超原住民，被视为国家级战略资源 |
| **潜在威胁** | 穿越者的异界来历和不可控进化方向使其永远无法被完全信任 |
| **被监控者** | 各国对穿越者实行"登记制"，名义上是保护，实际上是监控 |

穿越者的真实处境：**自由但不自由**。你可以选择为谁效力，但你不能选择不被监视。你可以拒绝任务，但你的竞争对手不会。系统任务驱动的黑暗森林，比任何外在束缚都更紧。

#### 穿越者与原住民的差异

| 特征 | 穿越者 | 原住民 |
|------|-------|-------|
| **进化方式** | 可自选（在匹配度范围内） | 被动，由环境决定 |
| **知识来源** | 来自异界的知识 | 来源于本世界的学习 |
| **能力体系** | 可能混合两套体系 | 只能使用本世界体系 |
| **世界观** | 可能有不同世界观 | 认可本世界的世界观 |
| **灵魂特质** | 异界特质，可能有特殊效果 | 本世界特质 |

## 法则碎片与规则漏洞

### 上世代大战的遗产

在上世代三神大战和王座陨落中，世界法则被打散成无数碎片：

| 类型 | 特点 | 位置 |
|------|------|------|
| **法则碎片** | 携带部分法则能力的以太结晶 | 散落各地，可能埋藏在地底或生物体内 |
| **法则残留** | 打散的法则能量，形成以太异常 | 王座陨落地、战场遗址 |
| **规则漏洞** | 世界法则被打散后形成的漏洞 | 以太风暴带、古代遗迹、时空裂缝附近 |
| **因果裂隙** | 因果链被打断的地方 | 上世代战场、王座陨落地 |

### 法则碎片的价值

| 碎片类型 | 能力 | 获得方式 |
|---------|------|---------|
| **创造碎片** | 强化治愈、创造生命、提升以太亲和 | 探险遗迹、挖掘、击杀特定怪物 |
| **毁灭碎片** | 强化破坏、腐蚀、爆炸能力 | 探险危险区域、击杀特定怪物 |
| **观测碎片** | 增强感知、预知、记录能力 | 极难获得，可能隐藏在古代文献中 |
| **虚空碎片** | 获得规则外能力，暂时突破规则 | 极其危险，可能来自虚无法则本身 |

### 规则漏洞的利用

**规则漏洞的表现**：
- 某些地区或时期以太规则会失效
- 特定条件下术式效果会扭曲或增强
- 因果链可能被临时切断

**规则漏洞的风险**：
- 漏洞会被世界法则修复，时间不定
- 漏洞区域以太极不稳定，可能导致疯狂或变异
- 利用漏洞可能引起观测者的注意

## 生命强度

生命强度是衡量生命体进化程度的数值指标，由系统量化显示。它不代表战力，代表的是**生命层次**——身体、灵魂、以太亲和度的综合进化程度。

### 物种基准

| 物种/状态 | 生命强度 | 说明 |
|---------|---------|------|
| 普通人类 | 10 | 未经以太觉醒的凡人 |
| 觉醒者（下位） | 50-100 | 刚开始操控以太 |
| 中位 | 100-300 | 能使用基础术式 |
| 上位（大师） | 300-600 | 精通某一系术式 |
| 传奇 | 600-1500 | 开创独有术式 |
| 王阶 | 3000+ | 触碰法则层面 |
| 巨龙/远古生物 | 500-2000 | 天生物种优势，千年寿命光靠呼吸就到这个数 |
| 神 | 不可测量 | 超越数值 |

### 生命压制

生命强度差距产生压制效果，以**倍数**衡量：

| 差距倍数 | 效果 |
|---------|------|
| 2倍以内 | 正常战斗，无压制 |
| 2-5倍 | 轻微压制：以太操控效率下降，身体本能不适 |
| 5-10倍 | 明显压制：术式效果减半，低阶者本能恐惧 |
| 10倍以上 | 降维打击：低阶者无法在高阶者面前凝聚以太 |

### 生命强度增长

| 方式 | 增幅 | 说明 |
|------|------|------|
| 系统任务完成 | 中等 | 主要来源，稳定增长 |
| 地下城本源解放 | 大量 | 一次性，与贡献度挂钩 |
| 濒死突破 | 少量 | 不可控，不可重复 |
| 长期修炼/战斗 | 微量 | 最慢但最稳 |
| 进化选择 | 质变 | 生命层次跃迁，见下文 |

## 进化方向与容器

### 核心机制

每次进化选择，生命体会选择一个**进化方向**。进化方向如同一个"容器"，有容量上限——该方向最多能承载多少生命强度。

生命强度触达容器上限后，必须再次进化突破天花板，否则生命强度不再增长。

### 进化阈值

生命强度达到特定阈值时，系统触发进化选择。阈值因人而异，取决于当前进化方向的容量上限：

| 情况 | 触发条件 |
|------|---------|
| **首次进化** | 生命强度达到 50（从凡人到觉醒者的跃迁） |
| **后续进化** | 生命强度触达当前进化方向的容量上限 |
| **强制进化** | 生命强度超过上限 10% 后仍未进化，身体开始崩溃 |

### 进化方向容量示例

| 进化方向 | 容量上限 | 进化难度 | 说明 |
|---------|---------|---------|------|
| 强化人类 | 200 | 低 | 最基础方向，很快触顶 |
| 兽化人 | 500 | 低-中 | 常见选择，中等容量 |
| 亚龙 | 1500 | 高 | 需要龙血等稀有素材 |
| 龙裔 | 5000 | 极高 | 传说级方向 |
| 纯血巨龙 | 20000 | 几乎不可能 | 几乎无上限 |

### 容量满了不进化的后果

| 状态 | 后果 |
|------|------|
| **停滞** | 生命强度不再增长，系统任务奖励转化为其他资源 |
| **溢出风险** | 强行突破上限，身体崩溃或异变 |
| **质量压制** | 同等生命强度但容器更大的人有隐性优势 |

## 体修与术士

战斗中不常使用以太术式的强者，一般被视为**体修**。与之相对，依赖以太回路释放术式战斗的人是**术士**。

### 核心差异

| 维度 | 术士 | 体修 |
|------|------|------|
| **战斗方式** | 以太回路 → 术式 → 远程/范围攻击 | 身体强化 → 肉搏/武器 → 近战 |
| **生命强度** | 中等（以太用于术式，不全用于强化身体） | 较高（以太全部灌注身体，生命强度是核心指标） |
| **位阶提升** | 通过术式练习和回路精炼 | 通过极限战斗和身体突破 |
| **优势** | 攻击手段多样、可远程、可范围 | 生命强度高、近战压制强、耐打 |
| **劣势** | 生命强度相对低、近战弱 | 攻击手段单一、缺乏远程和范围能力 |

### 体修的生命强度优势

同阶（位阶相同）的体修和术士相比，体修的生命强度普遍更高。原因是：

- 术士将大量以太用于维持回路和释放术式，分配给身体强化的以太有限
- 体修不维护回路，所有以太都灌注到身体和生命本质中
- 体修的战斗方式本身就是"以命搏命"，频繁的极限战斗推动生命强度快速增长

### 体修的战力评估

体修的战力不按位阶评估，直接按**生命强度**估算：

| 生命强度 | 体修战力等价 |
|---------|------------|
| 50-100 | 下位体修，可以和下位术士正面对抗 |
| 100-300 | 中位体修，近战中可以压制中位术士 |
| 300-600 | 上位体修，生命强度已经是上位术士的 1.5-2 倍 |
| 600+ | 传奇体修，近战中几乎无敌，但缺乏远程手段 |

### 体修的典型战斗风格

- 不用术式，或只用最简单的以太强化（灌注武器、强化肌肉）
- 依靠生命强度带来的身体素质碾压对手
- 耐打、持久、爆发力强
- 弱点：面对远程术式和范围攻击时被动，需要接近对手才能发挥

### 诺缇与体修

诺缇因无法形成以太回路，前期被迫走体修路线。他的以太全部用于身体强化（虽然他不擅长主动灌注，但身体在以太环境中自然吸收），加上频繁的地下城战斗和濒死经历，生命强度增长速度比同阶术士快。

这意味着：诺缇在位阶上是"下位"，但生命强度可能已经是"中位"水平。术士用术式弥补生命强度差距，诺缇用生命强度弥补术式缺失。
