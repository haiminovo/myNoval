# 题材定位

## 基本信息

- 书名：以太进化录
- 题材/类型：奇幻 / 群穿 / 群像 / 地下城
- 目标平台：起点
- 预计规模：2000+ 章，12 卷
- 目标读者：喜欢系统流、群像叙事、地下城探索的起点男频读者

## 一句话梗概

诺缇（Noctis）——一个占据失败试验体身体的异界穿越者，在系统任务的驱动下攻克地下城、回收世界本源，却发现所有穿越者都被卷入了一场由三神暗中操控的王座争夺战，而各国势力早已将穿越者视为最锋利的棋子。

## 核心梗三分法

- 表层卖点：系统任务 + 地下城探索 + 群穿竞争
- 深层爽点：以太操纵的独特战斗方式 + 穿越者之间的黑暗森林博弈 + 群像多视角叙事
- 长线钩子：三神的合谋真相 + 系统的真实本质 + 虚无王座的秘密 + 世界本源的终极归属

## 核心设定差异化

- 系统任务驱动：穿越者的成长来自系统任务（攻克地下城、回收世界本源），不是打怪升级
- 主角金手指：环境以太操纵极强但体内无法形成回路——前期只能肉搏，后期通过远古术式走出自己的路
- 地下城体系：上古大战遗落的世界本源沉入地底，吸引魔物聚集形成地下城，攻克即回收本源即获法则嘉奖
- 穿越者黑暗森林：所有穿越者都有系统任务，资源有限，竞争残酷
- 群像叙事：多个穿越者视角，各自有独立的成长线和阵营选择
- 六王座：不是力量等级，而是世界承认的"因果席位"
- 三神博弈：创造者和毁灭者暗中合谋击杀观测者，穿越者是棋子
- 各国势力：围绕地下城控制权和穿越者资源展开复杂博弈

## 对标分析

| 对标书 | 相似点 | 差异点 | 可借鉴 |
|-------|-------|-------|-------|
| 超神机械师 | 系统流、信息差、布局感 | 本书有进化变异、身体异变 | 数值爽感、长线布局节奏 |
| 无职转生 | 异世界成长、真实感、情感重量 | 本书有系统面板、更多战斗 | 角色成长弧线、世界真实感 |

## 题材框架

- 开局：试验体苏醒 → 逃亡 → 第一次地下城探索
- 第一阶段：地下城探索 → 系统任务 → 接触各国势力（第1-150章）
- 中期：穿越者博弈 → 各国势力斗争 → 王座争夺 → 真相浮现
- 后期：三神对决 → 世界本源归属 → 终局
