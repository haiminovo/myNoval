# 题材定位

## 基本信息

- 书名：以太进化录
- 题材/类型：奇幻 / 游戏异界 / 成长
- 目标平台：起点
- 预计规模：2000+ 章，12 卷
- 目标读者：喜欢系统流、进化变异、异界穿越的起点男频读者

## 一句话梗概

诺缇（Noctis）——一个占据失败试验体身体的异界穿越者，在进化系统的帮助下从地下实验室逃出，却发现所有穿越者都被卷入了一场由三神暗中操控的王座争夺战。

## 核心梗三分法

- 表层卖点：系统流 + 进化变异 + 异界穿越
- 深层爽点：行为驱动的进化选择（你吃了什么就可能变成什么）+ 穿越者之间的黑暗森林博弈
- 长线钩子：三神的合谋真相 + 系统的真实本质 + 虚无王座的秘密

## 核心设定差异化

- 行为驱动进化：进化路线由角色的行为和摄入决定，不是随机或固定路线
- 穿越者系统：所有穿越者都有系统，穿越者之间是黑暗森林关系
- 六王座：不是力量等级，而是世界承认的"因果席位"
- 三神博弈：创造者和毁灭者暗中合谋击杀观测者，穿越者是棋子

## 对标分析

| 对标书 | 相似点 | 差异点 | 可借鉴 |
|-------|-------|-------|-------|
| 超神机械师 | 系统流、信息差、布局感 | 本书有进化变异、身体异变 | 数值爽感、长线布局节奏 |
| 无职转生 | 异世界成长、真实感、情感重量 | 本书有系统面板、更多战斗 | 角色成长弧线、世界真实感 |
| 伊洛纳 | 猎奇进化路线、行为驱动 | 本书是小说不是游戏 | 进化选择的荒诞感和困境感 |
| 诡秘之主 | 试验体开局、多势力博弈 | 本书有进化系统、穿越者黑暗森林 | 悬念设计、势力博弈 |

## 题材框架

- 开局：试验体苏醒 → 逃亡 → 生存
- 第一阶段：生存 → 成长 → 接触势力（第1-150章）
- 中期：王座争夺 → 穿越者博弈 → 真相浮现
- 后期：三神对决 → 终局
