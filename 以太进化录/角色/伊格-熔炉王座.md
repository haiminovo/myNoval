## 性格特征

### 核心特质

- **炽热而理性**：内心如火般炽热，但决策如钢铁般理性
- **完美主义**：认为世界可以被锻造得完美，任何不完美都需要被"重塑"
- **责任感**：认为自己是世界的锻造者，有责任维护世界的有序
- **冷酷的仁慈**：他的"清洗"是为了让世界更完美，他真心这么认为
- **时间的紧迫感**：接近寿命极限，必须在死前为继承者铺路

### 行为模式

1. **锻造式治理**：将国家看作熔炉，将人民看作矿石，不断锻造和重塑
2. **定期清洗**：定期清洗"不完美"的部分，被清洗者包括腐败的贵族、反叛的势力
3. **追求永恒**：试图锻造一个永恒不灭的完美秩序
4. **直接行动**：不喜欢阴谋诡计，更喜欢直接的行动和结果
5. **放任战争**：为了快速磨练继承者，放任并发起战争，让继承者与铁血王座共鸣

## 成王之路

### 原本的起点

伊格出生在灰烬帝国的铁匠世家，从小在火炉前长大。他的天赋异常，能够感知火焰的本质，锻造出超越常理的武器。

### 王座的选择

在灰烬帝国内战最混乱的时期，伊格在火炉前锻造时突然"看见"了世界的真实形态——世界是一个巨大的熔炉，一切都可以被锻造、被重塑、被完美化。

在那一瞬间，熔炉王座选择了他。王座赋予他"锻造世界"的权限，他开始用这个权限"锻造"灰烬帝国。

### 成为熔炉王

伊格用熔炉王座的力量结束了内战，然后开始锻造一个新的秩序：

- 他清洗了所有腐败的贵族
- 他建立了新的法律和制度
- 他将国家看作一个巨大的锻造工程，不断优化和重塑
- 他成为了灰烬帝国的皇帝，也是熔炉王座的持有者

### 继承者的培养

280岁时，伊格意识到自己的寿命即将到达极限：

- 他开始培养继承者"灰烬之子"瓦伦（22岁）
- 瓦伦的锻造天赋异常，但缺乏经验和战场磨练
- 伊格意识到：瓦伦需要快速成长，必须在数年内达到能够继承熔炉王座和共鸣铁血王座的程度
- **战争是共鸣铁血王座的唯一途径**：只有通过征服和杀戮，瓦伦才能真正与铁血王座共鸣
- 为了快速磨练瓦伦，伊格决定发起战争，让瓦伦亲自指挥锻火军团进行铁血王座争夺

### 铁血王座的共鸣

瓦伦的锻造天赋本质是"锻造征服"——他能锻造军队、锻造战争、锻造征服。这个天赋与铁血王座天然共鸣：

- **锻造军队**：瓦伦能将军队锻造成最锋利的征服工具
- **锻造战争**：瓦伦能将战争锻造成最完美的征服过程
- **锻造征服**：瓦伦能将征服锻造成最强大的毁灭力量

伊格认为：只有让瓦伦亲自指挥锻火军团进行铁血王座争夺，瓦伦才能真正与铁血王座共鸣，成为双王持有者。

## 技能与天赋

### 熔炉法则

**第一层：火焰掌控**

- **无限火焰**：可以召唤和控制无限的火焰
- **火焰免疫**：完全免疫火焰的任何伤害
- **火焰锻造**：用火焰锻造任何材质，包括以太

**第二层：万物锻造**

- **物质重塑**：将任何物质重新锻造成任何形态
- **生命锻造**：可以锻造生命（但非常危险，容易导致死亡）
- **秩序锻造**：将混乱的领域锻造为有序的领域

**第三层：熔炉降临**

- **熔炉领域**：展开后，领域内的所有物质都会被"锻造"成伊格想要的形态
- **世界熔炉**：将整个世界视为熔炉，尝试锻造世界（需要巨大的消耗）
- **永恒火焰**：召唤永恒不灭的火焰，永远燃烧，永不熄灭

### 特殊天赋

- **完美感知**：能感知任何物质和生命的"完美度"
- **锻造本能**：对锻造有天然的直觉，能快速找到最优方案
- **火焰共鸣**：能与所有的火焰产生共鸣，控制它们

## 观念与诉求

### 核心观念

**世界是一个巨大的熔炉，一切都可以被锻造、被重塑、被完美化。**

- 不完美的东西都应该被"重新锻造"
- 秩序是完美的前提，混乱是锻造的原料
- 他的"清洗"不是屠杀，而是"锻造过程"
- 真正的完美是永恒不灭的秩序

### 诉求

1. **锻造完美帝国**：将灰烬帝国锻造为完美的帝国
2. **锻造完美世界**：最终将整个世界锻造为完美的形态
3. **维护秩序**：确保秩序不被动摇，混乱不蔓延
4. **达到永恒**：锻造一个永恒不灭的完美秩序

### 对双神和观测者的态度

| 神明 | 伊格的态度 | 原因 |
|------|-----------|------|
| **创造者** | 尊敬 | 创造者的创造本质与熔炉王座的锻造本质一致 |
| **毁灭者** | 轻蔑 | 毁灭者的破坏本质与熔炉王座的锻造本质对立 |
| **观测者** | 无视 | 观测者只是记录，不参与锻造，对伊格没有意义 |

## 所属势力：灰烬帝国

### 势力本质

灰烬帝国是一个以"锻造"为核心的国家。所有的事物——人民、法律、文化——都在不断地被"锻造"和"重塑"。

### 势力结构

| 阶层 | 描述 |
|------|------|
| **皇帝** | 伊格，熔炉王座持有者，绝对权力 |
| **锻火军团** | 伊格亲自锻造的军队，忠诚且强大 |
| **炼金院** | 专门研究"锻造"技术的机构，部分被进化教派渗透 |
| **灰烬教会** | 宣传"锻造即完美"的教会，支持伊格的清洗行动 |

### 势力活动

- **定期清洗**：清洗"不完美"的部分，维持秩序
- **锻造实验**：炼金院进行各种"锻造"实验，包括人体实验
- **寻找变量**：寻找能打破规则的存在（如诺缇），试图将其"锻造"成有用工具
- **维护边界**：确保灰烬帝国的边界不被侵犯

## 当前正在做什么

### 主要活动（发起冲突）

1. **让瓦伦亲率锻火军团**：伊格亲自将锻火军团的指挥权交给瓦伦，让他亲自指挥铁血王座争夺战
2. **为瓦伦打造完美战场**：伊格用熔炉王座的力量，将战场锻造成"完美"的训练场，确保瓦伦能快速成长
3. **让瓦伦共鸣铁血王座**：战争是共鸣铁血王座的唯一途径，瓦伦必须亲自征服和杀戮
4. **暗中支持瓦伦**：伊格在暗中用熔炉王座的力量帮助瓦伦，但不直接干预
5. **与其他势力的秘密合作**：伊格与莫特有秘密协议——伊格为莫特提供战场，莫特为伊格提供亡灵情报

### 对空缺王座的打算

| 空缺王座 | 伊格的打算 |
|---------|-----------|
| **万灵王座** | 不感兴趣，认为万灵王座的力量"不够完美"，不如熔炉王座 |
| **铁血王座** | 志在必得，让瓦伦通过战争共鸣铁血王座，成为双王持有者（熔炉+铁血） |
| **虚无王座** | 警惕，认为虚无王座是"不完美的变量"，准备让瓦伦将其锻造或清除 |

### 参与王座争夺的方式（直接冲突）

- **让瓦伦亲率锻火军团**：伊格亲自将指挥权交给瓦伦，让他亲自指挥铁血王座争夺战
- **锻造完美战场**：用熔炉王座的力量，将战场锻造成"完美"的训练场
- **暗中支持瓦伦**：在暗中用熔炉王座的力量帮助瓦伦，但不直接干预
- **与莫特的秘密合作**：伊格为莫特提供战场，莫特为伊格提供亡灵情报
- **阻止记录**：用炼金院的力量，阻止千衍王座的记录

### 与其他势力的直接冲突

| 势力 | 冲突原因 | 直接对抗 |
|------|---------|---------|
| **白骨王座** | 表面：莫特试图收集战争中死亡的强者并转化成亡灵，伊格派遣锻火军团直接对抗；秘密：双方有协议，伊格为莫特提供战场，莫特为伊格提供亡灵情报 |
| **千衍王座** | 维德试图记录战争，伊格认为这会影响战争的"纯粹性" | 伊格用炼金院的力量，干扰维德的千眼使者 |
| **争夺铁血王座的势力** | 多个势力都在争夺，伊格必须清除竞争对手 | 瓦伦亲自指挥锻火军团，用熔炉王座的力量直接清除 |
| **周边势力** | 为了给瓦伦更多战场，伊格开始快速扩张 | 瓦伦亲自指挥锻火军团，侵略周边势力 |

### 上个世代王座陨落的记忆

伊格是上个世代大战的亲历者，那时他还不是王座持有者：

- **万灵王座陨落**：上一个持有者试图创造"永生生命"，结果创造出无法控制的怪物，最终被自己的创造物吞噬
- **铁血王座陨落**：上一个持有者被自己的军队背叛，王座被污染后消失
- **虚无王座隐匿**：上一个持有者主动隐匿，所有人都以为王座陨落

伊格从这些"陨落"中学到了一个教训："不完美导致毁灭"。他发誓要用熔炉王座锻造一个"完美"的帝国，避免重蹈覆辙。

## 王座争夺的影响

### 近百年的规则涌动

近百年，伊格开始感受到王座规则的涌动：

- **万灵王座规则**：生命之力开始在各地涌动，多个势力感受到召唤
- **铁血王座规则**：征服之力开始在各地涌动，多个军阀开始扩张
- **虚无王座规则**：虚无之力在各地"闪烁"，但很快消失（因为露希的隐匿和诺缇的登座）

### 伊格的应对

| 王座 | 涌动表现 | 伊格的应对 |
|------|---------|-----------|
| **万灵王座** | 生命之力异常强大，有人试图创造永生 | 不参与，但密切观察，评估其中的威胁 |
| **铁血王座** | 征服之力异常强大，多个势力开始扩张 | 派遣锻火军团参与争夺，确保王座归属于自己 |
| **虚无王座** | 虚无之力在各地"闪烁" | 诺缇登座后，伊格开始观察他，评估他是威胁还是变量 |

### 潜在风险

| 风险 | 应对 |
|------|------|
| 诺缇成为威胁 | 如果诺缇过于危险，伊格会亲自锻造他 |
| 炼金院被渗透 | 伊格在暗中调查，准备清洗渗透者 |
| 帝国内部反叛 | 伊格会再次启动清洗行动 |
| 铁血王座被其他势力夺走 | 伊格会亲自出手，用熔炉王座的力量清除竞争对手 |
| 万灵王座被滥用 | 如果万灵王座创造出的怪物威胁灰烬帝国，伊格会直接清除 |

## 备注

- 伊格不认为自己是暴君，他认为自己是"世界的锻造者"
- 他真心相信自己的清洗是为了让世界更美好
- 熔炉王座的力量不是单纯的火焰，而是"锻造"本身
- 伊格的锻造不是创造，而是"重塑"，他能将任何东西锻造成他想要的形态
- 伊格对双神的态度是复杂的：他尊敬创造者，轻蔑毁灭者，无视观测者
- 伊格是上个世代大战的亲历者，他从王座陨落中学到了"不完美导致毁灭"的教训
- 伊格目前最重视铁血王座的争夺，他认为这个王座可以扩大灰烬帝国的"锻造范围"
- 伊格对诺缇保持警惕，他认为虚无王座是"无法锻造的不完美变量"
