# 世界观设定

## 地理

### 黑岛（Black Isle）
| 属性 | 内容 |
|------|------|
| **位置** | 被诅咒的海域中央 |
| **特点** | 暗精灵不敢靠近，以太能量异常活跃 |
| **居民** | 人类为主，少量其他种族 |
| **防御** | 天然屏障（以太风暴） |
| **秘密** | 岛上可能有上古遗迹 |

### 主城（Main City）
| 属性 | 内容 |
|------|------|
| **位置** | 黑岛内陆 |
| **特点** | 人类聚居地，冒险者公会总部 |
| **建筑** | 城墙、守卫塔、公会大厅、市场 |
| **势力** | 城主、守卫队、冒险者公会、商人联盟 |

### 暗精灵森林（Shadow Woods）
| 属性 | 内容 |
|------|------|
| **位置** | 大陆西侧 |
| **特点** | 永恒黄昏，月光永远照耀 |
| **居民** | 影月氏族及其他暗精灵部落 |
| **危险** | 追踪队、诅咒陷阱、暗影生物 |

### 以太风暴带（Aether Storm Belt）
| 属性 | 内容 |
|------|------|
| **位置** | 环绕黑岛的海域 |
| **特点** | 狂暴的以太能量，时空扭曲 |
| **作用** | 天然屏障，也是连接异次元的通道 |
| **传说** | 穿越风暴可到达"彼方" |

## 力量体系

### 八元素
| 元素 | 特性 | 代表颜色 | 常见用途 |
|------|------|----------|----------|
| 水 | 治愈、变化、流动 | 蓝 | 医疗、变形、净化 |
| 火 | 破坏、爆发、热情 | 红 | 攻击、锻造、照明 |
| 风 | 速度、感知、自由 | 青 | 侦查、加速、通讯 |
| 土 | 防御、坚韧、生长 | 褐 | 护盾、建筑、农业 |
| 光 | 神圣、净化、真相 | 金 | 驱邪、治愈、鉴定 |
| 暗 | 隐匿、诅咒、死亡 | 紫 | 暗杀、控制、亡灵术 |
| 生命 | 生长、恢复、繁衍 | 绿 | 治愈、催生、解毒 |
| 时空 | 稀有、禁忌、强大 | 银 | 传送、预知、时间操控 |

### 以太（Aether）
- **本质**：八元素之上的原初能量
- **特性**：可转化为任何元素，也可独立存在
- **危险**：过量接触会导致"以太侵蚀"
- **源晶**：以太的固态结晶，极其稀有

### 以太侵蚀
| 阶段 | 症状 | 结果 |
|------|------|------|
| 初期 | 皮肤结晶化 | 可逆，需特殊药剂 |
| 中期 | 内脏结晶化 | 难以治愈，力量增强 |
| 后期 | 全身晶体化 | 死亡或转化为"晶灵" |
| 特殊 | 与宿主融合 | 极少数案例，力量质变 |

### 魔术等级
| 等级 | 名称 | 描述 |
|------|------|------|
| 下位 | 基础术式 | 学徒级别，简单元素操控 |
| 中位 | 标准术式 | 正式魔术师，可组合元素 |
| 上位 | 高级术式 | 大师级别，复杂术式构建 |
| 超位 | 禁咒 | 传说中的术式，代价巨大 |
| 以太 | 原初术式 | 直接操控以太，近乎神迹 |

## 种族

### 哥布林（Goblin）
| 属性 | 内容 |
|------|------|
| **寿命** | 约30-40年 |
| **体型** | 矮小（成年约1.2-1.4米） |
| **特征** | 绿皮肤、尖耳、繁殖力强 |
| **社会** | 部落制，崇尚力量 |
| **偏见** | 被其他种族视为低级、野蛮 |
| **优势** | 敏捷、夜视、适应力强 |

### 暗精灵（Dark Elf）
| 属性 | 内容 |
|------|------|
| **寿命** | 约300-500年 |
| **体型** | 修长优雅（约1.7-1.9米） |
| **特征** | 深色皮肤、银白发、紫瞳 |
| **社会** | 氏族制，血脉至上 |
| **信仰** | 月神，崇尚纯洁 |
| **能力** | 暗元素亲和、隐匿、长寿 |

### 人类（Human）
| 属性 | 内容 |
|------|------|
| **寿命** | 约70-80年 |
| **特点** | 适应力强，学习能力优秀 |
| **社会** | 城邦/王国制，多样性 |
| **优势** | 无元素亲和限制，可学习任何魔术 |
| **劣势** | 寿命短，无种族天赋 |

### 矮人（Dwarf）
| 属性 | 内容 |
|------|------|
| **寿命** | 约200-300年 |
| **体型** | 敦实（约1.3-1.5米） |
| **特征** | 浓密胡须、强壮、耐操 |
| **社会** | 氏族制，重视工艺 |
| **能力** | 锻造大师、土元素亲和 |

## 历史与传说

### 上古时代
- 以太源晶从天而降，带来魔法文明
- 八元素体系建立
- 各种族和平共处（黄金时期）

### 大分裂
- 暗精灵与其他种族决裂
- 哥布林被驱逐到边缘地带
- 人类崛起，建立城邦

### 现今
- 种族隔阂严重，混血被视为禁忌
- 以太源晶成为传说中的宝物
- 冒险者探索遗迹，寻找上古力量
