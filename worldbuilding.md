# 世界观设定

## 核心设计理念

融合 **伊洛纳(Elona)** 的开放世界自由度、**勇者斗恶龙(DQ)** 的经典奇幻温情、**克苏鲁神话** 的宇宙恐怖、**龙与地下城(D&D)** 的严谨规则体系。

---

## 地理

### 黑岛（Black Isle）—— 避难所与牢笼

| 属性 | 内容 |
|------|------|
| **位置** | 被诅咒的海域中央，以太风暴带环绕 |
| **特点** | 暗精灵不敢靠近，以太能量异常活跃 |
| **居民** | 人类为主，少量其他种族（逃亡者、流放者） |
| **防御** | 天然屏障（以太风暴），易守难攻 |
| **秘密** | 岛上可能存在上古遗迹、被封印的"彼方之门" |
| **氛围** | 安全但压抑，居民对外来者既警惕又好奇 |

**参考伊洛纳**：像北提里斯一样，是冒险者的起点，也是各种族的交汇点。

### 主城（Main City）—— 微缩世界

| 区域 | 功能 | 特色 |
|------|------|------|
| **城墙区** | 防御、守卫驻扎 | 厚重石墙，刻有防御符文 |
| **公会区** | 冒险者公会总部 | 发布任务、评定等级、交易情报 |
| **市场区** | 商业交易 | 各族商人聚集，黑市暗藏 |
| **居住区** | 平民生活 | 人类为主，偶有混血儿躲藏 |
| **地下城入口** | 探索遗迹 | 被封印的上古通道，需许可进入 |
| **港口** | 与外界联系 | 船只稀少，以太风暴阻隔 |

**参考D&D**：像无冬城一样，有完整的城邦政治和势力斗争。

### 暗精灵森林（Shadow Woods）—— 永恒黄昏

| 属性 | 内容 |
|------|------|
| **位置** | 大陆西侧 |
| **特点** | 永恒黄昏，月光永远照耀，暗影生物出没 |
| **居民** | 影月氏族及其他暗精灵部落 |
| **危险** | 追踪队、诅咒陷阱、暗影生物、精神污染 |
| **秘密** | 月神殿地下封印着远古存在 |

**参考克苏鲁**：森林深处有不可名状的存在，暗精灵的月神信仰可能是对更古老存在的误读。

### 以太风暴带（Aether Storm Belt）—— 混沌之海

| 属性 | 内容 |
|------|------|
| **位置** | 环绕黑岛的海域 |
| **特点** | 狂暴的以太能量，时空扭曲，现实法则失效 |
| **作用** | 天然屏障，也是连接异次元的通道 |
| **传说** | 穿越风暴可到达"彼方"——另一个世界或时间线 |
| **危险** | 以太侵蚀、时空错位、遭遇"风暴中的东西" |

**参考克苏鲁**：以太风暴中可能存在着来自宇宙深处的存在，它们对人类的命运漠不关心。

### 大陆其他地区（待探索）

- **哥布林荒原**：被驱逐的哥布林部落聚居地，混乱而危险
- **矮人山脉**：地下王国，锻造圣地，对外来者封闭
- **人类王国**：繁华但腐败，贵族与教会权力斗争
- **遗忘废墟**：上古文明的遗迹，藏有以太源晶的秘密

## 力量体系

### 八元素

| 元素 | 特性 | 代表颜色 | 常见用途 | 对应情绪 |
|------|------|----------|----------|----------|
| 水 | 治愈、变化、流动 | 蓝 | 医疗、变形、净化 | 平静、哀伤 |
| 火 | 破坏、爆发、热情 | 红 | 攻击、锻造、照明 | 愤怒、激情 |
| 风 | 速度、感知、自由 | 青 | 侦查、加速、通讯 | 好奇、自由 |
| 土 | 防御、坚韧、生长 | 褐 | 护盾、建筑、农业 | 稳重、固执 |
| 光 | 神圣、净化、真相 | 金 | 驱邪、治愈、鉴定 | 希望、正义 |
| 暗 | 隐匿、诅咒、死亡 | 紫 | 暗杀、控制、亡灵术 | 恐惧、欲望 |
| 生命 | 生长、恢复、繁衍 | 绿 | 治愈、催生、解毒 | 慈爱、嫉妒 |
| 时空 | 稀有、禁忌、强大 | 银 | 传送、预知、时间操控 | 孤独、疯狂 |

**参考D&D**：类似D&D的法术学派，但更加元素化。

### 以太（Aether）—— 原初之力

- **本质**：八元素之上的原初能量，宇宙的基础构成
- **特性**：可转化为任何元素，也可独立存在；带有"意识"，会回应强烈的情感
- **危险**：过量接触会导致"以太侵蚀"，但适度接触可增强体质
- **源晶**：以太的固态结晶，极其稀有，是上古文明的遗物

**参考克苏鲁**：以太类似于克苏鲁神话中的宇宙能量，接触过多会导致理智丧失和身体变异。

### 以太侵蚀（Aether Corruption）

| 阶段 | 症状 | 结果 | 参考 |
|------|------|------|------|
| 初期 | 皮肤结晶化，微光流动 | 可逆，需特殊药剂 | 像轻微的辐射病 |
| 中期 | 内脏结晶化，力量增强但情绪不稳 | 难以治愈，可能失控 | 类似克苏鲁的"知识诅咒" |
| 后期 | 全身晶体化，意识模糊 | 死亡或转化为"晶灵" | 彻底的异化 |
| 特殊 | 与宿主融合，结晶成为身体的一部分 | 极少数案例，力量质变但人性逐渐丧失 | 诺提的状态 |

**参考伊洛纳**：类似伊洛纳中的以太病，是一种带有随机性的"祝福/诅咒"。

### 魔术等级

| 等级 | 名称 | 描述 | 学习难度 |
|------|------|------|----------|
| 下位 | 基础术式 | 学徒级别，简单元素操控 | 数月 |
| 中位 | 标准术式 | 正式魔术师，可组合元素 | 数年 |
| 上位 | 高级术式 | 大师级别，复杂术式构建 | 数十年 |
| 超位 | 禁咒 | 传说中的术式，代价巨大 | 罕见的天才 |
| 以太 | 原初术式 | 直接操控以太，近乎神迹 | 几乎不可能 |

**参考D&D**：类似D&D的法术环阶系统，但更加强调"代价"。

### 职业体系（参考D&D）

| 职业 | 核心能力 | 典型种族 | 特点 |
|------|----------|----------|------|
| 战士 | 近战、防御 | 人类、矮人 | 坚韧可靠 |
| 法师 | 元素魔术 | 人类、暗精灵 | 知识渊博但脆弱 |
| 牧师 | 神圣术、治疗 | 人类 | 信仰驱动 |
| 盗贼 | 隐匿、陷阱 | 哥布林、人类 | 灵活狡猾 |
| 游侠 | 追踪、自然 | 精灵、人类 | 独来独往 |
| 术士 | 血脉魔法 | 混血儿 | 力量不稳定 |
| 以太使 | 操控以太 | 极少数 | 被恐惧和觊觎 |

## 种族

### 哥布林（Goblin）—— 被歧视的求生者

| 属性 | 内容 |
|------|------|
| **寿命** | 约30-40年（但成长极快，3个月相当于人类3岁） |
| **体型** | 矮小（成年约1.2-1.4米） |
| **特征** | 绿皮肤、尖耳、繁殖力强、夜视能力 |
| **社会** | 部落制，崇尚力量，弱肉强食 |
| **偏见** | 被其他种族视为低级、野蛮、肮脏 |
| **优势** | 敏捷、夜视、适应力强、繁殖快 |
| **劣势** | 寿命短、魔法天赋低、社会地位低下 |

**参考伊洛纳**：哥布林像伊洛纳中的约恩村居民一样，是被主流社会边缘化的群体。

### 暗精灵（Dark Elf）—— 堕落的贵族

| 属性 | 内容 |
|------|------|
| **寿命** | 约300-500年 |
| **体型** | 修长优雅（约1.7-1.9米） |
| **特征** | 深色皮肤、银白发、紫瞳、尖耳 |
| **社会** | 氏族制，血脉至上，严格的等级制度 |
| **信仰** | 月神（可能是对更古老存在的误读） |
| **能力** | 暗元素亲和、隐匿、长寿、诅咒 |
| **禁忌** | 与外族通婚是死罪，混血儿必须处死 |

**参考D&D**：类似卓尔精灵，但更加强调"血脉纯洁"的执念。

### 人类（Human）—— 适应之王

| 属性 | 内容 |
|------|------|
| **寿命** | 约70-80年 |
| **特点** | 适应力强，学习能力优秀，多样性 |
| **社会** | 城邦/王国制，多样性，政治斗争频繁 |
| **优势** | 无元素亲和限制，可学习任何魔术；繁殖力和适应力平衡 |
| **劣势** | 寿命短，无种族天赋，容易内斗 |
| **特点** | 对混血儿的态度复杂：既恐惧又利用 |

**参考勇者斗恶龙**：像DQ中的人类一样，是世界的"默认"种族，但更加复杂。

### 矮人（Dwarf）—— 地下的守护者

| 属性 | 内容 |
|------|------|
| **寿命** | 约200-300年 |
| **体型** | 敦实（约1.3-1.5米） |
| **特征** | 浓密胡须、强壮、耐操、重视荣誉 |
| **社会** | 氏族制，重视工艺和血统，对外封闭 |
| **能力** | 锻造大师、土元素亲和、地下感知 |
| **特点** | 忠诚但固执，重视承诺胜过生命 |

**参考D&D**：经典矮人设定，但更加强调"地下王国"的神秘感。

### 混血儿（Half-Breed）—— 禁忌的存在

| 类型 | 特征 | 命运 |
|------|------|------|
| 哥布林×人类 | 体型中等，绿皮肤较浅 | 被哥布林排斥，被人类歧视 |
| 暗精灵×人类 | 肤色较浅，暗元素亲和弱 | 暗精灵追杀，人类恐惧 |
| 哥布林×暗精灵 | **诺提的种族**，极端罕见 | **双方追杀，被视为禁忌中的禁忌** |
| 混血×以太侵蚀 | 结晶化加速，力量变异 | 几乎无先例，诺提是首例 |

**参考克苏鲁**：混血儿的存在挑战了"种族纯洁"的观念，类似于克苏鲁神话中人类与深潜者的混血。

## 历史与传说

### 创世神话（参考克苏鲁）

> "在星辰尚未成型之时，原初的以太从虚空中涌出。它没有形状，没有意志，只是纯粹的存在。然后，"它们"从以太中醒来——不是神，不是魔，只是……更古老的东西。它们创造了世界，创造了生命，然后沉睡了。我们称它们为'上古者'，但这个名字对它们来说毫无意义。"

### 上古时代（黄金时期）

- 以太源晶从天而降，带来魔法文明
- 八元素体系建立，各种族和平共处
- 上古者沉睡，但留下了遗迹和守护者

**参考勇者斗恶龙**：像DQ的远古时代一样，是一个"失落的黄金时代"。

### 大分裂（悲剧的开端）

- 暗精灵发现月神信仰的"真相"，与其他种族决裂
- 哥布林被指控"亵渎上古者"，被驱逐到边缘地带
- 人类崛起，建立城邦，但失去了与以太的深层联系

**参考克苏鲁**：大分裂的原因可能是某个种族发现了"不该知道的真相"。

### 现今（动荡时代）

- 种族隔阂严重，混血被视为禁忌
- 以太源晶成为传说中的宝物，冒险者疯狂追寻
- 以太风暴逐渐增强，"彼方"的传说再次流传
- 暗精灵内部的"纯洁派"和"改革派"斗争加剧

## 组织与势力

### 冒险者公会（Adventurer's Guild）

| 等级 | 称号 | 权限 |
|------|------|------|
| 铜级 | 新手 | 接取简单任务 |
| 银级 | 正式冒险者 | 接取中等任务，进入地下城 |
| 金级 | 精英 | 接取高级任务，参与机密行动 |
| 白金级 | 传奇 | 自由行动，公会资源全力支持 |

**参考伊洛纳**：像伊洛纳的公会一样，是冒险者获取任务和资源的核心。

### 影月氏族（Shadowmoon Clan）—— 暗精灵的极端派

- **信仰**：月神（纯洁、秩序、血脉至上）
- **手段**：暗杀、追踪、诅咒、精神控制
- **目标**：消灭所有混血儿，维护暗精灵"纯洁"
- **秘密**：月神殿地下封印着某种存在，氏族首领可能已被"影响"

**参考克苏鲁**：影月氏族可能像克苏鲁神话中的邪教一样，崇拜着它们并不真正理解的存在。

### 以太研究会（Aether Research Society）

- **组成**：人类法师、学者、疯狂的炼金术师
- **目标**：研究以太的本质，寻找控制以太侵蚀的方法
- **秘密**：进行着不人道的实验，可能已与"彼方"的存在接触

## 货币与经济

| 货币 | 价值 | 说明 |
|------|------|------|
| 铜币 | 基础单位 | 平民日常使用 |
| 银币 | 100铜币 | 冒险者常用 |
| 金币 | 100银币 | 大额交易 |
| 以太结晶碎片 | 无价 | 黑市流通，可替代任何货币 |

**参考伊洛纳**：像伊洛纳一样，货币系统简单但实用。

## 克苏鲁元素融入

### 宇宙恐怖（Cosmic Horror）

- **上古者**：创造了世界但对人类漠不关心的存在
- **以太风暴中的东西**：偶尔出现在风暴中的巨大阴影，无法描述
- **月神殿下的封印**：暗精灵崇拜的"月神"可能只是某个上古者的投影
- **知识即疯狂**：了解以太本质越多，理智越脆弱

### 不可名状的存在

| 存在 | 形态 | 影响 |
|------|------|------|
| **风暴之主** | 以太风暴中的巨大阴影 | 引发以太侵蚀，扭曲现实 |
| **月之假面** | 暗精灵月神像的"另一面" | 低语、幻觉、血脉污染 |
| **晶灵** | 以太侵蚀后期的变异体 | 失去人性，成为能量的奴隶 |
| **彼方之物** | 从未被完整描述 | 穿越以太风暴的代价 |
