# 角色设定卡

## 诺提（Noti）

### 基本信息
| 项目 | 内容 |
|------|------|
| **本名** | 陈诺 |
| **现名** | 诺提（Noti） |
| **种族** | 哥布林 × 暗精灵 混血 |
| **年龄** | 3个月（体型相当于人类3-4岁） |
| **出生地** | 逃亡马车上（母亲艾莉娅临终产子） |

### 外貌特征
- **肤色**：暗绿色，皮肤表层覆盖淡蓝色以太结晶
- **体型**：小巧灵活，哥布林典型的小体型优势
- **面部**：尖耳、獠牙、突出颧骨，眼睛在暗处会发出微光
- **特殊**：结晶在情绪激动时会发光，觉醒后形成"以太印记"

### 性格
- **核心**：坚韧、好奇、保护欲强
- **矛盾**：人类记忆与哥布林身体的冲突，自我认同挣扎
- **成长线**：从恐慌逃避 → 接受现实 → 主动掌控力量

### 能力
| 阶段 | 能力 | 触发条件 |
|------|------|----------|
| 初期 | 身体强化（速度、敏捷） | 本能反应 |
| 训练期 | 基础元素感知 | 凯恩教导 |
| 觉醒期 | 以太印记·火焰爆发 | 生死危机 |
| 潜力 | 时空元素（待开发） | 以太源晶完全融合 |

### 背景故事
- 穿越前：大学生，游戏《伊瑟尔》玩家，为钱拼死通关
- 穿越后：成为游戏中哥布林盗贼"诺提"，保留游戏记忆和技能理解
- 特殊：携带"以太源晶"穿越，成为第四种力量来源

### 人际关系
```
诺提 ←父亲→ 加尔
    ←挚友/叔叔→ 格林
    ←导师→ 凯恩
    ←追杀→ 影月氏族
    ←潜在盟友→ 主城守卫
```

## 加尔（Garr）

### 基本信息
| 项目 | 内容 |
|------|------|
| **种族** | 哥布林 |
| **年龄** | 约35岁（哥布林成年早，但寿命短） |
| **职业** | 前冒险者，现逃亡者 |
| **身份** | 诺提的生父 |

### 外貌
- 接近人类的英俊面容（哥布林中的异类）
- 绿皮肤，身材精瘦但肌肉结实
- 常穿破旧皮甲，腰间挂着柴刀

### 性格
- **表面**：沉默寡言、冷硬、实用主义
- **内心**：深爱儿子，愿意付出一切
- **秘密**：自责未能保护艾莉娅，对诺提过度保护

### 能力
- 近战专家，柴刀运用出神入化
- 血液特殊：混合精灵泉水可制作生命药剂
- 战斗直觉敏锐，经验丰富

### 背景
- 年轻时离开哥布林部落，成为冒险者
- 与暗精灵艾莉娅相爱（禁忌之恋）
- 艾莉娅死后，独自抚养诺提逃亡

## 格林（Grin）

### 基本信息
| 项目 | 内容 |
|------|------|
| **种族** | 矮人 |
| **年龄** | 约80岁（矮人中年） |
| **职业** | 铁匠、战士 |
| **身份** | 加尔的挚友，诺提的叔叔辈 |

### 外貌
- 敦实矮壮，浓密红褐色胡须
- 常穿皮围裙，随身携带铁锤（也是武器）
- 手臂上有烫伤疤痕和战斗伤痕

### 性格
- 忠诚可靠，重情义
- 比加尔更健谈，是团队的气氛调节者
- 对诺提温柔，像对待自己孩子

### 能力
- 锻造大师：能修理和打造武器
- 锤术：铁锤作为武器威力惊人
- 野外生存：熟悉地形和追踪

### 背景
- 与加尔在冒险中结识，多次生死之交
- 自愿跟随加尔逃亡，放弃了自己的铁匠铺

## 凯恩（Kane）

### 基本信息
| 项目 | 内容 |
|------|------|
| **种族** | 人类 |
| **年龄** | 约40岁 |
| **职业** | 守卫队长 |
| **身份** | 诺提的导师 |

### 外貌
- 中年男性，体格健壮
- 常穿制式铠甲，披风上有主城徽记
- 面部有旧伤疤，眼神锐利但温和

### 性格
- 严厉但公正，训练时毫不留情
- 对诺提有特殊的关注和期待
- 沉稳可靠，深受部下信赖

### 能力
- 剑术大师，擅长教导
- 八元素基础魔术精通
- 战略眼光，善于判断形势

### 秘密
- 曾经见过类似的"以太侵蚀"案例
- 知道诺提体内的力量远超普通混血
- 可能与某个神秘组织有关联
