# 第1章 苏醒与排斥

黑暗。

永恒的黑暗。

然后是刺眼的光。

诺缇猛地睁开眼睛，大口喘息。肺部像被火烧一样疼，每一次呼吸都带着铁锈的味道。

他抬起手，映入眼帘的是一双手——苍白、瘦骨嶙峋，手背上爬满了青灰色的晶体纹路，像是某种奇怪的寄生生物在皮肤下生长。

这不是他的手。

记忆如潮水般涌入，但混乱不堪。破碎的片段、陌生的场景、诡异的人影...最后定格在一张狞笑的脸，然后是剧痛，然后是黑暗。

"呃..."

诺缇试图坐起来，但身体沉重得像灌了铅。他环顾四周——这是一个巨大的地下空间，四周是冰冷的石墙，空气中弥漫着血腥味和某种说不清的怪味。

他躺在...一个池子里。

这不是尸体堆，而是一排排劣质的营养仓池子，每个池子里都装着浑浊的液体。池子是用粗糙的石头砌成的，没有任何精密的设备，甚至连封盖都是简单的木板。

他的视线扫过周围的池子

第424号...池子里漂浮着一张苍白的脸，晶体纹路爬满了整张脸。

第425号...池子里的液体已经变成了暗红色。

第426号...一动不动，仿佛已经死了很久。

第427号...是他自己。

脚步声从远处传来。

诺缇的瞳孔收缩。屏住呼吸，假装自己还在昏迷中。

那个声音越来越近。

一个穿着白色长袍的人影出现在视野边缘。那人手中拿着一块发光的石板，正一边走一边记录着什么。

那人的脚步声在空旷的地下空间回荡，每一步都像是踩在诺缇的心脏上。诺缇能感觉到自己的心脏在剧烈跳动，几乎要从胸膛里跳出来。

他必须保持冷静。如果现在被发现还活着，会发生什么？

一个奇怪的声音在他脑海中响起。不是声音，更像是直接出现在脑海中的文字。

诺缇愣了一下，然后看到视野中浮现出一个半透明的面板。

【状态面板】
姓名：诺缇
穿越者类型：进化教派第427号试验体（意识占据体）
穿越时间：[已屏蔽]
身体状态：虚弱（需要营养）

穿越者？

诺缇的瞳孔收缩。他记得自己的过去——另一个世界，另一个人生。那些记忆虽然混乱，但确实存在。

所以他是穿越了？占据了这具失败试验体的身体？

"第426号..."那人继续说着，"确认死亡。"

"第427号..."那人走到了诺缇的池子边。

诺缇没有动，他强迫自己的呼吸保持平稳，甚至连眨眼都控制着。

那人的脚步声停在了池子边。

"奇怪..."那人自言自语道，"第427号...生命体征显示微弱但存在。看来还没完全死透。"

那人伸出手，掀开池子上的木板，准备检查。

大脑来不及思考现在的处境，但诺缇已经暴露，被发现还活着估计会继续实验，只有反抗了。

诺缇的心脏狂跳。他知道这是他唯一的机会。

就是现在。

诺缇猛地睁开眼睛，双手从浑浊的液体中伸出，死死扼住那人的脖子。

"咳...咳！"

那人完全没料到失败体会反击，脸色瞬间涨红，双手胡乱地抓挠着诺缇的手臂。

诺缇能感觉到那人的指甲深深嵌入他的皮肤，但他没有松手。他太弱了，如果让这人挣脱，自己必死无疑。

"呃...咳..."

那人的挣扎越来越弱，最终停止了。

诺缇松开手，身体发软地向后倒在池子里。

他杀人了。

他真的杀人了。

诺缇呆呆地看着自己的双手，颤抖个不停。血液残留在他苍白的皮肤上，触目惊心。

这是他第一次杀人。

恐惧、恶心、茫然...无数情绪在他心中翻涌。

然后，他感觉到了。

一股奇异的能量从那人的身体中渗出，缓缓流入他的体内。那种感觉很奇妙——像是温暖的液体流遍全身，带来了一丝久违的力量。

但也就仅此而已。

力量流入，然后消失在身体的深处。没有变强，没有恢复，甚至连疲惫感都没有减轻。

诺缇摇摇晃晃地从池子里爬出来。

他必须离开这里。

但怎么离开？

诺缇环顾四周，这里似乎是地下实验室的某处，四周都是石墙，没有出口。

就在这时，远处传来了说话声。

"废弃区又在暴动了！"

"该死，又要封锁了。"

"这次暴动比上次更严重，以太浓度已经超过了警戒线。"

废弃区？

诺缇愣了一下。

废弃区以太暴动...那意味着那里很危险。

但同时也意味着——那里可能没有守卫。

如果守卫们都在封锁废弃区入口，那么废弃区内部...应该就没有守卫了。

这是他的机会。

诺缇握紧金属棒，朝着声音的方向走去。如果他在废弃区入口附近，或许能找到入口。

但就在他走了几步之后，远处传来了一阵脚步声。

不是一个人的脚步声。

"报告！G-3失去信号！D区第427号池子！"

"立刻封锁D区！"

"启动警戒级别2！"

警报声突然响起，刺耳的红光开始在地下空间中闪烁。

诺缇的心脏狂跳。被发现了。

他加快脚步，朝着废弃区方向冲去。

但脚步声也越来越近。

"那边！"一个声音喊道。

诺缇猛地回头，看到几个穿着灰色制服的人影正从不同方向逼近。他们手中都拿着武器，脸上带着冷漠的表情。

废弃区入口就在前方，但距离还有几十米。

追兵却已经逼近。

"拦截！"灰色制服的人喊道。

诺缇咬紧牙关，用尽全身力气冲向废弃区入口。

"啪！"

一根鞭子般的武器抽在他的背上，带来一阵剧痛。

诺缇踉跄了一下，差点摔倒。

但他没有停下。

"快！别让他跑了！"

灰色制服的人加速，很快就会追上。

诺缇冲进了废弃区入口。

"该死！他进了废弃区！"追兵的声音从身后传来，"让开！那里以太浓度太高，通知监控中心！"

诺缇没有回头。

通道越来越窄，空气中的怪味越来越浓重。他能感觉到皮肤上的晶体纹路在蠕动——这里的某种东西正在影响他的身体。

但他没有停下。

因为身后，追兵的脚步声越来越近。

该死...

诺缇咬紧牙关，继续向前。他不知道这条通道通向哪里，也不知道会遭遇什么。

但他知道一件事——停下就是死。

所以他只能继续向前狂奔。
