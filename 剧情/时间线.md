# 伊瑟尔大陆时间线

---

## 远古时期：双神时代

### 未知纪元前（数万年前）

| 时间 | 事件 |
|------|------|
| **创造与毁灭的诞生** | 创造者和毁灭者作为世界的"双手"同时诞生，开始永恒的斗争。创造者不断创造万物，毁灭者不断毁灭万物。 |
| **世界的循环** | 创造与毁灭的循环维持着世界的运转。创造者追求"完美的创造"，毁灭者证明"所有创造都会毁灭"。 |
| **双神矛盾** | 创造者认为毁灭者是"破坏者"，毁灭者认为创造者是"重复者"。双方永无休止地斗争，但都无法彻底击败对方。 |
| **循环的疲惫** | 数万年的斗争让双方都感到疲惫。创造者因为无法创造真正的完美而沮丧，毁灭者因为无法停止毁灭而绝望。 |

## 三神时代：观测者的崛起

### 第一次转折：观测者的诞生

| 时间 | 事件 |
|------|------|
| **创造与毁灭的激烈战斗** | 双神的战斗打得异常激烈，世界法则被打散成无数碎片。 |
| **观测者摄取法则** | 观测者（或称"记录者"）利用这个机会，摄取了被打散的世界法则，开始进化成第三神。 |
| **第三神的诞生** | 观测者成功摄取足够的法则，成为世界的"第三神"。祂获得了"观测"和"记录"的权限，开始干涉世界。 |
| **双神的震怒** | 创造者和毁灭者发现第三神的存在，震怒于祂的"窃贼"行为。 |

### 第二次转折：三神的第一次大战

| 时间 | 事件 |
|------|------|
| **三神开战** | 创造者和毁灭者联手对抗观测者，试图夺回被摄食的力量。 |
| **世界的裂痕** | 战斗几乎摧毁了世界。三方都意识到：如果继续这样下去，世界会彻底崩溃。 |
| **休战协定** | 三方达成"休战协定"。创造者和毁灭者选择沉睡，观测者独自维持世界的因果律。 |
| **观测者获得优势** | 双神沉睡后，观测者成为相对最强的神。祂开始吞噬更多的法则碎片，试图补全自己的本质。 |

## 王座时代：六王座系统的建立

### 王座的起源

| 时间 | 事件 |
|------|------|
| **王座概念的诞生** | 为了维持世界的平衡，双神在沉睡前创造了六个"因果席位"，即王座。每个王座代表一种极致的理念。 |
| **王座规则的建立** | 六王座被分为三个层次：基础层（万灵王座、白骨王座）、构建层（熔炉王座、铁血王座）、超越层（千衍王座、虚无王座）。 |
| **王座争夺的意义** | 王座争夺不是为了权力，而是为了"争夺理念的极致"。只有能将对应理念推到极端的人，才能坐上王座。 |
| **虚无王座的特殊之处** | 虚无王座是双神创造的"击杀观测者的工具"。它代表"规则之外"的理念，可以与双神的力量结合。 |

### 六王座的初代持有者

| 时间 | 事件 |
|------|------|
| **初代万灵王座** | 某个致力于"生生不息"的强者登上王座。王座赋予他"延续生命"的权限。 |
| **初代白骨王座** | 莫特成为白骨王座持有者。王座赋予他"记录死亡"的权限。 |
| **初代熔炉王座** | 伊格成为熔炉王座持有者。王座赋予他"锻造世界"的权限。 |
| **初代铁血王座** | 某个致力于"征服一切"的强者登上王座。王座赋予他"力量与征服"的权限。 |
| **初代千衍王座** | 维德成为千衍王座持有者。王座赋予他"时空记录"的权限。他不知道自己是观测者的投影。 |
| **初代虚无王座** | 露希（真名Lucis）成为虚无王座持有者。王座赋予他"抹除存在"的权限。 |

## 露希的时代：隐匿与谋划

### 露希的觉醒

| 时间 | 事件 |
|------|------|
| **露希的崛起** | 露希从某个被遗忘的纪元崛起。他在首次开创独有术式后（传奇阶），被刚成神的观测者选择为种子。 |
| **王座本质的发现** | 露希意外发现虚无王座的本质：被三神授予的允许引导世界法则力量的特权席位。 |
| **毁灭者的暗中标记** | 毁灭者（在沉睡中）对露希产生了共鸣，暗中标记了他。露希本人完全不知道。 |

### 露希的隐匿

| 时间 | 事件 |
|------|------|
| **接近半神** | 露希在观测者的引导下，逐渐接近王阶的极限。但在即将突破至半神时，被创造与毁灭双神同时"察觉"。 |
| **主动抹除** | 露希意识到自己太危险了，主动发动最大规模的因果抹除，将自己的名字、存在、历史全部删除。 |
| **隐匿成功** | 露希成功隐匿于因果之外，成为准神级存在。观测者无法锁定他。 |
| **毁灭者的引导** | 毁灭者暗中"允许"露希隐匿成功，将其保留为"终结执行者"的工具。 |

### 数千年的隐匿

| 时间 | 事件 |
|------|------|
| **观察每一次王座更迭** | 露希在隐匿期间，观察每一次王座更迭，寻找规则的薄弱点。 |
| **发现王座是锁链** | 露希发现：三神的授予是锁链，观测者本身也是规则的产物。 |
| **寻找无法被控制的存在** | 露希转而寻找"无法被完全控制"的存在，试图通过他们制造悖论。 |
| **双神的暗中引导** | 创造者和毁灭者在沉睡中暗中引导露希，让他等待"创造者的种子"成长。 |

## 上世代：王座陨落与新时代

### 上世代的大战

| 时间 | 事件 |
|------|------|
| **王座争夺的激烈化** | 上世代的王座争夺异常激烈，多个王座同时更迭。 |
| **万灵王座陨落** | 上一个持有者试图创造"永生生命"，结果创造出无法控制的怪物，最终被自己的创造物吞噬。王座空置。 |
| **铁血王座陨落** | 上一个持有者被自己的军队背叛，王座被污染后消失。王座空置。 |
| **虚无王座隐匿** | 露希主动隐匿，所有人都以为王座陨落。各国史书的记录互相矛盾。 |
| **莫特记录所有死亡** | 莫特（白骨王座）记录了所有王座陨落的死亡。他记住这些死亡，等待王座重新选择持有者。 |
| **伊格目睹陨落** | 伊格（熔炉王座）是上世代大战的亲历者，他从王座陨落中学到了"不完美导致毁灭"的教训。 |

### 双神的暗中谋划

| 时间 | 事件 |
|------|------|
| **双神达成共识** | 创造者和毁灭者早已达成共识：击杀观测者，夺回被摄食的力量。 |
| **代理人计划的制定** | 双神决定用代理人击杀观测者，因为正面对抗会加速世界崩溃。 |
| **两个种子的选择**：诺缇（创造者标记）和露希（毁灭者标记） | 创造者标记诺缇，毁灭者标记露希。双神暗中引导他们，但两人完全不知道自己的真实角色。 |
| **虚无王座的规划** | 虚无王座被规划为"击杀观测者的工具"。当诺缇登上王座，创造与毁灭的力量会在虚无王座上结合，击杀观测者。 |

## 当前世代：诺缇的穿越与觉醒

### 诺缇的穿越

| 时间 | 事件 |
|------|------|
| **异界意识降临** | 陈诺（前世）的异界意识降临到伊瑟尔大陆，进入进化教派第427号试验体的身体。 |
| **创造者的标记** | 创造者（在沉睡中）对诺缇进行了标记，因为他的异界意识携带的"不属于这个世界"的知识代表了"不同的创造模板"。 |
| **异界意识覆盖标记** | 诺缇的异界意识"覆盖"了创造者的标记，使观测者无法扫描到。观测者误判诺缇"没有被任何神选中"。 |
| **观测者的误判** | 观测者认为诺缇是一个"自由变量"和"可控种子"，开始引导他的成长。 |

### 诺缇的觉醒（穿越后约8-10岁）

| 时间 | 事件 |
|------|------|
| **在废弃试验场醒来** | 诺缇在进化教派废弃试验场（位于黑礁滩地下）的垃圾场和尸体堆中醒来。 |
| **初期状态**：身体虚弱，皮肤下有淡蓝色结晶纹路（源晶排斥反应的痕迹），无记忆，无身份 | 诺缇对外声称"不记得以前的事"。 |
| **第一阶段：求生** | 诺缇的核心驱动力是"活下去"。他谨慎、多疑、善于伪装。 |
| **第二阶段：觉醒** | 诺缇第一次杀人，发现体内有"东西"在帮他——超速感知、战斗直觉。他开始怀疑这是"系统"还是"天赋"。 |
| **第三阶段：冷血化** | 诺缇发现进化教派的真相，意识到自己不是"穿越者"，而是"被制造的"。开始冷血化。 |

### 诺缇的权力发掘

| 时间 | 事件 |
|------|------|
| **因果观测**：诺缇获得了来自观测者标记的"因果观测"能力 | 包括预知危险、时间变慢、噩梦成真、杀完人后的平静。诺缇以为这是"第六感"或"天赋"。 |
| **吞噬进化**：诺缇获得了自身的"吞噬进化"能力 | 击杀生命后汲取力量，达成条件后选择进化方向。 |
| **观测者的引导** | 观测者通过梦境植入、直觉提示、认知加速、情绪抑制、因果污染等方式引导诺缇。诺缇逐渐意识到"观测者"的存在。 |
| **观测者标记的持续存在** | 诺缇在登上王座之前，观测者标记一直存在，观测者可以实时追踪他。 |

## 当前世代：王座争夺的序幕

### 王座规则的涌动

| 时间 | 事件 |
|------|------|
| **近百年的规则涌动** | 近百年，莫特开始感受到王座规则的涌动。万灵王座规则、铁血王座规则、虚无王座规则开始在各地涌动。 |
| **莫特的应对** | 莫特记录所有尝试永生的人的死亡（万灵王座），记录所有因征服而死的战士的死亡（铁血王座），感知到诺缇的登座并记录这个异常（虚无王座）。 |
| **伊格的应对** | 伊格不参与万灵王座的争夺，派遣锻火军团参与铁血王座的争夺，诺缇登座后开始观察他，评估他是威胁还是变量。 |

### 露希的引导

| 时间 | 事件 |
|------|------|
| **寻找种子** | 露希在隐匿期间，寻找"无法被完全控制"的存在。他发现诺缇的"异常性"——异界意识体的"规则外"特质。 |
| **暗中观察** | 露希在诺缇实力达到准王级之前，一直在暗中观察，不直接接触。 |
| **初次接触** | 当诺缇实力达到准王级并达成一些比较传奇的成就后，露希评估诺缇有机会靠吞噬进化成为下一位准神。 |
| **忌惮观测者标记** | 露希忌惮诺缇被观测者标记为种子，无法现身见面，于是常在诺缇周围引导。 |
| **透露成神真相** | 露希试图在观测者无法观测的地方向诺缇透露成神真相并拉拢。 |
| **引导登上王座** | 露希引导诺缇登上虚无王座，但诺缇不知道露希就是前任持有者。 |

## 当前世代：伊格的谋划

### 伊格的培养继承者

| 时间 | 事件 |
|------|------|
| **接近寿命极限** | 伊格约280岁，接近人类寿命极限，力量正在衰退。他必须在死前为继承者铺路。 |
| **瓦伦的出生** | 伊格培养继承者"灰烬之子"瓦伦（Valen，22岁），锻造天赋异常但缺乏经验和战场磨练。 |
| **铁血王座的共鸣** | 伊格意识到：瓦伦的锻造天赋本质是"锻造征服"——他能锻造军队、锻造战争、锻造征服。这个天赋与铁血王座天然共鸣。 |
| **战争是共鸣的唯一途径** | 伊格认为：只有通过征服和杀戮，瓦伦才能真正与铁血王座共鸣，成为双王持有者（熔炉+铁血）。 |

|------|------|
| **让瓦伦亲率锻火军团** | 伊格亲自将锻火军团的指挥权交给瓦伦，让他亲自指挥铁血王座争夺战。 |
| **为瓦伦打造完美战场** | 伊格用熔炉王座的力量，将战场锻造成"完美"的训练场，确保瓦伦能快速成长。 |
| **暗中支持瓦伦** | 伊格在暗中用熔炉王座的力量帮助瓦伦，但不直接干预。 |
| **与莫特的秘密合作** | 伊格与莫特有秘密协议——伊格为莫特提供战场，莫特为伊格提供亡灵情报。 |
| **阻止记录** | 伊格用炼金院的力量，阻止千衍王座的记录。 |

| **怀疑虚无王座** | 莫特怀疑虚无王座与打破循环有关，因此对诺缇格外关注。 |

## 当前世代：观测者的吞噬计划

### 观测者的补全本质渴望

| 时间 | 事件 |
|------|------|
| **被剥夺的本质** | 观测者原本是全知全能的存在，但在与双神的战斗中被剥夺了部分本质。 |
| **当前状态** | 观测者现在只有"记录"的本质，缺乏"创造"和"毁灭"的本质。 |
| **补全方法** | 吞噬王座法则碎片是补全本质的唯一方法。 |
| **最终目标** | 吞噬所有王座法则碎片，恢复被剥夺的本质，成为真正的全知全能。 |

### 观测者的吞噬计划

| 时间 | 事件 |
|------|------|
| **白骨王座** | 观测者在莫特虚弱时直接吞噬，补全"毁灭"本质。莫特在战争中消耗大量力量，观测者在寻找机会。 |
| **熔炉王座** | 伊格即将死亡，观测者在等待他死后直接吞噬，补全"创造"本质。 |
| **千衍王座** | 维德与观测者本质相似，观测者在考虑吞噬维德后，千衍王座是否会崩塌。 |
| **铁血王座** | 争夺者死亡后，直接吞噬法则碎片，补全"毁灭"本质。 |
| **万灵王座** | 争夺者死亡后，直接吞噬法则碎片，补全"创造"本质。 |

### 观测者的误判

| 时间 | 事件 |
|------|------|
| **对诺缇的误判** | 观测者认为诺缇"没有被任何神选中"，因此将其视为"自由变量"和"可控种子"。 |
| **创造者的暗中操作** | 创造者标记被异界意识"覆盖"，观测者扫描时发现"没有生效空间"。 |
| **观测者的引导** | 观测者引导诺缇成长，试图将他培养成潜在的盟友。但不知道诺缇已经被创造者暗中选中。 |
| **对双神的误判** | 观测者以为双神在沉睡中无法干预，实际上双神在暗中引导诺缇和露希。 |
| **对虚无王座的误判** | 观测者以为虚无王座是规则的一部分，不是击杀工具。 |

## 当前世代：诺缇登上王座

### 诺缇登上虚无王座
