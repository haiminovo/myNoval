# myNoval — 修订大纲 v2.0

## 核心主题

> **"怪物不是天生的，是被逼出来的。"**

主角诺提的冷血化不是突然的黑化，而是被世界一步步逼出来的必然结果。被排斥、被恐惧、被背叛——最终他发现，唯一能信任的只有自己。

## 角色关系演变

### 诺提与加尔

```
初期：保护与被保护
    ↓
中期：信任出现裂痕（噩梦、偷听对话）
    ↓
高潮：弑父（分不清是自卫还是谋杀）
    ↓
后期：永远的疑问——加尔到底想不想杀他？
```

**核心设计**：读者和诺提一样，永远不知道加尔那天是不是真的想杀他。

- **证据A**：加尔确实和格林讨论过"如果诺提变成怪物就杀了他"
- **证据B**：加尔走进帐篷时确实拿着刀
- **反证A**：那可能只是削木头的柴刀
- **反证B**：加尔看到诺提 awake 时的表情是担忧，不是杀意

### 诺提与凯恩

```
初期：导师与学生
    ↓
中期：同类（都杀过自己的"父亲"）
    ↓
后期：互相利用还是真正信任？
```

### 诺提与格林

```
初期：叔侄般的亲情
    ↓
弑父后：决裂（"不要叫我叔叔"）
    ↓
后期：能否修复？
```
