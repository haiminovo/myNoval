## 性格特征

### 核心特质

- **绝对冷静**：死亡已经带走所有情绪波动，只剩下理性的观察
- **通透**：对生命和死亡有深刻的理解，不执着于任何一方
- **耐心**：死亡是终点，没有终点就没有过程，他愿意等待一切终结
- **温和而冷酷**：对待亡灵温和，对待生命冷酷，因为生命终将死亡
- **贪婪而谨慎**：想要收集更多强者的灵魂，但必须避免被其他王座发现

### 行为模式

1. **收集者心态**：将战争看作"灵魂的收获"，他主动收集战争中死亡的强者
2. **直接对抗**：为了收集灵魂，他会直接派遣亡灵军队对抗活着的军队
3. **转化强敌**：将死亡的强者转化成亡灵战士，成为亡者国度的战力
4. **劫持战场**：在战争结束后，立即劫持战场，将所有死者转化

## 成王之路

### 原本的生命

莫特曾是古代的"送葬人"，一个专门处理尸体和葬礼的仪式师。他的工作是确保每个死者都能得到合适的送别，确保死亡被妥善记录。

### 死亡与重生

在一次大规模的瘟疫中，莫特处理了数千具尸体。瘟疫没有放过他，他在死亡时突然"看见"了死亡的本质——死亡不是终结，而是生命的另一种存在形式。

在那一瞬间，白骨王座选择了他。王座不是赋予力量，而是让他"看见"死亡的真实形态。他成为亡灵，但保留了完整的意识和记忆。

### 成为白骨王

莫特在死亡后建立了一个概念上的国度——"亡者国度"。这不是一个地理上的国度，而是所有死者的意识汇聚的地方。

- 他成为了白骨王座的持有者
- 他的存在从"活着的人"变成了"死亡本身"
- 他开始记录每一个死亡，维护死亡和生命的平衡

## 技能与天赋

### 白骨法则

**第一层：死亡凝视**

- **直接死亡**：被凝视的生命会在数息内自然死亡
- **寿命感知**：一眼看出任何生命的剩余寿命
- **死亡预演**：可以提前看到某人的死亡场景

**第二层：亡灵操控**

- **召唤亡灵**：可以召唤任何已死亡的存在为亡灵
- **亡灵对话**：可以与任何死者的意识对话
- **白骨造物**：可以用白骨构建任何形态的亡灵战士

**第三层：生命消解**

- **渐进死亡**：让任何生命逐渐失去活力，最终成为白骨
- **死亡领域**：展开后，领域内的所有生命都会加速走向死亡
- **白骨降临**：召唤大量的白骨，覆盖整个战场

### 特殊天赋

- **死亡免疫**：他已经是死亡本身，无法再次死亡
- **亡灵共鸣**：可以感知所有亡灵的存在和状态
- **生死边界**：他生活在生死边界，可以自由进出生死

## 观念与诉求

### 核心观念

**死亡不是终结，是生命的另一种存在形式。强者死后应成为亡灵，继续战斗。**

- 生命终将死亡，这是必然的规律
- 死亡不是惩罚，是生命成为亡灵的机会
- 强者的灵魂太珍贵，不应该消散
- 他的存在是为了收集强者的灵魂，让它们成为亡灵战士

### 诉求

1. **收集强者灵魂**：确保每个死亡的强者都被转化成亡灵
2. **建造亡灵大军**：用收集的强者灵魂，建造强大的亡灵军队
3. **对抗活着的军队**：为了收集灵魂，必须对抗活着的军队
4. **等待终局**：等待世界的最终死亡，届时所有灵魂都将成为亡灵

### 对双神和观测者的态度

| 神明 | 莫特的态度 | 原因 |
|------|-----------|------|
| **创造者** | 中立 | 创造与死亡是对立但不冲突的 |
| **毁灭者** | 中立 | 毁灭与死亡有关但不相同 |
| **观测者** | 冷漠 | 观测者试图记录一切，但莫特也在记录一切，两者有重叠 |

## 所属势力：亡者国度

### 势力本质

亡者国度不是一个地理上的国度，而是一个"意识汇聚的地方"。所有死者的意识最终会汇聚到这里，接受莫特的记录和分类。

### 势力结构

| 阶层 | 描述 |
|------|------|
| **白骨王** | 莫特，王座持有者 |
| **亡灵守卫** | 由莫特召唤的强大亡灵，维护亡者国度的秩序 |
| **死者意识** | 所有死者的意识，处于不同的阶段（新生、成熟、消散） |
| **白骨使者** | 由莫特赋予力量的特殊亡灵，可以被派遣到现世 |

### 势力活动

- **记录死亡**：记录每个死亡的详细信息
- **分类亡灵**：将亡灵分类到不同的"领域"
- **维护边界**：确保现世与亡者国度的边界稳定
- **避免污染**：防止死亡被扭曲成非正常形态

### 上个世代王座陨落的记忆

莫特记得上个世代的大战，那是他成为白骨王后经历过的最惨烈的死亡：

- **万灵王座陨落**：上一个持有者在试图创造"永生生命"时意外死亡，王座空置
- **铁血王座陨落**：上一个持有者被自己的军队背叛，王座被污染后消失
- **虚无王座隐匿**：上一个持有者主动隐匿，所有人都以为王座陨落

莫特记录了所有这些死亡，他知道王座的规则从未消失，只是在等待新的持有者。

## 王座争夺的影响

### 近百年的规则涌动

近百年，莫特开始感受到王座规则的涌动：

- **万灵王座规则**：生命之力开始在各地涌动，多个势力感受到召唤
- **铁血王座规则**：征服之力开始在各地涌动，多个军阀开始扩张
- **虚无王座规则**：虚无之力在各地"闪烁"，但很快消失（因为露希的隐匿和诺缇的登座）

### 莫特的应对

| 王座 | 涌动表现 | 莫特的应对 |
|------|---------|-----------|
| **万灵王座** | 生命之力异常强大，有人试图创造永生 | 莫特记录所有尝试永生的人的死亡 |
| **铁血王座** | 征服之力异常强大，多个势力开始扩张 | 莫特记录所有因征服而死的战士的死亡 |
| **虚无王座** | 虚无之力在各地"闪烁" | 莫特感知到诺缇的登座，记录这个异常 |

### 潜在风险

| 风险 | 应对 |
|------|------|
| 死亡被滥用 | 派遣白骨使者阻止非正常的死亡 |
| 亡灵叛乱 | 直接镇压，将叛乱的亡灵重新分类 |
| 世界即将崩溃 | 加速记录，确保所有死亡都被保存 |
| 万灵王座被滥用 | 记录所有尝试永生的人的死亡，确保他们最终死亡 |

## 备注

- 莫特不认为自己是"死神"，他认为自己是"死亡记录者"
- 他对生命没有恶意，对死亡也没有恶意，他只是记录
- 白骨王座的力量不是"杀死"，而是"记录死亡"
- 莫特的"白骨之书"记录了从世界诞生以来的所有死亡
- 莫特对双神和观测者的态度都是中立的，他不参与任何一方的争斗
- 莫特记得上个世代的大战，他记录了所有王座陨落的死亡
- 莫特对王座争夺不感兴趣，他只记录争夺者的死亡
- 莫特偶尔会向特定的争夺者泄露某个敌人的死亡方式，这是他唯一"干涉"王座争夺的方式
