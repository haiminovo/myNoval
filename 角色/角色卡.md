# 角色设定卡 v2.0

## 诺提（Noti）

### 基本信息
| 项目 | 内容 |
|------|------|
| **本名** | 陈诺（穿越前） |
| **现名** | 诺提（Noti） |
| **种族** | 哥布林 × 暗精灵 × 人类 × 以太源晶 |
| **年龄** | 8个月（实际存活时间） |
| **阵营** | 观测者（混血种，被世界视为观测者阵营） |
| **称号** | 学徒（？），实际权限未知 |

### 外貌特征
- **肤色**：暗绿色，皮肤表层覆盖淡蓝色以太结晶（正在消退）
- **体型**：小巧灵活，哥布林典型小体型优势
- **面部**：尖耳、獠牙、突出颧骨
- **特殊**：眼睛在情绪激动时会变成金黄色（观测者力量觉醒的标志）
- **以太印记**：淡蓝色纹路，情绪激动时发光，能力觉醒时爆发出强光

### 性格演变
| 阶段 | 性格 | 触发事件 |
|------|------|----------|
| 初期 | 恐惧、迷茫、依赖 | 穿越后发现自己是哥布林 |
| 中期 | 沉默、警觉、麻木 | 被人类孩子扔石头，意识到自己是"怪物" |
| 冷血化 | 平静、冷漠、不信任 | 噩梦频发，怀疑加尔要杀他 |
| 弑父后 | 空洞、麻木、自我厌恶 | 杀死加尔，但没有哭泣 |
| 觉醒期 | 冷静、锐利、寻求真相 | 跳海穿越风暴带 |

### 核心矛盾
- **人类记忆 vs 哥布林身体**：陈诺的善良与诺提的生存本能冲突
- **信任 vs 背叛**：渴望亲情，但不再相信任何人
- **力量 vs 失控**：拥有强大力量，但害怕自己会伤害他人

### 能力
| 阶段 | 能力 | 表现 |
|------|------|------|
| 本能 | 身体强化（速度、敏捷） | 战斗中时间变慢的感觉 |
| 噩梦 | 既视感/预知 | 看到未来场景（加尔杀他/他杀加尔） |
| 引导 | 以太感知 | 掌心浮现蓝光 |
| 失控 | 湮灭之力 | 愤怒时释放强光，分解目标 |
| 觉醒 | 因果观测 | 观测者力量的完全觉醒（第三卷展开） |

### 人际关系
```
诺提 ←杀→ 加尔（父亲，已死）
    ←决裂→ 格林（叔叔，"不要叫我叔叔"）
    ←导师→ 凯恩（进化教派叛逃者，同类）
    ←追杀→ 影月氏族（暗精灵极端派）
    ←引导→ 观测者（三神之一，暗中注视）
```

## 加尔（Garr）— 已死

### 基本信息
| 项目 | 内容 |
|------|------|
| **种族** | 哥布林 × 人类 混血 |
| **身份** | 诺提的生父 |
| **状态** | 被诺提杀死（第11章） |

### 性格
- **表面**：沉默寡言、冷硬、实用主义
- **内心**：深爱儿子，但过度保护导致不信任
- **矛盾**：保护诺提 vs 害怕诺提变成怪物

### 关键悬念
**加尔之死的真相**：
- 加尔确实和格林讨论过"如果诺提变成怪物就杀了他"
- 但加尔走进帐篷时，手里拿着的到底是柴刀还是武器？
- 他的表情是担忧还是杀意？
- **诺提永远不知道答案**

## 凯恩（Kane）

### 基本信息
| 项目 | 内容 |
|------|------|
| **种族** | 人类 |
| **年龄** | 约40岁 |
| **身份** | 守卫队长，诺提的导师 |

- 知晓进化教派的秘密

### 与诺提的关系
```
导师 → 同类 → ？
```

凯恩看到了诺提身上的"进化教派痕迹"，但最终发现那是"天然源晶"的印记。他选择教导诺提，既是投资，也是赎罪。

## 影月氏族（反派）

### 基本信息
- **种族**：暗精灵
- **信仰**：月神，崇尚血脉纯洁
- **首领**：永夜君王·影月（王阶十二人之一）

### 追杀诺提的原因
1. 诺提是哥布林×暗精灵混血（禁忌）
2. 艾莉娅（诺提母亲）背叛氏族
3. 诺提的存在是氏族的耻辱

### 关键特征
- 追兵中最高那个暗精灵摘兜帽微笑——猎手的笑
- 暗示影月氏族不只是追杀，是在"观察"诺提的变化

## 观测者（三神之一）

### 本质
- 三神中最弱的一个
- 上个时代，创世者与毁灭者大战后元气大伤，逐渐沉沦
- 观测者相对变强，因为其他两位的影响减弱

### 与诺提的关系
- 诺提是混血种，被世界"视为"观测者阵营
- 观测者暗中引导诺提的因果线
- 诺提不知道观测者的存在，以为是"运气"或"天赋"

### 表现形式
- 噩梦（既视感）
- 眼睛变色（金黄色）
- 以太风暴共鸣
