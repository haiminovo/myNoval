# 诺提·暗痕（Notte Darkscar）

## 基本信息

| 项目 | 内容 |
|------|------|
| **真名** | 陈诺（前世）→ 诺提·暗痕（今生） |
| **身份** | 进化教派第427号试验体 |
| **种族** | 混血（哥布林×暗精灵） |
| **年龄** | 穿越时约8-10岁（身体年龄） |
| **外貌** | 暗绿色皮肤，尖耳，暗金瞳孔，淡蓝结晶纹路 |
| **前世** | 《伊瑟尔》玩家，通关后心脏骤停 |

## 核心设定

### 起源

诺提不是"穿越到婴儿身上"，而是**以太源晶将陈诺的意识投射到一具已死的试验体身上**。

这具身体的原主是进化教派制造的**第427号试验体**——哥布林与暗精灵的混血，被植入以太源晶碎片后死于排斥反应。陈诺的意识在死亡瞬间被源晶捕获，"借用"了这具刚刚死去的身体。

**关键：诺提不知道自己是"试验体"。** 他以为自己是普通穿越者，占据了某个混血孤儿的身体。直到很久以后，他才会发现真相。

### 觉醒地点

**进化教派废弃试验场**，位于黑礁滩地下。诺提在垃圾场和尸体堆中醒来，身边是原主的尸体和其他试验体的残骸。

### 初期状态

- **身体虚弱**：试验体长期营养不良，肌肉萎缩
- **结晶化**：皮肤下有大片淡蓝色结晶纹路（源晶排斥反应的痕迹）
- **无记忆**：不继承原主记忆，只继承**肌肉记忆**（本能动作）
- **无身份**：没有名字，没有过去，没有"父母"
- **伪装失忆**：对外声称"不记得以前的事"

## 性格演变

### 第一阶段：求生（第1-3章）

- **核心驱动力**：活下去
- **行为特征**：谨慎、多疑、善于伪装
- **心理活动**：大量观察和分析，极少信任他人
- **标志性行为**：
  - 在垃圾场翻找食物时，先观察再行动
  - 遇到其他流浪儿，第一反应是躲藏而非对抗
  - 给自己取名"诺提"，但对外只说"不记得名字"

### 第二阶段：觉醒（第4-6章）

- **转折点**：第一次杀人（狼人夜袭）
- **变化**：发现体内有"东西"在帮他——超速感知、战斗直觉
- **困惑**：不确定这是"系统"还是"这具身体的天赋"
- **标志性行为**：
  - 杀狼人后，没有恐惧，而是**饥饿感**
  - 开始记录自己的"异常"（结晶发光、噩梦、金色瞳孔）
  - 对凯恩既依赖又警惕

### 第三阶段：冷血化（第7-9章）

- **触发点**：发现进化教派的真相
- **变化**：意识到自己不是"穿越者"，而是"被制造的"
- **心理冲击**：如果连"穿越"都是假的，那什么是真的？
- **标志性行为**：
  - 杀死进化教派追兵时，面无表情
  - 凯恩战死时没有哭，只把他争来的时间用来逃跑
  - 跳海前，回头看了一眼黑岛，第一次主动选择"不像路的路"

### 第四阶段：虚无（第10章后）

- **核心状态**：存在主义危机
- **行为特征**：高效、冷静、无情绪波动
- **内心独白**：
  > "我不是陈诺。我不是诺提。我是什么？"
  > 
  > "一个被制造出来的容器，装着一个死去的玩家的意识，被另一个世界观测着。"

## 能力系统：因果观测

### 第一阶段：被动触发（第1-5章）

诺提不知道自己在被"引导"。他以为自己的判断来自"直觉"或"经验"。

| 现象 | 诺提的理解 | 实际机制 |
|------|-----------|---------|
| 偶尔预知危险 | "第六感" | 因果律预演：观测者分支模拟死亡可能性 |
| 战斗时时间"变慢" | "肾上腺素爆发" | 认知加速：观测者提高信息处理速度 |
| 噩梦中的场景成真 | "巧合" | 因果污染：噩梦是观测者传递的"未来碎片" |
| 杀完人后感到"平静" | "我天生冷血" | 观测者反馈：执行"最优解"后的奖励机制 |

### 第二阶段：主动怀疑（第6-8章）

诺提开始注意到"巧合"太多，产生怀疑。

**关键场景**：

> 诺提在日记本上记录：
> 
> "第47次'直觉'正确。
> 
> 今天凯恩让我走左边巷道，我'感觉'右边更安全。走了右边，避开了一场斗殴。
> 
> 但左边巷道……我后来去看了，什么都没有。
> 
> 是我的直觉错了？还是……左边本来会有危险，但我走了右边，所以危险没有发生？"
> 
> "如果危险是因为我走了左边才会发生，那我的'直觉'不是在预测未来，而是在……选择未来？"

### 第三阶段：认知崩溃（第9章后）

诺提意识到"观测者"的存在，但不知道它是什么。

**关键独白**：

> "它不是系统。它没有界面，没有任务，没有奖励。
> 
> 它只是……看着我。在我快要死的时候，推我一把。在我犹豫的时候，给我一个'直觉'。
> 
> 它不想让我死。至少现在不想。
> 
> 为什么？我在它眼里是什么？一个有趣的实验？一个必要的工具？还是一个……笑话？"

## 与观测者的关系

### 观测者的"引导"方式

1. **梦境植入**：通过噩梦传递"未来碎片"（通常是死亡场景）
2. **直觉提示**：在关键时刻产生强烈的"应该这样做"的感觉
3. **认知加速**：战斗中提高信息处理速度，表现为"时间变慢"
4. **情绪抑制**：在需要冷静时抑制恐惧和悲伤（表现为"冷血"）
5. **因果污染**：让诺提偶尔"看到"不该看到的东西（如加尔的日记）

### 诺提的反应演变

| 阶段 | 态度 | 行为 |
|------|------|------|
| 无知期 | 无感知 | 完全依赖"直觉" |
| 怀疑期 | 警惕 | 试图对抗"直觉"，但总是失败 |
| 认知期 | 恐惧 | 意识到被操控，试图寻找观测者 |
| 共存期 | 冷漠 | 接受被观测的事实，利用它生存 |
| 反噬期 | 反抗 | 试图切断与观测者的连接（后期剧情） |

## 关键关系

### 凯恩·铁壁（Kane Ironwall）

**定位**：导师 / 替代父亲 / 另一个试验体

**关系演变**：

1. **初遇**：格林把诺提带到旧矿井，凯恩先确认他是否会立刻死亡/失控
2. **收容**：凯恩向回收派申请临时样本权限，记录者给七天担保期
3. **训练**：凯恩教诺提战斗和控制以太，但**从不解释全部原因**
4. **秘密**：凯恩也是进化教派产物（第13号试验体），但不是成功体，而是精神控制失败的**半成品**
5. **冲突**：凯恩留在教派体系内，靠自己的力量守住黑岛矿井这块地盘；回收派认为他可用，销毁派认为他迟早失控
6. **死亡**：凯恩被销毁派灭口，死前告诉诺提稳定剂只是锁，风暴带才是出路
7. **遗留影响**：凯恩不是父亲，却是第一个教诺提如何在记录者面前活的人；他的稳定剂不是救赎，而是通往风暴带的三小时锁

**凯恩的遗言**：

> "药不是解药，是锁。三小时。
> 
> 去风暴带。只有那里能压住源晶。
> 
> 因为记录到这里……已经没意义了。"

### 格林（Grim）

**定位**：意外盟友 / 知情者

**背景**：
- 不是矮人，是**人类矿工**，因长期接触以太矿石而身体变异（矮小、强壮）
- 在黑岛地下矿井工作，偶然发现了进化教派的废弃试验场
- 救了诺提，但**知道诺提是试验体**

**关系**：
- 格林对诺提有**愧疚感**（因为他没早点发现试验场，没能救其他孩子）
- 诺提对格林**不信任**（格林知道太多，但不说）
- 最终：格林为掩护诺提逃入排水渠而死，留下干粮、火石、黑色金属牌和013铜牌

### 进化教派

**定位**：幕后黑手 / 诺提的"制造者"

**诺提的认知演变**：

1. **不知道**：以为自己是普通穿越者
2. **怀疑**：发现身体异常（结晶、金瞳、超速感知）
3. **确认**：发现凯恩记录册和记录者评估，确认自己是427号异常体
4. **复仇**：单枪匹马摧毁教派分部（第15章）
5. **虚无**：复仇后发现，教派只是更大势力的工具

## 标志性场景

### 场景1：垃圾场醒来

> 诺提在尸体堆中醒来。
> 
> 他的第一感觉是**饿**。不是普通的饿，是胃在啃自己的那种饿。
> 
> 第二感觉是**疼**。全身都疼，尤其是胸口——皮肤下有硬块，像嵌了碎玻璃。
> 
> 第三感觉是**冷**。他躺在一堆垃圾和尸体中间，身上只有一件破布。
> 
> 他爬起来，开始翻身边的尸体。
> 
> 不是找食物。是找**信息**。
> 
> 尸体是绿色的，和他一样。有尖耳朵，和他一样。但死了。
> 
> 他翻到一个布包，里面有：半块发霉的面包、一把生锈的小刀、一张写着数字的纸条——"427"。
> 
> 他吃了面包。收好小刀。把纸条塞进嘴里嚼碎吞下。
> 
> 然后他开始**观察环境**。
> 
> 这是一个地下空间，穹顶很高，有通风口透进微光。墙壁上有管道和电缆——不是中世纪，是**工业时代**。
> 
> 远处有脚步声。
> 
> 诺提躲进阴影里，握紧小刀。
> 
> 他没有喊"这是哪里"。他没有哭。他没有做任何"穿越者"会做的事。
> 
> 他只是**等待**。

### 场景2：第一次杀人

> 狼人扑向格林。
> 
> 诺提没有思考。他的身体自己动了——**
> 这不是他的动作，是原主的肌肉记忆**。
> 
> 他滚到狼人腹下，小刀刺入柔软的腹部，划开，内脏涌出。
> 
> 狼人倒下。诺提浑身是血。
> 
> 他看着自己的手，没有发抖。
> 
> 他看着狼人的尸体，感到**饿**。
> 
> 不是想吃。是某种更深的东西——像是身体在说："更多。我需要更多。"
> 
> 格林看着他，眼里有恐惧。
> 
> 诺提知道那种恐惧。他低头看自己的胸口——淡蓝色的结晶纹路在发光，像呼吸一样脉动。
> 
> 他拉紧衣服，遮住光芒。
> 
> "走。"他说，声音沙哑，"它们还会来。"

### 场景3：发现真相

> 诺提在凯恩的遗物中找到一本笔记。
> 
> 第1页："试验体427，植入日期……"
> 
> 第13页："意识投影成功，宿主死亡，外来意识接管……"
> 
> 第27页："观测者反应活跃，试验体表现出超常生存能力……"
> 
> 第50页："建议终止试验，该个体不可控……"
> 
> 诺提合上笔记。
> 
> 他没有愤怒。没有悲伤。
> 
> 他只是……**确认了一件一直知道的事**。
> 
> "所以，"他对自己说，"我不是穿越者。我是……一个实验。"
> 
> 他烧了笔记。
> 
> 然后他开始**计划**。

## 角色弧光

### 主题：存在与虚无

诺提的故事不是"逆袭"，而是**存在主义危机**。

- **起点**：一个死去的玩家，占据了一具试验体的身体
- **过程**：逐渐发现"自己"可能是被制造的、被观测的、被引导的
- **终点**：？？？（开放式——他是否能找到"真正的自我"？）

### 核心矛盾

| 层面 | 矛盾 |
|------|------|
| **身份** | 我是陈诺？诺提？427号？还是……什么都不是？ |
| **自由意志** | 我的选择是我的，还是观测者引导的？ |
| **人性** | 如果我是被制造的，我还有"人性"吗？ |
| **存在** | 被观测的存在，算存在吗？ |

### 最终问题

> **"如果观测者停止观测，我会消失吗？"**
> 
> "还是说……我会终于自由？"
