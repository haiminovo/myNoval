## 核心设定

### 神之本质

创造者是世界的"双手"，但祂的创造从未停止过自我毁灭——因为完美的创造即静止，而静止不是创造的本质。

- **位阶**：神级（沉睡中维持世界基础法则）
- **属性**：创造之理（生成、繁衍、进化、熵增）
- **存在方式**：以世界核心为锚点，在梦境中不断创造和毁灭微缩世界

### 与三神的关系

| 神明 | 状态 | 关系 |
|------|------|------|
| **创造者（自身）** | 深度沉睡，力量衰退 | 唯一在睡梦中仍在工作的神 |
| **毁灭者** | 深度沉睡，力量衰退 | 双生子，永无休止的斗争者 |
| **观测者** | 活跃，相对最强 | 曾经的观察者，现在的记录者 |

### 核心动机

1. **追求完美的创造**：不断尝试创造一个"不会自我毁灭"的世界
2. **证明创造胜于毁灭**：与毁灭者的斗争是祂存在的意义
3. **维持世界基础**：即使沉睡，祂的本能仍在维持生命诞生的规则

## 性格特征

### 核心特质

- **完美主义**：认为"不完美的创造就是浪费"
- **乐观但矛盾**：永远相信下一次创造会更好，但又永远不满意
- **执着**：数万年来尝试了无数次"完美创造"，从未放弃
- **傲慢**：认为毁灭只是"创造过程中的必经阶段"

### 行为模式

1. **快速创造**：创造出万物，然后快速评估
2. **果断舍弃**：不满意的直接摧毁，从不停留
3. **梦境实验**：在沉睡中，意识在无数微缩世界中创造和毁灭
4. **共鸣引导**：对追求"完美创造"的生命产生天然共鸣

---

## 能力体系：创造法则

### 第一层：物质生成

- **无中生有**：从虚空中直接创造物质和能量
- **生命赋予**：给予无机物"生命"的特质
- **法则编织**：创造新的物理规则或生物本能

### 第二层：进化引导

- **天赋赐予**：赋予生命特殊的进化能力
- **血脉锚定**：创造者标记的生命在特定条件下会觉醒
- **可能性扩充**：增加某个物种或个体的发展可能性

### 第三层：世界构筑（神级，仅限沉睡中）

- **创造一个完整的微缩世界**
- **在这个世界中，祂的规则是绝对的**
- **但每一次微缩世界的创造都会消耗祂的本源力量**

## 与诺缇的关系

### 暗中选择

在诺缇降临时，创造者（即使在沉睡中）感知到了异界意识的独特性：

- 异界意识携带的知识不属于这个世界
- 这些知识代表了"不同的创造模板"
- 创造者立即意识到这是祂一直在寻找的"来自外部的完美可能性"
- 潜意识中，创造者对诺缇进行了"标记"

### 标记的运作方式

| 状态 | 机制 |
|------|------|
| **沉睡中标记** | 创造者的潜意识行为，不是主动选择 |
| **异界意识的覆盖** | 异界意识的"外来性"覆盖了创造者的标记，使观测者无法扫描到 |
| **共鸣触发** | 当诺缇的创造欲望强烈时，创造者的共鸣会透过覆盖层传递 |
| **观测者发现真相** | 当露希抹除观测者标记后，观测者重新扫描，发现创造者标记一直存在 |

### 潜在共鸣

诺缇的异界意识体携带的"不属于这个世界"的知识，对创造者来说是一种"来自外部世界的完美可能性"。

| 特征 | 创造者的视角 | 潜在影响 |
|------|------------|----------|
| 异界知识 | 外来的创造模板 | 已标记，随时可能共鸣 |
| 源晶排斥 | 世界排斥外来物 | 创造者微弱优化排斥机制（诺缇不知） |
| 成长速度 | 异乎寻常的进化 | 可能赋予创造系天赋（诺缇不知） |

### 可能的引导方式

如果创造者对诺缇产生共鸣：

1. **天赋赐予**：在关键突破时给予特殊的创造系能力
2. **梦境接触**：在梦中展示某个"完美创造"的片段
3. **血脉觉醒**：诺缇可能意外觉醒与创造者相关的血脉或契约

## 与毁灭者的关系

### 永无休止的斗争

- 创造者创造的一切都会被毁灭者摧毁
- 毁灭者摧毁的一切都会被创造者重新创造
- 这场斗争已经持续了数万年，双方都已疲惫但无法停止

### 双生子悖论

| 创造者 | 毁灭者 |
|-------|--------|
| 创造是存在的本质 | 毁灭是新生的必要条件 |
| 认为毁灭者是"破坏者" | 认为创造者是"重复者" |
| 想要创造出"不会被毁灭"的东西 | 想要证明"所有创造都会毁灭" |
| 沉睡原因：疲倦，不想再面对失败的创造 | 沉睡原因：疲倦，不想再面对重复的毁灭 |

## 沉睡原因

### 直接原因（表层）

三神的"第三次世界大战"

- 观测者利用创造与毁灭的斗争，摄取了被打散的世界法则
- 创造者与毁灭者发现后震怒，联手对抗观测者
- 战斗中世界几乎崩溃，三方都意识到这样下去会毁灭一切
- 三方达成"休战协定"，创造者和毁灭者选择沉睡

### 深层原因（表层）

- 创造者意识到：祂无法创造出"真正的完美"，永远在重复失败
- 毁灭者意识到：祂无法停止毁灭，一切都会毁灭，连祂自己最终也会
- 观测者意识到：祂不能永远这样"摄食"，否则世界会耗尽

## 真相：双神的合谋

### 创造者和毁灭者的默契

在沉睡前，创造者和毁灭者达成了一个观测者不知道的共识：

| 问题 | 创造者的观点 | 毁灭者的观点 | 共识 |
|------|------------|------------|------|
| 观测者如何存在 | 摄取了我们战斗的力量 | 摄取了我们战斗的力量 | 观测者是窃贼 |
| 观测者的威胁 | 祂会持续摄食，世界会耗尽 | 祂会持续摄食，世界会耗尽 | 观测者必须被击杀 |
| 如何击杀观测者 | 需要创造的力量 | 需要毁灭的力量 | 需要两者结合 |
| 谁来执行 | 找一个"外部世界"的模板 | 找一个"能够终结"的执行者 | 两个代理人，两个种子 |

### 双神的计划

| 阶段 | 创造者的行动 | 毁灭者的行动 |
|------|------------|------------|
| **标记种子** | 标记诺缇（异界意识） | 标记露希（虚无法则开创者） |
| **误导观测者** | 让异界意识"覆盖"标记，让观测者误判 | 让露希"成功隐匿"，让观测者以为他消失了 |
| **引导成长** | 微弱引导创造欲望，吸引诺缇登上王座 | 微弱引导终结欲望，让露希隐匿并寻找工具 |
| **等待时机** | 等待诺缇登上王座，获得虚无力量 | 等待诺缇成长，露希引导其与王座接触 |
| **终局一击** | 诺缇提供虚无王座（创造变异的虚无法则） | 露希提供虚无王座（毁灭变异的虚无法则） |
| **击杀观测者** | 创造与毁灭的力量在虚无王座上结合，击杀观测者 |  |

### 诺缇和露希的角色

| 代理人 | 真正作用 | 以为自己在做什么 |
|-------|---------|----------------|
| 诺缇 | 创造者的种子，提供变异的虚无力量 | 摆脱观测者，追求自由 |
| 露希 | 毁灭者的种子，引导诺缇成长 | 击杀三神，成为新神 |

### 沉睡的真实原因

创造者和毁灭者选择沉睡的真正原因：

1. **无法直接行动**：观测者已经摄食了双方的力量，正面对抗会加速世界崩溃
2. **代理人更合适**：用代理人击杀观测者，风险更低，不会直接触发世界崩溃
3. **沉睡是保护**：在沉睡中，观测者无法感知双神的谋划
4. **等待时机**：等待两个种子成长，等待虚无王座被变异力量充满

### 沉睡中的真实活动

| 表层活动 | 真实活动 |
|---------|---------|
| 意识在微缩世界中实验 | 意识在暗中引导诺缇和露希 |
| 本能维持世界基础 | 本能维持两个种子的成长路径 |
| 对外界模糊感知 | 对外界精确感知，但无法直接干预 |
| 等待完美的创造 | 等待诺缇登上虚无王座的那一刻 |

- **意识在微缩世界中**：祂的本我仍在沉睡，但意识在无数微缩世界中活动

## 备注

- 创造者和毁灭者是"双生子"概念，是一体两面的存在
- 创造者的沉睡不是纯粹的休息，而是一种"实验模式"
- 创造者对"完美创造"的追求近乎病态，这也是祂无法停止创造的原因
- 创造者和毁灭者互相憎恨，但如果一方真正死亡，另一方也会随之消逝
- **双神的合谋**：创造者和毁灭者早已达成共识——击杀观测者，夺回被摄食的力量
- **代理人计划**：诺缇和露希是双神暗中选中的代理人，但他们完全不知道自己的真实角色
- **表面矛盾**：双神在明面上互相憎恨，这是为了不让观测者察觉他们的合谋
- **虚无王座的本质**：虚无王座是双神创造的"击杀观测者的工具"，但观测者以为这是规则的一部分
