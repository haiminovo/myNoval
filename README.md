# 以太进化录 — 小说项目

## 目录结构

```
以太进化录/
├── 正文/              # 小说正文（按卷分目录）
│   └── 第一卷-黑岛/
├── 设定/              # 世界设定（小说圣经为核心）
│   ├── 小说圣经.md    # 唯一权威设定文件
│   ├── 异界穿越者.md
│   └── 进化与以太.md
├── 角色/              # 角色卡
├── 大纲/              # 剧情大纲、时间线、爽点设计
│   ├── 大纲.md
│   ├── 时间线.md
│   └── 爽点设计.md
├── 追踪/              # 写作状态追踪（compact 后恢复用）
│   ├── 上下文.md      # 当前写作进度
│   ├── 伏笔.md        # 伏笔追踪
│   └── 角色状态.md    # 角色状态追踪
└── 对标/              # 对标作品分析（待创建）
```

## 工具集

本项目使用 story-setup 网文写作工具集，支持：
- `/story-long-write` — 长篇写作
- `/story-long-analyze` — 拆文分析
- `/story-deslop` — 去 AI 味
- `/story-review` — 多视角审查
- `/story-cover` — 封面生成
