## 世界观设定

### 地理
- **黑岛** — 被诅咒的海域，暗精灵不敢靠近，主角避难所
- **主城** — 人类聚居地，冒险者公会所在地
- **暗精灵森林** — 影月氏族领地，追杀主角的势力
- **以太风暴带** — 危险的魔法区域，连接异次元

### 力量体系
| 元素 | 特点 | 代表角色 |
|------|------|----------|
| 水 | 治愈、变化 | |
| 火 | 破坏、爆发 | **诺提（觉醒后）** |
| 风 | 速度、感知 | |
| 土 | 防御、坚韧 | |
| 光 | 神圣、净化 | |
| 暗 | 隐匿、诅咒 | **暗精灵** |
| 生命 | 生长、恢复 | |
| 时空 | 稀有、禁忌 | **以太源晶** |

### 以太侵蚀
- 接触过量以太能量后的身体变异
- 诺提出生时全身结晶化，靠父亲血液+精灵泉水维持
- 结晶消退速度异常快，暗示体内有第四种力量

## 角色设定

### 主角
| 属性 | 内容 |
|------|------|
| **姓名** | 诺提（Noti） |
| **原名** | 陈诺（穿越前） |
| **种族** | 哥布林×暗精灵混血 |
| **年龄** | 3个月（相当于人类3岁体型） |
| **特征** | 绿色皮肤、尖耳、獠牙、皮肤表层以太结晶 |
| **能力** | 以太印记（火元素爆发）、身体强化 |
| **目标** | 活下去 → 找到回家方法 → 保护重要的人 |

### 主要配角

#### 加尔（父亲）
- **种族**：哥布林
- **身份**：前冒险者，诺提的生父
- **特征**：接近人类英俊面容，绿皮肤，战斗力强悍
- **秘密**：体内流淌着特殊血脉，血液可制作生命药剂
- **动机**：保护儿子，逃离暗精灵追杀

#### 格林
- **种族**：矮人
- **身份**：加尔的挚友，铁匠
- **特征**：敦实，浓密胡须，忠诚可靠
- **作用**：战斗支援、情感支持

#### 凯恩
- **种族**：人类
- **身份**：守卫队长
- **特征**：中年，沉稳有力，严厉但关怀
- **作用**：诺提的导师，系统训练其战斗能力
- **秘密**：似乎知道诺提的特殊之处
